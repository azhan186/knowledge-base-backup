## 一、狼人文化

### 1. 狼人阶级体系

| 阶级 | 说明 | 角色定位 |
|------|------|----------|
| 狼王 | 狼群绝对大佬，手握最高权威，执掌所有决策与安全 | 皇帝/霸总型主角 |
| 狼辅 | 大佬左膀右臂，帮首领打理琐事、传达命令 | 助理/副手角色 |
| 庶裔 | 狼群底层群体，弱小易被排挤 | 前期受虐/后期逆袭角色 |
| 狼后 | 首领的正式命运伴侣，共同管理狼群 | 女主/女配 |

### 2. 狼人基础设定

- **内在狼**：每个狼人都有专属内在狼，有独立意识，会影响宿主的情绪、判断和力量
- **月亮女神**：狼人唯一信仰，掌控命运与伴侣分配（可推动剧情用，不涉及宗教）
- **变身**：一般16-18岁解锁，也可后期觉醒，适配女主成长线
- **无狼**：部分狼人到年龄仍无狼形，多为弱势者，前期虐点+后期逆袭设定
- **后期觉醒**：前期无狼者有概率觉醒狼形，还能解锁治愈者等特殊技能

### 3. 情感 & 命运设定

| 设定 | 说明 |
|------|------|
| 命运伴侣 | 狼人一生唯一灵魂伴侣，天生互相吸引，被背叛会承受极致痛苦 |
| 拒绝缔结 | 单方/双方拒绝时遭彼此反噬 |
| 二次机会伴侣 | 被拒绝后极小概率重新缔结，破镜重圆 |
| 伴侣舞会 | 成年狼人专属仪式，用于寻找确认命运伴侣 |
| 伴侣仪式 | 狼人版婚礼，命运伴侣正式绑定 |
| 意识联结 | 结为命运伴侣后解锁专属技能，靠意念交流 |

## 二、吸血鬼文化

> 原型：东欧、巴尔干地区民间传说，19世纪维多利亚时代经《德古拉》定型

### 1. 基础设定

**外型特征**
- 面容苍白，有尖锐獠牙，体温冰冷
- 没有影子，无法在镜子中留下倒影
- 瞳孔在进食或情绪激动时改变（变红、竖瞳等）

**能力**
- 永生与自愈
- 变身：蝙蝠、狼、老鼠或薄雾
- 心灵感应与精神控制
- 远超常人的力量与速度
- 魅力与诱惑

**弱点**
- 惧怕阳光
- 惧怕圣物
- 木桩刺心、斩首、火焚
- 大蒜驱赶（东欧民间传说）
- 无法进入未经邀请的人类住宅

### 2. 社会等级

| 等级 | 头衔 | 说明 |
|------|------|------|
| 高等 | 亲王 | 一家之长，权利顶峰，继承传承之力 |
| 高等 | 公爵 | 家族长老，实力接近亲王 |
| 高等 | 伯爵 | 家族后起之秀，力量强大 |
| 中等 | 子爵/男爵 | 初级积累阶段 |
| 低等 | 末代血族 | 刚被初拥，未得亲王承认，猎人首选目标 |

**家族规则**
- 领权：各种族有地盘自治权，外来者须尊重领主
- 后裔：发展后裔需长老许可，创造者对被初拥者承担教育责任
- 规则：擅自杀死同类者死；遮蔽法则（不得暴露于人类）

### 3. 情感 & 命运设定

| 设定 | 说明 | 剧情用途 |
|------|------|----------|
| 初拥 | 吸干对方血再让其饮下自己的血，创造者与被初拥者形成特殊血缘、精神纽带 | 甜宠/虐心 |
| 血契 | 通过交换血液形成的契约，象征信任、忠诚和保护 | 建立羁绊 |
| 血契影响 | 增强情感联系、力量共享、可不再受日光伤害 | 能力提升 |
| 解除血契 | 需特定仪式或魔法，可能导致力量丧失或精神痛苦 | 虐点/情感拉扯 |
