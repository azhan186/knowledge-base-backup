## 一、核心定义

心脏病是总称，指所有让心脏结构或功能异常、无法正常泵血的疾病。

### 主要类型
| 类型 | 描述 |
|-----|------|
| **冠心病** | 冠状动脉变窄或堵住，心肌缺血→胸痛（心绞痛）或心肌梗死（细胞坏死） |
| **心律失常** | 心脏跳得太快/太慢/乱跳 |
| **心力衰竭** | 心脏泵血无力 |
| **瓣膜病** | 心脏"门"关不严或打不开 |

**一句话总结**：心脏这个"泵"本身，或它的管道、电路、阀门出了问题，都属于心脏病。

## 二、生活类比：自来水供水系统

| 心脏组件 | 类比 | 功能 |
|---------|-----|------|
| 心脏 | 水泵 | 日夜不停把血液打到全身 |
| 血管 | 全屋水管 | 输送血液 |
| 冠状动脉 | 给水泵供电的专用电缆 | 给心肌供血供氧 |

### 四种故障对应
1. **冠心病（最常见）**：电缆绝缘层沉积油垢（动脉粥样硬化斑块），变脆变窄→水泵缺电→胸闷；某处突然堵死→电机烧坏（心梗）
2. **心力衰竭**：水泵马达老化无力，只能打到二楼打不到三楼→乏力、气喘、脚肿
3. **心律失常**：电路板/传感器坏了→水泵转得快/乱抖/停几秒→心慌、头晕、晕倒
4. **心脏瓣膜病**：阀门关不严（水倒流）或生锈打不开（水流不出）→水泵效率大打折扣

## 三、历史演变

| 时代 | 里程碑 |
|-----|-------|
| 1628年 | 英国医生哈维发现血液是循环的，心脏是泵 |
| 1816年 | 法国医生雷奈克发明听诊器，能"听见"心脏异常 |
| 1903年 | 心电图机出现，能画心脏"电路图" |
| 1929年 | 德国医生福斯曼把导尿管从手臂静脉插到右心房→获诺贝尔奖，开启心导管时代 |
| 1953年 | 心肺机发明，可在停跳心脏上手术；随后诞生冠状动脉搭桥术 |
| 1977年 | 德国医生做第一例球囊血管成形术；后来发明支架防止血管回缩 |
| 近20年 | 导管换瓣膜（TAVI手术），不用开胸 |

## 四、现代心脏病学体系

### 预防层面
管理危险因素：控制"三高"（高血压、高血糖、高血脂）、戒烟、健康饮食、规律运动。阿司匹林、他汀类降脂药用于预防高危人群心梗、脑梗。

### 诊断层面
| 工具 | 用途 |
|-----|------|
| 心电图 | 看电路 |
| 心脏超声 | 看结构和阀门 |
| 运动平板试验 | 看心脏在负荷下的反应 |
| 冠状动脉CT造影 | 无创看血管狭窄 |
| 冠脉造影（金标准） | 直接伸导管进去看，可当场放支架 |

### 治疗层面
- **药物**：降压、降糖、降脂、抗凝、控制心率——基石
- **介入治疗**：放支架、植入起搏器/除颤器、封堵器、经导管换瓣膜
- **外科手术**：搭桥、瓣膜修复或置换、心脏移植、心室辅助装置（"人工心脏"）
- **心脏康复**：常被忽略但极其重要（运动、教育、心理支持）

## 五、主要争议与前沿挑战

### 主要争议

| 争议 | 核心问题 |
|-----|---------|
| 胆固醇理论与饮食 | 吃蛋黄和饱和脂肪有多坏？糖和精制碳水是不是更坏的元凶？生酮、低碳水饮食对心脏长期影响？ |
| 支架是否放太多？ | 稳定心绞痛中，支架缓解症状优于单纯吃药，但预防死亡和远期心梗未必优于最佳药物 |
| 阿司匹林一级预防 | 新研究显示对健康人，消化道出血风险可能超过微弱获益，指南已大幅收紧 |

### 前沿挑战
- **AI与可穿戴设备**：手表检出房颤，AI从心电图早期发现心衰。挑战：海量警报管理，避免假阳性恐慌和医疗浪费
- **心衰治疗"四驾马车"**：新药"黄金组合"让很多人心脏功能大幅恢复，挑战在于让所有符合条件的病人用上、用对
- **心肌再生与基因治疗**：心脏再生能力极差，坏死肌肉变疤痕。干细胞疗法、将成纤维细胞"重编程"为心肌细胞——旨在真正修复心肌，仍处实验阶段
- **异种移植**：基因编辑猪心脏移植给人，已有几例尝试。解决器官短缺的终极想象，但排斥、感染和伦理问题巨大
