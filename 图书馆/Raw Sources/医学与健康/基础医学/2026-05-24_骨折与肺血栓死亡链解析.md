# 骨折与肺血栓：死亡链解析

## 分类
- 来源：医学科普
- 标签：骨折、肺栓塞、静脉血栓、骨科、围手术期、高龄患者

---

## 一、为什么腿骨骨折会引来致命的肺血栓？

> 魏尔肖"血栓三要素"在骨折病人身上同时、急剧、高强度爆发

### 三要素在骨折中的激活

| 三要素 | 骨折中的激活机制 | 类比 |
|--------|-----------------|------|
| **血管壁损伤** | 骨折骨茬刺伤、撕裂静脉血管内壁 | 供水管道源头"严重爆管" |
| **血流瘀滞** | 剧痛和固定（石膏/牵引）让整条腿完全不能动，肌肉泵作用消失 | 一潭彻底静止的死水 |
| **血液高凝状态** | 骨折引发强烈全身应激反应，肝脏大量生成纤维蛋白原等凝血因子 | 水里人为撒入大量速干水泥粉 |

### 死亡链全过程

```
腿骨骨折
  ↓
血管壁损伤 + 血流瘀滞 + 血液高凝
  ↓
下肢深静脉血栓（DVT）形成
  ↓
患者开始活动（甚至只是翻身）
  ↓
血栓脱落
  ↓
顺着血流：下腔静脉 → 右心房 → 右心室 → 肺动脉
  ↓
肺栓塞（"淤泥块"卡死"净水过滤器"）
```

## 二、医生如何打破这条死亡链？

### 危险分层
- 骨折患者入院后用 **Caprini评分表** 评估血栓风险
- 下肢大手术 → 直接极高危

### 基础预防（所有人必做）

- **早期活动**：术后麻醉一过，立刻做"踝泵运动"（勾脚尖、踩刹车）
- **物理预防**：穿医用弹力袜、间歇充气加压装置，从外面把静脉血"挤回去"

### 药物预防（高危者核心措施）

- **低分子肝素/磺达肝癸钠**（"肚皮针"）
- 术前术后规律注射，目的：暂时降低血液凝固能力
- 同时精确平衡抗凝与伤口出血风险

### 早期筛查

- 定期监测 **D-二聚体**（突然升高提示血栓）
- 出现单侧腿肿、胸痛呼吸困难 → 立刻安排腿部血管彩超或CTPA

## 三、高龄病人专项管理

### 核心认知
- 高龄老人身体储备差，一旦肺栓塞，代偿空间极小，**一击致命**
- 但多重用药、肾功能减退、易摔倒 → 抗凝治疗出血风险也成倍增加

### 检查：不典型信号的识别

| 系统 | 不典型信号 |
|------|-----------|
| **呼吸系统** | 不明原因呼吸急促（安静时>20次/分）、气短、血氧饱和度下降 |
| **神经系统** | 突发神志改变（精神萎靡、胡言乱语、嗜睡、晕厥）— 心肺衰竭导致脑缺氧 |
| **循环系统** | 血压莫名降低、心跳不规律加快 |
| **肢体信号** | 单侧腿肿、皮温升高 |

### 检查手段特别说明

- **D-二聚体**：价值在"阴性排除"。老年人即使无血栓也常因慢性炎症/肿瘤使其升高 → **阴性预测价值远大于阳性确诊价值**
- **CTPA**：确诊金标准。造影剂肾损伤顾虑 → 检查前查肾功能，权衡利弊。高度怀疑高危肺栓塞需紧急确诊时，救命第一
- **床边检查**：床旁心脏超声（观察右心扩大等间接征象）、床旁腿部静脉超声

### 预防：物理方法用尽，药物极度审慎

**基础与物理预防（安全基石，必须穷尽）**

- 踝泵运动与肌肉收缩：每小时5-10分钟"勾脚尖、踩刹车"，被动运动期间照护者被动屈伸和按摩
- 机械性加压装置：首选间歇充气加压装置，次选医用梯度压力袜（禁忌症：腿部溃疡、坏疽、未处理的缺血病）
- 早期床上活动：不影响骨折固定的前提下尽早翻身、活动健侧肢体
- 基础支持：保证水分，防止脱水导致血液浓缩

**药物预防（精细平衡的艺术）**

评估三点：血栓风险有多高？出血风险有多大？肾功能允不允许？

| 药物 | 说明 |
|------|------|
| 低分子肝素（"肚皮针"） | 主流，按体重和肾功能调整剂量；严重肾功能不全者需减量或禁用 |
| 磺达肝癸钠 | 出血风险相对更低，肾功能不全时需调整 |
| 新型口服抗凝药 | 方便，但高龄体重轻、肾功能临界者剂量需极精准，有消化道出血风险 |
| 阿司匹林 | 骨科大手术预防中地位回归，可作为备选；但高龄老人需警惕胃肠道出血 |

**当出血风险极高时**（近期胃肠道出血、脑出血、严重高血压未控）：暂停或不用抗凝药，完全依靠物理预防，高强化监测

## 四、核心管理思路：骨科-老年科共管模式

| 骨科 | 老年科 |
|------|--------|
| 处理骨折 | 评估心脑肾功能、多重用药、营养、跌倒风险、认知状况 |

目标：保证预防和治疗决策是**平衡了全局**的，而非仅盯住血栓一项。
