# 宇宙星体种类总览（ Classification of Celestial Objects ）

> 核心定义：宇宙中的星体可以分为恒星、行星、卫星、矮行星、小天体，以及由它们组成的恒星系统和宇宙大尺度结构等多种类型。

---

## 宇宙星体主要类型

| 类别 | 核心特征 | 实例 |
|------|----------|------|
| **恒星** | 由热气体组成，能自己发光发热的球体 | 太阳（G型黄矮星），天狼星，参宿四（红巨星），天狼星B（白矮星） |
| **行星** | 自身不发光，绕恒星运行的质量较大、近似球形的天体 | 水星、金星、地球、火星、木星、土星、天王星、海王星，及众多系外行星 |
| **卫星** | 绕行星运行的天体 | 月球，木卫三，土卫六 |
| **矮行星** | 介于行星与小行星之间，质量足以形成球形，但不能清除轨道邻域 | 冥王星，谷神星，阋神星 |
| **小天体** | 太阳系内除行星、矮行星外环绕太阳运行的小型天体 | 小行星带的小行星、彗星、流星体 |
| **恒星系统** | 由多个天体因引力作用形成的聚集结构 | 星团、星系、星系团、超星系团 |

---

## 恒星的多样分类

### 按演化阶段（太阳的一生）

```
主序星（壮年太阳）→ 红巨星（衰老膨胀）→ 白矮星（坍缩致密残骸）
                                                    ↓（更大质量恒星）
                                            超新星 → 中子星 / 黑洞
```

### 按表面温度（颜色/光谱）

| 类型 | 温度 | 颜色 | 实例 |
|------|------|------|------|
| O | 最高 | 蓝色 | 蓝巨星 |
| B | 高 | 蓝白 | B型主序星 |
| A | 较高 | 白色 | 织女星 |
| F | 中等 | 黄白 | 南河三 |
| **G** | 中等 | **黄色** | **太阳** |
| K | 较低 | 橙色 | 毕宿五 |
| M | 低 | 红色 | 红矮星（如比邻星）、参宿四 |

> 记忆口诀：**O B A F G K M**（Oh Be A Fine Girl Kiss Me）

### 特殊与终极形态

| 形态 | 说明 |
|------|------|
| **褐矮星** | "失败恒星"，质量不足以维持氢聚变 |
| **超新星** | 爆发前夜的大质量恒星 |
| **中子星** | 恒星坍缩后的致密残骸 |
| **黑洞** | 恒星演化的终极状态，逃逸速度超过光速 |

---

## 有趣的"异类"星体

| 类型 | 说明 |
|------|------|
| **变星** | 亮度随时间发生变化的恒星。**造父变星**是测量宇宙距离的重要工具 |
| **类星体（Quasar）** | 极其明亮、遥远的活跃星系核，能量来源可能是正在吞噬物质的**超大质量黑洞** |
| **星际物质** | 星际间稀薄的气体和尘埃，是**恒星诞生的原料** |
| **人造天体** | 人造卫星、空间探测器，属于人类发射的太阳系小天体 |

---

## 总结：宇宙的多样性

从炽热的恒星到冰冷的行星，从弥漫的星云到浩瀚的星系：

| 层级 | 类型 |
|------|------|
| **微观** | 星际物质（气体和尘埃） |
| **恒星级** | 恒星（主序星→红巨星→白矮星/超新星→中子星/黑洞）、褐矮星 |
| **行星级** | 行星（类地/类木）、矮行星、卫星 |
| **小天体** | 小行星、彗星、流星体、人造天体 |
| **系统级** | 星团、星系、星系团、超星系团 |
| **特殊** | 变星（造父变星）、类星体（活跃星系核） |
