# 行星形成（ Planet Formation ）

> 核心定义：行星形成是年轻恒星周围的一盘气体和尘埃（原行星盘），在几千万年内，通过微观粘附、引力聚集和剧烈碰撞，逐步从微米级的尘埃成长为数千公里级的天体的过程。

---

## 基本概念

### 三步走：吸积、碰撞、清空

| 阶段 | 说明 |
|------|------|
| **吸积** | 极小的尘埃先粘在一起，像滚雪球一样变大 |
| **碰撞** | 大到一定程度后，通过引力吸引彼此，发生大规模碰撞，合并成行星胚胎 |
| **清空** | 最终的行星胚胎扫荡自己轨道，清空附近所有其他物质，成为真正独立的行星 |

### 四步详细分解

**第一阶段：尘埃凝聚（0.1微米 → 1公里）**

- 原行星盘：99%气体 + 1%微米级尘埃
- 尘埃因静电力或化学粘性粘在一起，慢慢长成毫米、厘米大小的团块
- **最大谜题"径向漂移障碍"**：厘米到米级的团块会因气体阻力而快速螺旋坠入中心恒星，时间短到来不及继续长大
- 必须快速跨越这个尺寸区间，直接形成公里级的星子

**第二阶段：星子形成与吸积（1公里 → 1000公里）**

- 直径约1公里的天体叫**星子**
- 它们的引力已能施加影响，互相吸引、碰撞
- 一些质量较大的星子开始"**引力聚焦**"，像小型真空吸尘器一样
- 进入"**失控式吸积**"，迅速长到月球到火星大小
- 这个大小的天体叫**行星胚胎**

**第三阶段：巨型撞击（行星胚胎的最终合并）**

- 盘里剩下几十个月球到火星大小的胚胎
- 引力让彼此轨道变得混乱 → 极其暴力的巨型撞击时代
- **月球形成**：一颗火星大小的胚胎（忒伊亚）与原始地球发生巨大斜撞，溅出的物质形成月球
- 水星高密度：可能是剥掉外层的一次大撞击
- 天王星躺着自转：可能被侧撞了一次

**第四阶段：清空轨道与两条岔路**

| 岔路 | 位置 | 结果 |
|------|------|------|
| **类地行星（岩石行星）** | 雪线之内（靠近恒星） | 太热，水和甲烷只能以气体存在，抓不住 → 只形成岩石和金属构成的小个头行星（水星、金星、地球、火星） |
| **类木行星（气态巨行星）** | 雪线之外（远离恒星） | 水、甲烷、氨凝结成冰，固体原材料总量翻了几倍 → 行星胚胎长到地球质量10倍左右 → 引力直接吸引氢和氦气体，急剧膨胀成气态巨行星（木星、土星） |
| **冰巨星** | 形成得晚 | 盘中气体已大量消散，只裹上较少大气（天王星、海王星） |

> 雪线：越过它，水冰才能稳定存在。

### 类比：从一盒静电吸附的杂粮粉说起

| 阶段 | 场景 | 对应 |
|------|------|------|
| 第一阶段 | 静电吸附，面粉和糖霜颗粒粘成微小絮状团块 | 微米尘埃通过范德华力粘附，形成毫米到厘米级团块 |
| 第二阶段 | 絮状团块随风翻滚，碰撞粘住，变成小米球、花生米大小 | 星子阶段，达到公里级 |
| 第三阶段 | 某几颗"花生米"长得特别大，用引力把周围杂粮碎屑和小米球拉向自己，迅速长成葡萄 | **行星胚胎**，引力主导的失控式吸积 |
| 第四阶段 | 几颗互相竞争的葡萄，轨道混乱，发生巨大碰撞，合并成大核桃 | **巨型撞击**，行星胚胎合并 |
| 最后清场 | 大核桃吃掉周围碎屑，或把碎屑弹飞到盒子边缘或中心 | **清空轨道**，行星最终形成 |

## 历史演变

| 时代 | 理论 | 说明 |
|------|------|------|
| **18世纪** | **康德-拉普拉斯星云假说** | 太阳系从旋转的气体云中形成，行星是星云收缩时甩出的环凝聚而成。哲学雏形正确，但具体机制错误 |
| **20世纪初** | **遭遇危机** | 星云假说算出的太阳角动量不对，被抛弃，"潮汐假说"（行星是另一颗恒星接近太阳时撕出来的）盛行，但有严重物理缺陷 |
| **1940年代** | **魏茨泽克的涡旋模型** | 重新把行星形成拉回星云盘的正确道路上 |
| **1960-1970年代** | **萨夫龙诺夫的星子假说** | 苏联科学家萨夫龙诺夫用数学建立了微行星（星子）吸积形成行星的理论框架，是现代行星形成理论的**真正奠基人** |
| **1990年代至今** | **系外行星的革命** | 1995年发现第一颗类日恒星行星——飞马座51b（热木星），完全不在传统理论预期位置 → 加入"行星迁移"机制完善理论 |

## 前沿挑战与争议

### 1️⃣ 米级障碍（Meter-size barrier）

- 尘埃如何跨越从厘米到公里星子的"死亡谷"？
- 可能机制：尘埃颗粒在湍流中的"**压力陷阱**"里集体快速引力坍缩，直接跨过这个阶段
- **ALMA射电望远镜**已在一些原行星盘中拍到疑似这类过程的环和旋涡结构

### 2️⃣ 巨行星的迁移与秩序

- 按标准模型，**热木星不应该存在**
- 这揭示巨行星会大规模向内或向外迁移
- 迁移会摧毁沿途的任何小行星，类地行星还有机会形成吗？
- 太阳系巨行星似乎经历过一次大幅度外迁（尼斯模型），但**精确时间和幅度仍有争议**

### 3️⃣ 超级地球的形成之谜

- **超级地球**（质量介于地球和海王星之间）是银河系最常见的行星类型
- 但在太阳系里根本没有
- 它们是原地的"大号岩石行星"，还是迷你的海王星？
- 形成超级地球的典型盘环境，为何太阳系星云没有？

### 4️⃣ 直接观测行星形成中的过程

- 现在能在原行星盘中看到环和缝隙（可能是正在吸积的行星胚胎撬开的）
- 甚至直接拍到仍嵌在盘中、正在吸积物质的热"行星候选体"
- 把观测到的盘结构和行星形成模型一一对应，**是当前最令人激动的领域**

## 与《维度黎明》的关联

- **"吸积、碰撞、清空"三步**：可用于设计维度形成的三个阶段
- **失控式吸积**：某颗星子迅速长大成巨行星 → 可用于设计"维度黎明"中某个迅速崛起的势力
- **巨型撞击（月球形成）**：剧烈碰撞产生新天体 → 可用于设计维度碰撞后产生的新维度空间
- **雪线**：水冰稳定存在的分界线 → 可用于设计维度中"适宜生存条件"的分界
- **热木星不应该存在但存在**：理论无法预测的现象 → 可用于设计《维度黎明》中"超出预料的存在"
- **"尼斯模型"**：巨行星大幅外迁 → 可用于设计维度的大规模重组
- 关联：→ [[小行星带]]（星子→小行星）→ [[太阳]]（年轻的太阳有原行星盘）→ [[双子星]]（双星系统中行星形成可能不同）
