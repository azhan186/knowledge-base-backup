# 接触传染率：从公式到心理学

> 来源：用户输入（2026-05-23）
> 分类：技术参考 / 流行病学 / 行为心理学
> 标签：传染率, 流行病学, R₀, SIR模型, 行为心理学, 风险感知

---

## 核心公式：有效接触率

```
β = c × p
```

- **c（接触率）**：单位时间内，一个感染者平均接触的易感者人数
- **p（传播概率）**：单次接触中，病原体成功传播导致感染的概率

**干预逻辑：** 戴口罩/保持距离 → 降低 p；隔离/封锁 → 降低 c

---

## 两种传播模式

### 1. 密度依赖传播
- 假设：接触率 c 与总人口密度 N 成正比（c = kN）
- 传染率：β = kpN
- 适用：流感在学校、大城市等人群大规模随机混合场景

### 2. 频率依赖传播
- 假设：每个人单位时间内的接触总数是固定的（c 为常数）
- 传染率：β = cp/N（人多不增加总接触数，只是稀释）
- 适用：固定社交圈的性传播疾病或结构稳定的社区

---

## 核心应用：基本再生数 R₀

最简单的 SIR 模型中：

```
R₀ = β / γ
```

- γ：移除率（恢复或死亡速度的倒数，即感染期时长）
- 当 R₀ > 1 时，疫情扩散
- 控制目标：使 R_t < 1

---

## 影响接触传染率的复杂因素

- **社会网络结构**：超级传播者现象会极大扭曲传染率
- **行为反馈**：疫情严重时人们自发减少接触，c 动态下降
- **异质性**：不同年龄、职业群体的接触模式差异巨大

---

## 心理学视角下的 β 因子

> 接触传染率（β）本质上是一个受人类心理和行为深度调节的动态参数

### 风险感知与行为决策

| 心理机制 | 对 c/p 的影响 |
|----------|---------------|
| 健康信念模型 | 易感性感知下降 → c 上升 |
| 保护动机理论 | 反应效能低 → 防护放弃，p 维持高位 |
| 乐观偏差 | 低估计自身风险 → c 维持高位 |
| 心理距离 | 危机感弱 → c 维持高位 |

### 社会影响与群体心理

| 心理机制 | 对 c/p 的影响 |
|----------|---------------|
| 从众/社会规范 | 多数人戴口罩 → 少数人也跟进 → p 下降 |
| 心理抗拒 | 强制措施引发反抗 → c 上升 |
| 疫情疲劳/习惯化 | 防护行为退化 → c/p 恢复高位 |

### 认知捷径与进化烙印

| 心理机制 | 对 c/p 的影响 |
|----------|---------------|
| 习惯与默认动作 | 握手/拥抱等自动化行为 → c 上升 |
| 风险补偿 | 高防护措施 → 更多风险行为 → c 上升 |
| 行为免疫系统 | 对可见病原体回避 → c 下降；对无症状感染者无效 |
| 污名化与隐瞒 | 隐瞒病情 → 隐性传染链 → 传播扩大 |

---

## 心理学视角下的干预思路

1. **精准风险沟通**：用贴近自身的故事拉近心理距离，提升效能感
2. **正向社会规范**："保护他人"成为身份信号，减少抗拒
3. **行为助推**：重新设计环境（如自动洗手液、地面距离标记），让安全行为成为默认选项

---

> 归根结底，接触传染率是一个心理-行为参数。要影响它，就得理解人心如何感知风险、形成习惯，并深受周围环境影响。
