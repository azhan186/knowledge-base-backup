## 一、核心定义：什么是光伏发电？

**光伏发电** = 利用太阳光直接产生电能的技术

**光生伏特效应**：
- **光**：太阳光由无数微小能量粒子"光子"组成
- **伏特**：电压的单位，代表电的"压力"
- **效应**：光子照射到硅等特殊材料上，把电子"撞"出来，形成电流

核心设备：太阳能电池 → 封装成光伏组件（太阳能板）

## 二、生活类比：一片"电子雨林"

| 类比 | 说明 |
|------|------|
| 普通植物 | 光合作用，把阳光转化为化学能（养分） |
| 光伏板 | 把光子"捕捉"住，转化成电能 |

阳光越强，"电子养分"越多。类似太阳能小摆件里的小硅片——无噪音、无转动部件，阳光照上去电就产生了。

## 三、历史演变

| 时间 | 里程碑 |
|------|--------|
| 1839年 | 贝克勒尔发现"光生伏特效应"，当时无法应用 |
| 1954年 | 美国贝尔实验室研制出第一块实用硅太阳能电池（效率约6%） |
| 1958年 | 美国"先锋1号"卫星首次用光伏供电，光伏进入太空 |
| 1970-90年代 | 石油危机推动，偏远地区通讯站、计算器等民用市场 |
| 2000年至今 | 全球规模化，中国将光伏成本降低近90%，成为全球最便宜新能源之一 |

## 五、主要争议与前沿挑战

### 主要争议

| 争议 | 内容 |
|------|------|
| 制造的环保账 | 多晶硅提纯曾高耗能高污染，但1-2年寿命期发的清洁电即可抵消生产能耗 |
| 退役后回收 | 约25-30年寿命，未来大规模"退役潮"，含少量铅等重金属需处理 |
| 土地利用冲突 | 大型电站占地与农业/生态矛盾，解决方案"农光互补""渔光互补" |
| 电网"鸭子曲线" | 白天发电多，傍晚骤降而用电高峰刚好到来，需配合储能解决 |

### 前沿挑战

| 挑战 | 内容 |
|------|------|
| 效率天花板 | 理论单结硅电池效率极限约29.4%，已逼近极限；叠层电池（钙钛矿/硅）理论效率可超40% |
| 储能的"心脏" | 配合安全、低廉、长寿命储能系统，使光伏成为稳定主力电源 |
| 智能与柔性 | 可调度、与其它能源智慧协同；薄膜电池可弯曲轻便，应用场景更多 |

---

> 现代光伏板在其20多年寿命中发的清洁电力，只需1-2年就能抵消其生产全过程的能耗。
