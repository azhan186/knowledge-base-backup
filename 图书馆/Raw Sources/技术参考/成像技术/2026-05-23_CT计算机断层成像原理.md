## CT是什么？

**计算机断层成像（Computed Tomography）**

利用X射线束从多个角度穿过人体，由探测器测量衰减后的射线强度，通过计算机重建任意方向横断面图像。

- **发明者**：亨斯菲尔德（Godfrey Hounsfield）和科马克（Allan Cormack）
- **荣誉**：1979年诺贝尔生理学或医学奖
- **与MRI最大区别**：CT使用X射线（有电离辐射），MRI使用磁场和射频（无电离辐射）

---

## 基本物理原理：X射线衰减

遵循**朗伯-比尔定律**：
```
I = I₀ × e^(-μd)
```

- I₀：入射X射线强度
- I：穿过物质后出射的X射线强度
- d：物质厚度
- μ：线性衰减系数（与物质密度、原子序数、X射线能量相关）

**示例：**
- 骨骼：密度高、原子序数大，μ大，X射线大量吸收，出射I很小
- 肺：充满空气，密度极低，μ极小，X射线几乎畅通无阻

## 图像重建：从投影到断层图像

### 1. 数据采集：多角度投影

- 现代CT采用宽扇形束和多排探测器，X射线管围绕人体连续旋转
- 形成**螺旋扫描**，一次屏气即可完成胸腹部扫描
- 每个角度的投影 = 该方向上线积分总和

### 2. 图像重建：求解每个体素的μ值

核心工具：**雷登变换（Radon Transform）**

| 重建方法 | 说明 |
|----------|------|
| 简单反投影 | 会产生"1/r"模糊，图像中心过度叠加，边缘发散 |
| 滤波反投影 | 先高通滤波（斜坡滤波器），再反投影——黄金标准 |
| 迭代重建 | 主流方法，模拟正投影与实际比对修正，可降低辐射剂量并抑制金属伪影 |

## CT值（亨氏单位，HU）

```
CT值(HU) = 1000 × (μ_组织 - μ_水) / μ_水
```

| 组织 | CT值范围 |
|------|----------|
| 水 | 0 HU |
| 空气 | -1000 HU |
| 脂肪 | -100 ～ -50 HU |
| 脑白质 | 25 ～ 35 HU |
| 脑灰质 | 35 ～ 45 HU |
| 急性出血 | 60 ～ 90 HU |
| 致密骨 | > +1000 HU |

**窗宽/窗位技术**：人眼只能分辨约16个灰阶，CT值有2000多阶差，通过窗宽窗位调整可清晰展示不同密度组织。

## 设备技术与核心应用

**现代CT技术：**
- **多排探测器**：320排、双源CT，时间分辨率极高，可在一个心动周期内完成心脏冠脉成像
- **能谱/双层探测器CT**：不同能量X射线衰减差异→物质分离（碘/钙/尿酸）

**核心优势：**
- 速度快（秒级完成全身扫描）
- 骨骼与钙化显示极佳
- 急性出血敏感
- 肺部HRCT无可替代
- CTA（血管成像）精准评估狭窄/栓塞

**王牌应用：**
- 急诊创伤、脑卒中（先排除出血）
- 骨折、关节病变、尿路结石
- 肺结节、间质性肺病
- 冠状动脉、主动脉、肺动脉成像

---

## 局限性与风险

- **电离辐射**：关键安全考量，遵循"合理达到的尽可能低"原则
- **软组织对比度差**：观察脑白质、半月板韧带等不及MRI
- **含碘对比剂风险**：过敏反应、对比剂肾病
- **伪影**：金属植入物产生条索状伪影；射线硬化效应产生骨-脑交界伪影

## CT vs MRI 对比

| 特性 | MRI | CT |
|------|-----|-----|
| 辐射 | 无 | 有（X射线） |
| 核心原理 | 氢质子共振/弛豫 | X射线衰减系数 |
| 最佳观察 | 软组织、神经、关节、脑白质 | 骨骼、钙化、肺、急性出血 |
| 图像基础 | T1/T2/质子密度等参数 | 单一密度参数（CT值） |
| 扫描时间 | 长（数十分钟） | 极快（秒级） |
| 安全禁忌 | 心脏起搏器等铁磁植入物 | 无绝对磁场禁忌，但有辐射与碘剂风险 |

---

> CT将X射线的穿透力与计算机运算深度结合，以极快速度提供高精度人体密度解剖图，与MRI互为补充，是临床诊断中最重要的两大断层成像手段。
