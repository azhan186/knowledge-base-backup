# 股票常用词汇解释

**日期：** 2026年6月5日
**分类：** 财经 / 投资 / 股票词汇
**来源：** 原创/整理

---

## 一、关于「股票本身」

### 总股本 / 流通股本
- **总股本**：公司一共切成多少份（总股数）
- **流通股本**：能在市场上自由买卖的份数（部分由大股东锁定）

### 总市值
- 总市值 = 股价 × 总股本
- 例：股价60万 × 5股 = 300万总市值

### 蓝筹股 / 成长股 / 周期股
| 类型 | 特征 |
|------|------|
| 蓝筹股 | 规模大、赚钱稳、老牌公司，不太可能倒闭 |
| 成长股 | 现在赚得少，增速极快，市场赌未来 |
| 周期股 | 业绩随经济好坏剧烈波动（航运等） |

## 二、关于「价格与估值」

### 市盈率（PE）
- PE = 股价 / 每股盈利
- 含义：按现在赚钱速度，多少年回本
- 注意：高PE不一定是坏事（隐含高增长预期）；低PE不一定是好事（盈利可能一次性）

### 市净率（PB）
- PB = 股价 / 每股净资产
- 净资产 = 现金 + 厂房 + 存货 - 全部负债 = 清算法还剩的价值
- PB < 1：破净，股价比"砸锅卖铁"还便宜，常见于银行/钢铁等重资产行业

### 股息率
- 股息率 = 每股年度分红 / 股价
- 含义：每年现金回报相对于股价的收益率
- 陷阱：股价暴跌时股息率会变高，但如果分红金额下一年被砍，实际回报下降

### 除权除息
- 分红后股价强制减去每股分红额 → 除息
- 例：股价60元，每股分红1元 → 除息日后开盘基准价变为59元
- 总财富不变：股票市值降了，但拿到1元现金
- **填权**：分红后股价涨回原价，即赚到了那1元分红对应的未来盈利增长能力

## 三、关于「交易现场」

### 开盘价 / 收盘价
- **开盘价**：集合竞价成交量最大的价格
- **收盘价**：A股下午3点前最后一笔成交价

### 涨停 / 跌停
- A股涨跌幅限制：主板一般 ±10%
- 涨停：股价涨到当日最高限，买盘封死无法再买
- 跌停：股价跌到最低限，无法卖出
- 作用：人为价格暂停，极端行情下放大恐惧和贪婪

### 成交量 / 换手率
- **成交量**：当日成交股数或金额
- **换手率** = 成交量 / 流通股本
  - 3% = 活跃
  - 超过10% = 极高，分歧巨大，大量人进出

### 一手
- A股最小购买单位 = 100股
- 港股因公司而异，可能是100、500或1000股

## 四、关于「市场整体」

### 牛市 / 熊市 / 猴市
| 市场 | 特征 |
|------|------|
| 牛市 | 长期普涨，大家乐观 |
| 熊市 | 长期普跌，大家极度悲观 |
| 猴市 | 上蹿下跳，方向不明，反复震荡 |

### 指数
- 按选样规则选一批代表性股票，计算一个数字反映整体温度
- 例：上证50（大盘蓝筹）、中证1000（小盘成长）
- 无法直接买卖指数，但可买卖跟踪指数的指数基金
