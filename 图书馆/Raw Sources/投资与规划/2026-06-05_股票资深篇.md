## 核心认知升级

**股票的本质不是所有权，是"预期差"的定价**

公式升级版：**超额收益 = 你的认知 – 市场的共识**

- 市场共识已100%反映在当前股价中（财报/新闻/研报全是价格的一部分）
- 你的认知必须是非共识且正确的：看到别人没看到的信息，或对同一信息做出更接近真相的解读
- 研究股票的最高境界 = 研究"市场共识中哪里有裂缝"

## 高阶定价模型：DCF现金流折现

**企业终极价值 = 未来所有年份自由现金流的总和，按折现率换算回今天**

公式：V = C₁/(1+r) + C₂/(1+r)² + C₃/(1+r)³ + …

### DCF告诉你的三件事

1. **价值载体是现金流，不是利润** — 账上利润高但变成积压库存 → 真实价值远低于会计利润
2. **增长只在于"超额增长"** — 投入每分钱必须赚回远超资本成本的回报，否则扩张就是毁灭价值
3. **久期决定估值范式** — 现金流越远，估值对利率越敏感 → 加息对高估值成长股是灾难

## 市场微观结构

- **流动性是海市蜃楼**：极端时刻流动性瞬间蒸发，"市场价"卖出可能以极差价格成交
- **两种对手盘**：
  - 知情交易者（有信息优势）
  - 噪音交易者（被迫交易，如指数基金调仓、对冲基金平仓）
- 顶尖交易者核心技能：分辨价格波动是信息驱动还是强制性资金流动驱动，并利用后者造成的极端偏差获利

---

## 风险的本质：不是波动，是脆弱性

| 概念 | 定义 |
|------|------|
| 脆弱性 | 遭受外部冲击时系统/组合会崩溃 |
| 反脆弱 | 外部冲击反而让它更强/从中获益 |

- 高杠杆+现金流紧绷 = 极度脆弱
- 手握大量净现金+行业寒冬能反向扩张 = 反脆弱
- **尾部风险对冲**：持有稳健资产 + 用极少比例资金布局极端事件期权

## 思维框架

### 叙事经济学
- 市场交易的不是事实，是关于事实的"叙事"
- 分析叙事在哪个阶段：
  1. 无人相信（机会）
  2. 逐步共识（趋势）
  3. 狂热成流行语（危险极近）

### 边际定价者
- 股价由最迫切想成交的一小撮人决定
- 目标：理解边际定价者动机，评估价格扭曲程度，在足够安全的位置接手
