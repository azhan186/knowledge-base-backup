# 量子纠缠（深入篇）

一、纠缠的精确定义：从叠加态到贝尔态
最干净的纠缠系统是两量子比特（量子比特）系统。每个量子比特可以是|0⟩（比如自旋向下）或|1⟩（自旋向上）。
一个普通的叠加态是：
|ψ⟩ = α|0⟩ + β|1⟩
这只是一个粒子的私有状态，不涉及纠缠。
现在考虑两个粒子（A和B）。它们的联合状态有四种可能：|00⟩, |01⟩, |10⟩, |11⟩。它们可以处于一个乘积态，例如：
|ψ⟩ = |0⟩_A ⊗ |1⟩_B
这表示 A 肯定是 0，B 肯定是 1。你能独立描述 A 和 B，它们没有纠缠。
但如果状态是：
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2
这就是一个贝尔态，四个最大纠缠态之一。它不能写成两个独立粒子状态的乘积。无论你用什么数学技巧，都无法单独说 A 是什么、B 是什么。整个系统的信息，不可分割地存储在两个粒子的关联中。这就是量子纠缠的严格数学定义：一个无法因式分解为子系统状态的纯态。
二、贝尔不等式破坏的精髓：为什么这比"手套"强？
在经典手套图景中，如果 A 是左手，B 就是右手，这是预先存在的属性。贝尔不等式提供了判决性测试。
想象 Alice 和 Bob 各自随机的选择两种测量方式之一（比如测偏振方向是 0° 还是 45°）。他们记录下各自的+1或-1结果。然后计算一个组合量 S，由四种测量组合的概率期望值构成。
- 任何局部隐变量理论（经典关联）都必须满足：|S| ≤ 2。 这就是贝尔不等式（CHSH 形式）。
- 量子力学预言，对于最大纠缠的贝尔态，S 可以达到 2√2 ≈ 2.828。
实验上，这个上限被反复突破。这意味着什么？在测量之前，单个光子的偏振方向不仅未知，它根本就不存在。 你不能用"箱子里早有确定的左手/右手"来解释，因为如果有，它必然服从不等式。大自然的随机性是根本的，且非局域的。相距甚远的两个测量结果，以一种超越任何可能事先约定信号的方式相互配合。
三、纠缠的度量：一个宝贵的资源
纠缠不是一个有或无的性质，它可以有"多少"。在量子信息论中，纠缠被看作一种资源，像能量一样可以被量化、转化、消耗。
纠缠熵
对一个由 A 和 B 组成的纯态 |ψ⟩，你把 B 部分的信息全部舍去（求迹），得到 A 的密度矩阵 ρ_A。如果 |ψ⟩ 是乘积态，ρ_A 代表一个纯态，其冯·诺依曼熵 S(ρ_A) = 0。没有纠缠。
如果 |ψ⟩ 是贝尔态，ρ_A 是完全混合态（50% |0⟩，50% |1⟩），此时 S(ρ_A) = 1（最大）。你失去了关于 A 个体的一切确定性信息，但在 AB 的关联中拥有全部信息。纠缠熵，度量了当你看不到另一半时，你失去了多少确定性。 这个熵，直接就是纠缠的量化单位"ebit"。一个贝尔态就是 1 ebit 的纠缠。
纠缠蒸馏与稀释
你有一大堆弱纠缠的、不纯的量子对。通过一种称为纠缠蒸馏的协议，你和 Bob 只做局部操作和经典通信（LOCC），就可以从很多对质量差的纠缠对中，提取出少量、近乎完美的最大纠缠贝尔对。反过来，你也可以把一个贝尔对稀释成很多弱纠缠对。在渐近情况下，这两者的转化效率相同，其比率定义了纠缠的代价。这证明纠缠资源在 LOCC 下遵循类似热力学的定律。
四、纠缠赋予的超能力：超越经典的通信与计算
为什么纠缠是核心资源？因为它能完成经典无法完成的任务。
1. 量子隐形传态（更精确版）
目标：Alice 要将一个未知量子态 |ψ⟩_1 = α|0⟩ + β|1⟩ 传给 Bob，但她不能直接发送粒子，只能打经典电话。
前提：他们共享一副魔法手套，即一个贝尔态 |Φ⁺⟩_AB。
流程：
- Alice 将她要传的粒子 1 和她的那半副手套 A 做一个联合测量（贝尔基测量），得到 2 比特的经典结果（00,01,10,11）。
- 这个测量同时导致 Bob 手里的半副手套 B 瞬间塌缩成四种状态之一，每一种都与 |ψ⟩ 差一个固定的幺正操作。
- Alice 用经典电话告诉 Bob 这 2 比特结果。
- Bob 根据这 2 比特信息，对他的粒子 B 施加对应的幺正操作（I, X, Y, Z），粒子 B 的状态就被完全矫正为 |ψ⟩。
核心点： 传送过程中，最初的 |ψ⟩ 被测量破坏了（不可克隆定理），但 Alice 也没有知道 |ψ⟩ 是什么。2 比特经典信息必须被传递，所以没有超光速。纠缠此时扮演的是"量子通道"的角色，它预先铺设了一种非局域关联，允许用经典比特传送量子态。
2. 量子超密编码
反过来，利用 1 个贝尔对，Alice 可以通过只发送 1 个量子比特 给 Bob，来传递 2 比特的经典信息。这颠覆了经典信道容量的极限。前提仍是预先共享的纠缠。
五、纠缠的结构：单配性与多体纠缠
纠缠不是共享越多越好。它有一个极强的排他性：纠缠单配性。
如果 A 和 B 处于最大纠缠态（纠缠熵为 1），那么 A 与任何第三个粒子 C 的纠缠必须为零。纠缠的分享是零和的。这直接量化了"专一性"。
这也引出了多体纠缠的丰富结构。三个量子比特的纠缠有两种不等价类：
- GHZ态 (|000⟩ + |111⟩)/√2：这是一种"薛定谔猫"态的极致，如果测量一个粒子发现是 0，另两个也必定是 0。但只要你部分求迹掉一个粒子，剩下两体之间的纠缠完全消失。它的纠缠是整体性的。
- W态 (|001⟩ + |010⟩ + |100⟩)/√3：即使丢掉一个粒子，剩下的两体之间仍保留着有限纠缠。W 态的纠缠更"坚固"。
这种分类表明，纠缠可以编织出极其复杂的多体网络，形成所谓的张量网络结构，这正是现代多体物理和全息引力的数学基础。
六、退相干：纠缠的头号天敌
现实世界中，为什么我们看不到宏观物体的纠缠？为什么建造量子计算机这么难？因为纠缠极其脆弱。
当一个系统与环境发生任何不可控的相互作用时，环境就像无数个窥视的观察者，不断地、无意识地"测量"系统，将其中的纠缠信息泄露并稀释到环境的巨大自由度中。这个过程就是退相干。它摧毁了量子系统内部的相干性，使纠缠退化。这是从量子世界过渡到经典世界的主要机制，也是量子信息处理必须克服的终极障碍。
