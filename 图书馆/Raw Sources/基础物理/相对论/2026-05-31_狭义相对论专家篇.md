# 狭义相对论（专家篇）

一、理论基石：作为庞加莱群表示的物理世界
狭义相对论的公理可以凝结为一句话：物理定律在庞加莱群 ISO(3,1) 的变换下是协变的。 这个群是闵可夫斯基时空 R^{1,3} 的所有等度规映射，包含时空平移和洛伦兹变换。洛伦兹群 O(3,1) 保持不变量 η_{μν}x^μx^ν = -c²t² + x²。
你的任务不再是接受钟慢尺缩，而是理解为什么这些效应必须存在：因为 洛伦兹变换是双曲旋转，它混合了时间与空间坐标，但保持时空间隔 ds² = η_{μν}dx^μdx^ν 绝对不变。同步性的相对性，不过是你选取不同时空切片的结果——类空超曲面的倾斜。
真正使狭义相对论成为量子场论脊椎的是它的李代数 so(3,1) ≃ sl(2,C)。这个复化代数告诉我们，洛伦兹群的有限维不可约表示由两个 su(2) 的角动量标记：(j₁, j₂)。这就是为什么粒子世界出现了自旋：
- 标量场（0,0）：希格斯粒子
- 左手外尔旋量（1/2,0）：中微子（假设无质量）
- 右手外尔旋量（0,1/2）：反中微子
- 狄拉克旋量（1/2,0）⊕（0,1/2）：电子
- 规范矢量场（1/2,1/2）：光子、胶子
狄拉克方程不是任意写下的；它是唯一能在（1/2,0）⊕（0,1/2）空间中给出正定概率密度、一阶时间导数的洛伦兹协变方程。反物质的预言，根源是方程在洛伦兹群的完备表示下必须同时包含正负能解——正是 CPT 定理在单粒子层面的映射。在专家眼中，不存在"狄拉克方程导出反物质"这样的惊喜，只有"洛伦兹对称性结合量子力学排斥了其他可能"的必然。
二、因果性的公理地位：为什么超光速不是速度问题
超光速被禁绝非技术限制，而是几何禁忌。在闵可夫斯基时空中，每一个事件都有一个绝对的光锥，将时空分解为类时未来、类时过去和类空别处。如果你能发送超光速信号，就意味着某个惯性系会看到该信号逆时间流动——接收者在发射者过去的光锥内。这会在 CTC（闭合类时曲线） 存在的时空中产生逻辑悖论。
量子场论将此纳入公理：微观因果性 要求类空间隔的场算符对易（或反对易），即 [ϕ(x), ϕ(y)]₊ = 0 for (x-y)² > 0。这确保任何测量都不能以超光速传递信息。量子纠缠之所以不违反此律，正是因为测量结果的统计关联无法本地检测——任何可以提取信息的序列必须借助经典亚光速信道。贝尔不等式检验强化的不是"幽灵"，而是 量子态的非局域实在性 与 操作信息的光速限制 之间的完美共处。
三、广义相对论的引子：局域化洛伦兹对称性
为什么一定要有广义相对论？专家看的是 对称性的局域化。狭义相对论的全局庞加莱对称性无法与引力共存，因为能量-动量张量本身作为引力源，在时空中各点不同。正如把全局 U(1) 对称性局域化必然引入规范场（电磁场），把全局洛伦兹对称性局域化，就必然引入自旋联络和时空曲率。
用标架场（tetrad）e^a_μ(x) 的语言：在每一点建立起一个局部闵可夫斯基空间，局域洛伦兹变换 Λ^{ab}_μ(x) 保持局部度规 η_{ab}。要求理论在此局域变换下不变，最小耦合命令将普通导数替换为协变导数 D_μ = ∂_μ + ω_μ^{ab}Σ_{ab}，其中 ω_μ^{ab} 就是自旋联络，Σ_{ab} 是洛伦兹群的生成元。爱因斯坦-希尔伯特作用量在这个语境下是 (1/16πG)∫e ∧ e ∧ R，完全是规范理论的构造，只不过规范群是洛伦兹群，且作用量不含度规的动力学，而是由标架和联络携带着几何。
这就是为什么引力量子化如此困难：它涉及对时空本身的洛伦兹对称性做量子涨落。普通量子场论依赖一个刚性的闵可夫斯基背景来定义粒子和因果结构，而量子引力却要去量子化这个背景。
四、冲击与萌发：洛伦兹对称性会破缺吗？
在普朗克尺度（10⁻³⁵ m），量子涨落可能让时空变成"泡沫"，闵可夫斯基的平滑图像崩塌。多种量子引力方案都暗示 洛伦兹不变性可能只是一种低能涌现：
- 弦论：非可换几何背景下，时空坐标本身成为非对易算符，[x_μ, x_ν] = iθ_{μν}，洛伦兹对称性显式破缺，但可以通过希格斯机制使其恢复。
- 圈量子引力：面积和体积算符有离散谱，时空间隔在微观层次可能无法定义，但相对论原则如何以某种"变形"的形式存活，是双狭义相对论（DSR）探索的方向——它引入一个与普朗克能量相关的不变量，修改动量空间的庞加莱代数。
- 超对称与额外维：高维庞加莱群 ISO(1,d-1) 在紧化后遗留下四维洛伦兹对称性，某些方案下洛伦兹破缺以矢量或张量场非零真空期望值的形式出现。
实验上，对洛伦兹破坏的搜索已进入极高精度：伽马暴光子到达时间的色散测量将洛伦兹破缺尺度逼到普朗克能标之上 10¹⁹ GeV；引力波 GW170817 与电磁对应体的同时性将引力波速度与光速的差异限定在 10⁻¹⁵ 以内；原子钟比对、中微子振荡、宇宙微波背景偏振等处处设卡。狭义相对论目前仍是零实验结果的理论赢家。
