## 二、冲

### 六冲（方位相对）
子午、丑未、寅申、卯酉、辰戌、巳亥

冲主变动、突破。若冲忌神反为吉。

### 冲的细分
| 类型 | 地支 | 特性 |
|------|------|------|
| 生冲 | 寅申巳亥 | 四生地藏干复杂，冲战多致两败俱伤 |
| 败冲 | 子午卯酉 | 四专气力量纯粹，胜负取决于双方旺衰强弱 |
| 库冲 | 辰戌丑未 | 冲开墓库则所藏之物（财官印）能量被引发 |

> 《滴天髓》："支神只以冲为重，刑与穿兮动不动"

### 冲的吉凶总纲
- **冲去忌神** → 吉
- **冲去喜用神** → 凶

## 三、刑

### 三刑
| 类型 | 地支 |
|------|------|
| 无礼之刑 | 子卯 |
| 无恩之刑 | 寅巳申 |
| 恃势之刑 | 丑未戌 |
| 自刑 | 辰辰、午午、酉酉、亥亥 |

### 各刑应事范围
- **寅巳申（无恩之刑）**：冷酷薄情、恩将仇报，易引发官非、血光
- **丑戌未（恃势之刑）**：体弱多病、夫妻不和，因利益引发信任危机
- **子卯（无礼之刑）**：风流淫乱、色情灾祸，为人处事欠圆融、疏于礼数
- **辰午酉亥（自刑）**：自我矛盾、心性纠结、自寻烦恼，严重时精神压抑或情绪困扰

## 四、害（穿）

### 六害
子未、丑午、寅巳、卯辰、申亥、酉戌

害主暗中阻碍、小人。力量较弱但隐蔽。

---

## 优先级（力量强弱）

1. **合**（尤其三合、三会）
2. **冲**
3. **刑**
4. **害**

> 若紧贴（月日支相邻），即使非合也优先论其作用

---

## 核心原则

**用神被冲则凶，忌神被冲则吉；三刑破吉，三刑助吉。**

## 实战应用

### 婚姻判断（女命）
- 日主坐长生且四柱无刑冲 → 婚姻和谐（夫星不旺仍吉）
- 日坐冠带遇刑冲 → 易婚变
- 子卯无礼之刑 + 辰午酉亥自刑 → 多应验于感情风波或六亲矛盾
- 日支月支子卯相刑再刑入羊刃 → 婚姻波折重重
- 自刑 → 多因自我矛盾、猜疑心重引发感情困扰

### 格局分析
- 乙木日主生于午月 → 刑冲破坏食神生财格局
- 戊土日主寅月七杀当令 → 忌见刑冲
- 丑戌未恃势之刑：土为喜用神 → 刑冲土旺反为吉兆，助事业官运

### 宫位与应事
| 冲何宫位 | 应事 |
|----------|------|
| 年柱 | 祖上、父母变动 |
| 月令（提纲） | 家庭、工作环境变动 |
| 日支（夫妻宫） | 婚姻、合作动荡 |
| 时柱（子女宫） | 子女、投资变化 |

### 运势预测（刑冲流年点断法）
影响层级：原局 → 大运 → 流年 → 流月

- **时柱帝旺遇刑冲** → 晚年孤独
- **月柱临官逢刑冲** → 中年事业波动

---

## 实战要点

- **无绝对吉凶**：关键看作用对象是喜用神还是忌神
  - 喜用神被冲 → 动荡失财
  - 忌神被冲 → 去灾得利
- **合而不化** → 只是"合绊"，能量受阻
- **刑冲并见** → 需综合判断气势与调候
