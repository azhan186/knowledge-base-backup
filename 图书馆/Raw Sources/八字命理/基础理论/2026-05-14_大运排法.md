# 大运排法

> 来源：公开资料整理 | 收录时间：2026-05-14

---

## 核心三步

> 年干定顺逆，节气算起运，月柱排干支

---

## 第一步：定顺逆

### 判断规则
| 年干 | 男性 | 女性 |
|------|------|------|
| 甲丙戊庚壬（阳干） | 顺排 | 逆排 |
| 乙丁己辛癸（阴干） | 逆排 | 顺排 |

- **阳男阴女**：大运顺排，从月柱往后数
- **阴男阳女**：大运逆排，从月柱往前数

---

## 第二步：算起运岁数

### 核心原则
- 以**节**为基准（不是中气），算出生日到最近节气之间的天数
- 顺排：数从出生日到**下一个节**的天数
- 逆排：数从出生日到**上一个节**的天数

### 换算公式
| 单位 | 折算 |
|------|------|
| 3天 | = 1岁 |
| 1天 | = 4个月 |
| 1时辰 | = 10天 |

### 实例

**男命（阳男顺排）**
- 出生：2026年5月14日
- 下一个节：芒种（6月5日）
- 天数：5月14日 → 6月5日，约22天
- 22 ÷ 3 ≈ 7.33 → **7岁4个月起运**
- 第一步大运：7-16岁

**女命（阳女逆排）**
- 出生：2026年5月14日
- 上一个节：立夏（5月5日）
- 天数：5月5日 → 5月14日，9天
- 9 ÷ 3 = 3 → **3岁起运**
- 第一步大运：3-12岁

---

## 第三步：排大运干支

### 顺排（阳男阴女）
月柱壬辰，下一干支为癸巳，依次顺排：
> 癸巳 → 甲午 → 乙未 → 丙申 → 丁酉 → 戊戌 → 己亥 → 庚子 → …

### 逆排（阴男阳女）
月柱壬辰，倒推前一干支为辛卯，依次逆排：
> 辛卯 → 庚寅 → 己丑 → 戊子 → 丁亥 → 丙戌 → 乙酉 → 甲申 → …

---

## 关键注意事项

### 1. 节气必须咬死
- 查万年历确定"节"的准确时刻
- 例：立夏是5月5日15时，若孩子在14时出生，未到交节时刻，月份仍算**辰月**
- 大运排法完全不同

### 2. 上运前看月柱
- 小孩在起运岁数前，运势主要受**月柱**影响（小运/童运）

### 3. 十年一大运
- 每个大运干支管**十年**
- 例：7岁起癸巳运 → 管7-16岁
- 大运天干主**前五年**，地支主**后五年**，需整体参看

### 4. 岁运并临
- 大运和流年干支完全相同，是人生**吉凶重要关口**

### 5. 交节当天出生
- 按交节时辰**精确排布**（前后的月份和大运可能不同）
