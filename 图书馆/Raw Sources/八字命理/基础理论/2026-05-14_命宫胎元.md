# 命宫与胎元

> 来源：公开资料整理 | 收录时间：2026-05-14

---

## 命宫概述

命宫是除原局四柱外的关键一柱，被称为**"第五柱"**，可补原局之不足。

---

## 一、命宫起法（主流气数法）

### 操作步骤

1. **定寅为正月**：手掌上"寅"位当作正月，无论年柱为何，固定此位为起点
2. **顺数至生月**：农历几月，就从寅位顺数几格，生月落在某地支
3. **逆数生时至卯**：将时支放在生月地支上，逆时针数到"卯"位，该位地支即为命宫地支
4. **推算天干**：用"五虎遁"从年干查月干，查到命宫地支对应的月干天干

### 实例

当前时间：2026年5月14日（农历三月廿八），巳时

1. 正月建寅 → 顺数三月：寅(1) → 卯(2) → 辰(3)，生月落**辰**位
2. 辰位起巳时 → 逆数：辰(巳) → 卯(辰) → 寅(卯) → 命宫地支落**寅**
3. 丙午年，正月庚寅 → 命宫天干为**庚**
4. 结果：**庚寅**

## 二、另一种古法（中气法）

- 以出生月所交的"中气"为界（如正月立春至雨水为寅月）
- 余法同气数法，步骤完全一致

## 三、胎元

### 概念
胎元与命宫合称"三垣"，代表生命孕育之始。

### 计算口诀
**月柱天干顺推一位，地支顺推三位**

### 实例
月柱壬辰：
- 天干：壬 → 顺推一位 → **癸**
- 地支：辰 → 顺推三位（辰→巳→午→未）→ **未**
- 胎元：**癸未**

### 注意事项
- 胎元天干地支均不可与日时柱冲克
- 若有刑冲，则主根基不稳

---

## 四、关键实战细节（忽略易断错）

### 1. 真太阳时是前提
- 出生时辰必须换算成当地时间，而非统一北京时间
- 例：上海上午10:30 → 真太阳时约10:41，仍在巳时
- 例：如果在新疆，此时可能还没到辰时
- 时辰一错，命宫全错
- 计算方法：平太阳时 + 时差

### 2. 子时要不要拆分
- 传统命理分早子时（00:00-01:00）和晚子时（23:00-00:59）
- 晚子时排盘：时柱天干按次日日柱推算，地支仍属当日
- **此点争议大**：建议先按常规不分早夜的算法来定命宫
- 排紫微斗数盘时尤其要小心

### 3. 节气过宫要咬死
- 命宫推算核心是"生月"，月令跟着节气走
- 例：2026年农历三月廿八，出生在阳历5月5日立夏之前 → 月柱壬辰
- 若出生在立夏之后 → 月柱变成癸巳，命宫起算点变为四月（巳），结果完全不同
- **节气是分界，不是农历月份**

### 4. 气法排命宫天干
- 得到命宫地支后，用"五虎遁"定天干，看的是**年干**
- 口诀："甲己之年丙作首"，从寅月开始按干支顺序排
- 不能直接套用月柱天干

## 五、命宫的作用与意义

### 核心作用
1. **神煞系统依据**：命宫是许多神煞（如将星、天乙贵人等）的安宫依据
2. **补原局不足**：八字缺火调候，命宫恰好丙午，就是很好的补充
3. **根基判断**：命宫与日柱或时柱天克地冲（如日柱甲子、命宫庚午）→ 基石有不稳

### 注意事项
- **命宫是辅助柱，不是日主**
- 论命主体还是八字原局
- 命宫再好也无法完全化解原局冲克
- 主要在流年、大运与命宫产生作用时显现力量

## 六、身宫（重要补充）

身宫与命宫并称"三垣"，代表后天发展，主35-40岁后的运势。

### 身宫起法
1. 安好命宫地支后，从该地支位起正月
2. 顺数到出生月
3. 从该位起初一
4. 逆数到出生日
5. 最终所在位置就是**身宫地支**
6. 天干同样用五虎遁
