# 十神体系

> 来源：百度百科 | 收录时间：2026-05-14

## 十神定义

十神是以**日干**为中心，根据五行生克关系划分的十个专有名词：

| 十神 | 五行关系 | 阴阳 |
|------|----------|------|
| 比肩 | 同我 | 偏（同性） |
| 劫财 | 同我 | 正（异性） |
| 食神 | 我生 | 偏（同性） |
| 伤官 | 我生 | 正（异性） |
| 偏财 | 我克 | 偏（同性） |
| 正财 | 我克 | 正（异性） |
| 七杀 | 克我 | 偏（同性） |
| 正官 | 克我 | 正（异性） |
| 偏印 | 生我 | 偏（同性） |
| 正印 | 生我 | 正（异性） |

**同性为偏，异性为正**——这是判断偏正的核心原则。

## 十神分类

### 正神 vs 偏神
- **正神**：劫、伤、才、官、印
- **偏神**：比、食、财、杀、枭

### 相生循环（正神）
劫 → 伤 → 才 → 官 → 印 → 劫

### 相生循环（偏神）
比 → 食 → 财 → 杀 → 枭 → 比

> 口诀：劫伤才官印，比食财杀枭

## 生克法则

### 同性生克力大，异性生克力小

**相生**：正偏财生官杀 → 官杀生印枭 → 印枭生日主比劫 → 比劫生食伤 → 食伤生正偏财

**相克**：正偏财克印枭 → 印枭克食伤 → 食伤克官杀 → 官杀克日主比劫 → 比劫克正偏财

## 十神功能（详细）

### 正官
- **关系**：克制日干，阳干见阴干克，或阴干见阳干克
- **职能**：善意之管，遵从政府与法律管束，一般以吉神论
- **扶抑能力**：卫财，生印，抑身，制劫
  - 身强财弱 → 正官生印
  - 日干旺盛 → 正官拘身
  - 日旺劫多 → 正官制劫

### 七杀
- **关系**：克制日干，阳干见阳干，阴干见阴干
- **职能**：杀身之对手，专以攻身为尚。无制为七杀，有制或杀弱极为偏官。一般以凶神论
- **扶抑能力**：耗财，生印，攻身，制劫
  - 日强财弱 → 七杀耗财
  - 日强印轻 → 七杀生印
  - 印轻财重 → 七杀攻身
  - 日强劫多 → 七杀制劫

### 正偏财
- **关系**：受日干克制。正财：阳干见阴干，阴干见阳干；偏财：阳干见阳干，阴干见阴干
- **职能**：养命之源，人人所欲，一般以吉神论
- **扶抑能力**：生官杀，泄食伤，制枭神，坏正印
  - 日旺官杀弱 → 财生官杀
  - 日旺财弱 → 财泄食伤
  - 日旺枭神旺 → 偏财制枭神
  - 日旺正印旺 → 正财坏正印

### 正偏印
- **关系**：生我日干。正印：阳干见阴干，阴干见阳干；偏印：阳干见阳干，阴干见阴干
- **职能**：我之气源，如父母生身之义。正印为吉神，偏印为凶神（枭神见食而夺）
- **扶抑能力**：生身，泄官杀，御伤，挫食
  - 日弱官杀强 → 印泄官杀生身
  - 日弱食伤重 → 正印御伤，偏印挫食

### 食神伤官
- **关系**：我生。伤官：阳干见阴干，阴干见阳干；食神：阳干见阳干，阴干见阴干
- **职能**：伤见官仗势欺之（凶），食见杀能制服（吉）
- **扶抑能力**：泄身，生财，敌杀，损官
  - 身强财官弱 → 食伤泄身
  - 身强财弱 → 食伤生财
  - 身弱官杀重 → 食伤敌杀损官

### 比劫
- **关系**：同我日干。劫财：阳干见阴干，阴干见阳干；比肩：阳干见阳干，阴干见阴干
- **职能**：财之敌，日旺一般以凶神论
- **扶抑能力**：帮身，任官杀，化泄，夺财
  - 日弱 → 比劫帮身、任官杀、不怕化泄
  - 日旺 → 比劫夺财

## 六亲对应（核心原则）

> **正异偏同**：正神为异性六亲，偏神为同性六亲

### 六亲十神对应表

| 十神 | 男命 | 女命 |
|------|------|------|
| 正官 | 女儿 | 丈夫 |
| 七杀（偏官） | 儿子 | 外婆 |
| 正印 | 母亲 | 祖父、女婿 |
| 偏印 | 祖父 | 母亲 |
| 正财 | 妻子 | 父亲 |
| 偏财 | 父亲 | 婆婆、姑 |
| 比肩 | 兄弟 | 姐妹 |
| 劫财 | 姐妹 | 兄弟 |
| 食神 | 孙子 | 女儿 |
| 伤官 | 祖母、孙女 | 儿子 |

## 六亲推算进阶

**核心原理**：十神对应六亲，推六亲以日干为中心，结合合化关系无限延伸。

**例：甲己合化土**
- 土生金 → 庚辛金为子女
- 甲木为阳，己土为阴 → 甲为妻（正财），己为夫（正官）
- 甲木的子女：庚阳金（七杀），辛阴金（正官）
- 庚金的父亲：甲木（偏财）
- 庚金的祖母：甲木的正印 → 癸水（伤官）

> 推六亲非常重要，知道这个道理以后可以无限延伸。外婆、祖父等都可以此法推出。

## 十神心性

### 正官
正直负责，端庄严肃，循规蹈矩，但易流于刻板，墨守成规，意志不坚。

### 七杀（偏官）
豪爽侠义，积极进取，威严机敏，但易流于偏激，叛逆霸道，堕落极端。

### 正印
聪颖仁慈，淡泊名利，逆来顺受，但易流于庸碌，缺乏进取，迟钝消极。

### 偏印
精明干练，反应机敏，多才多艺，但易流于孤独，缺乏人情，自私冷漠。

### 比肩
稳健刚毅，勇敢冒险，积极进取，但易流于孤僻，缺乏合群，孤立寡合。

### 劫财
热诚坦直，坚韧志旺，奋斗不屈，但易流于盲目，缺乏理智，蛮横冲动。

### 食神
温文随和，带人宽厚，善良体贴，但易流于虚伪，缺乏是非，迂腐懦弱。

### 伤官
聪明活跃，才华横溢，逞强好胜，但易流于任性，缺乏约束，桀骜不驯。

### 正财
勤劳节俭，踏实保守，任劳任怨，但易流于苟且，缺乏进取，懦弱无能。

### 偏财
慷慨重情，聪明机灵，乐观开朗，但易流于虚浮，缺乏节制，浮华风流。
