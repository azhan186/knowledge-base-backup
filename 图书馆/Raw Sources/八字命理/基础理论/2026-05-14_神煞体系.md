# 神煞体系

> 来源：公开资料整理 | 收录时间：2026-05-14

---

## 核心认知

神煞**不是独立存在**的，它是依附在八字原局、大运、流年、命宫、身宫之上的"象"的标签。

学习神煞体系，关键在于掌握：
- **安法**（怎么找）
- **分类**（代表什么）
- **核心法则**（怎么用）

---

## 一、神煞的核心安法（找法）

### 1. 以年支/日支为基准（最重要）

大部分吉神凶煞都靠此法：
- **年支查**：代表大环境、祖上、少年运势对命主的影响
- **日支查**：代表婚姻、配偶、自身中年的内心状态

#### 天乙贵人（最大吉神）
口诀：甲戊庚牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸蛇兔藏，六辛逢马虎，此是贵人方。

| 日干 | 贵人所在 |
|------|----------|
| 甲戊庚 | 丑（牛）、未（羊） |
| 乙己 | 子（鼠）、申（猴） |
| 丙丁 | 亥（猪）、酉（鸡） |
| 壬癸 | 巳（蛇）、卯（兔） |
| 辛 | 午（马）、寅（虎） |

> 例：日干甲木，贵人就是丑和未。身宫辛丑、胎元癸未就带了天乙贵人。

### 4. 以命宫/身宫为基准（三垣专属）

把命宫和身宫当作独立命局来安神煞：

- **命宫庚寅**：天干庚的禄在申，逢"申"填实禄位
- **身宫辛丑**：地支丑的将星在酉（巳酉丑见酉为将星），逢"酉"的事业领域易掌权

## 二、神煞的核心用法

### 核心原则：神煞必须"落地"，不能悬空

神煞必须安在具体干支上，才能论吉凶。悬空则无力。

### 1. 吉神要"生旺"
- 不能空亡
- 不能被刑冲穿破
- 否则贵人无力

> 例：贵人在未和丑，但胎元未和身宫丑相冲（丑未冲），叫"贵人互冲"，虽有贵气，但容易在抉择关头因纠结而自毁长城。

### 2. 凶神要"制服"
- 不能被生扶
- 最好被克制

> 例：阳刃为凶，若有己土伤官，叫"阳刃化伤"，暴戾之气化为艺术创造力。
> 若有壬水七杀，叫"阳刃架杀"，是掌大权的格局。

### 3. 神煞要看"组合套餐"

| 组合 | 意象 |
|------|------|
| 驿马+贵人 | 马星逢贵人，出门遇贵 |
| 驿马+劫煞 | 马星逢劫煞，开车要小心 |
| 桃花+贵人 | "桃花贵"，情人是贵人，或靠异性关系得助 |
| 桃花+羊刃 | "桃花煞"，因色招灾 |

## 三、案例断语

案例条件：丙午年 三月廿八 巳时，命宫：庚寅，身宫：辛丑，胎元：癸未

**若日柱为甲子，则三垣神煞组合为：**
- 命宫庚寅 → 带驿马
- 身宫辛丑 → 带天乙贵人
- 胎元癸未 → 带天乙贵人
- 月柱壬辰 → 天德+月德双重护佑

**结构评价："德盖身命，马踏贵乡"**

仅凭三垣这几个神煞组合，意味着：
1. **危难有救**：天月二德加持，绝境中常有转机
2. **贵人来自远方**：马星在命宫，贵人多为异乡人，或必须动起来才能遇贵
3. **成败皆因"情"与"面"**：丑未冲贵人，桃花若入局，容易在人际关系上栽跟头
