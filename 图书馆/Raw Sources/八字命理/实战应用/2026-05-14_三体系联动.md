# 三体系联动实战

> 来源：公开资料整理 | 收录时间：2026-05-14

---

## 核心认知

三套体系不是孤立的，各有核心"组合套路"，合起来就是一套完整的吉凶推导逻辑：

- **大运排法**：核心是"动静"与"应期"
- **神煞体系**：核心是"组合象"与"套餐"
- **格局论法**：核心是"顺用"与"逆用"

---

## 一、大运排法：核心组合

### 1. 干支一柱（盖头截脚）
| 类型 | 特征 | 效果 |
|------|------|------|
| 盖头 | 运干克运支（如戊子） | 好的部分被压制，吉凶都打折 |
| 截脚 | 运支克运干（如甲申） | 内在能量不足，想做但力不从心 |

### 2. 大运与原局冲合
| 类型 | 特征 | 效果 |
|------|------|------|
| 冲提运 | 大运干支与月柱天克地冲 | 主十年人生环境、事业剧烈变动 |
| 合官合财 | 大运天干合走命局官星/财星 | 防配偶、职位、钱财被人合走或牵绊 |

### 3. 岁运并临与交战
| 类型 | 效果 |
|------|------|
| 岁运并临 | 大运和流年干支完全相同，能量加倍。是人生重大关口，需看吉凶神煞 |
| 天克地冲 | 大运与流年天克地冲，主该年变动极大（奔波、吵架、换工作等） |

### 4. 交运与脱运
新旧大运交接的前后一年（通常是生日前后），是运气最不稳、最容易出事的时期。

## 二、神煞体系：核心套餐

### 驿马套餐（主奔走、变动）
| 组合 | 意象 |
|------|------|
| 马星+贵人 | 越跑越发，动中得利 |
| 马星+羊刃 | 注意车祸、械斗，出门易有血光 |
| 马星+桃花 | 为情奔波，或人在外易有艳遇 |
| 马星+空亡 | 白跑一趟，奔波劳碌却难成 |

### 桃花套餐（主人缘、情欲）
| 组合 | 意象 |
|------|------|
| 桃花+贵人 | 异性就是你的贵人 |
| 桃花+七杀/劫煞 | "桃花煞"，易因色惹祸、破财或被威胁 |
| 桃花+文昌 | 才华横溢、风流才子，靠才艺吸引异性 |

### 贵人套餐（主逢凶化吉）
| 组合 | 意象 |
|------|------|
| 天乙贵人+天月二德 | 一生危难有救，道德感强 |
| 贵人+驿马 | 贵人多在远方或来自外地 |
| 贵人互冲（如丑未冲） | 贵人意见打架，或因人情关系纠结难断 |

## 三、格局论法：核心组合

### 吉神顺用组合（正官、正印、食神、正财）
| 组合 | 效果 |
|------|------|
| 正官格+印 | 官生印，印护身，贵气，有权力地位 |
| 食神格+比劫 | 食神吐秀，有才华，靠技艺吃饭 |

### 顺用大忌
- **官见伤**：伤官克官易惹祸
- **印见财**：财星破印易因利失节
- **食遇枭**：枭神夺食主病灾、失业

### 凶神逆用组合（七杀、伤官、劫财、枭神）
| 组合 | 效果 |
|------|------|
| 七杀格+食神 | "食神制杀，英雄独压万人"，靠智谋本领解决问题，大贵 |
| 伤官格+印 | "伤官配印"，聪明才华得以收敛，成大家风范 |

### 逆用大忌
- **杀无制**：七杀无制易受欺压
- **伤无配**：伤官无配狂妄惹祸
- **劫无泄**：劫财无泄蛮横冲动

## 四、三套体系联动实战

推算一个完整的人生事件，往往是三套语言同步解析。

### 案例示范
假设命宫庚寅，身宫辛丑，大运走**甲午**运：
- 运支"午"是全局的**羊刃**
- 神煞看，午是**桃花**，且甲木合身
- 格局看，若为七杀格，走食神甲运，本应是**"食神制杀"**大有作为

### 综合断语
> 食神制杀主事业提升，但自坐羊刃桃花，又在午运，**极可能因办公室恋情或混乱的男女关系（桃花配羊刃），把好不容易打下的事业基础给毁了**。
