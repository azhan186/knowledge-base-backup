# 量子计算机深入篇

## 深入一：量子比特的数学描述（零基础友好版）

经典比特就像一个硬币，只能正面（1）或反面（0）朝上。

量子比特就像一个正在快速旋转的硬币——在它被拍停（测量）之前，它既是正面也是反面，还可能是介于两者之间的任何"状态"。

### 1. 用数字描述这种"同时存在"

在量子力学中，我们用一个公式来表示旋转硬币的状态：

|ψ⟩ = a × |0⟩ + b × |1⟩

其中：
- |0⟩ 代表"确定是0"的状态
- |1⟩ 代表"确定是1"的状态
- a 和 b 是两个数字（可以是正数、负数或复数，但零基础可先想象为0到1之间的普通正数）
- 有一个规则：a² + b² = 1

**例子**：
- 如果 a=1, b=0，那么 |ψ⟩ = |0⟩ → 测量时100%得到0
- 如果 a=0, b=1，那么 |ψ⟩ = |1⟩ → 测量时100%得到1
- 如果 a=√0.5 ≈ 0.707, b=√0.5 ≈ 0.707，那么状态是"0和1各一半"

### 2. 测量时的"魔术"

当你去"拍停"这个旋转硬币（测量量子比特）时：
- 得到 0 的概率是 a²
- 得到 1 的概率是 b²
- 测量后，量子比特的状态会立刻变成测量得到的结果（如得到0，状态就变成|0⟩）。原来的叠加信息就丢失了——这叫**坍缩**

**关键点**：你不能同时知道 a 和 b 的精确值，只能通过多次制备相同的量子比特并测量，来统计出 a² 和 b²。这就像你只能看到硬币倒下后的正反面，却永远看不到它旋转时的样子。

【可靠度】这是量子力学公认的基本公理，可靠度极高。

## 深入二：量子门——改变叠加状态的操作

经典计算机用"逻辑门"（如与门、或门）改变比特。量子计算机用量子门改变 a 和 b 的值。

### 最简单的量子门：Hadamard门（H门）

它的作用：把确定状态变成"半叠加"状态。

- 输入 |0⟩（即 a=1,b=0） → 输出：a = √0.5, b = √0.5 的状态（0和1各50%）
- 输入 |1⟩（即 a=0,b=1） → 输出：a = √0.5, b = -√0.5 的状态（仍然是各50%概率，但正负号会影响后续门——这叫"相位"）

**类比**：
- 经典 NOT 门就像把硬币从正面翻到反面
- Hadamard 门就像把一枚平放的硬币抛向空中，让它旋转起来——从确定状态变成不确定状态

**为什么需要这种门？**

因为量子算法的核心思路是：
1. 用 H 门让所有量子比特同时进入所有可能状态的叠加（如2个量子比特可以同时表示 00,01,10,11 四种状态）
2. 用量子门巧妙地对这些"同时存在的状态"进行运算（如肖尔算法中的量子傅里叶变换）
3. 最后通过测量提取正确答案，但必须设计算法让正确答案的概率远高于错误答案

【可靠度】这部分是量子计算的基本模型（量子电路模型），可靠度很高。

## 深入三：两个量子比特的奇特现象——纠缠

这是量子计算机区别于经典计算机最神奇的资源。

### 制备纠缠态

对两个量子比特进行某种操作后，可以产生如下状态：

|ψ⟩ = (|00⟩ + |11⟩) / √2

意思是：系统同时处于"两个都是0"和"两个都是1"的叠加，且没有"一个是0一个是1"的成分。

### 诡异的行为

- 如果你测量第一个量子比特，随机得到0或1，各50%概率
- 但一旦得到0，第二个量子比特瞬间就坍缩为0（无论它们相隔多远，甚至几光年）
- 如果得到1，第二个也瞬间变成1

这就像一对魔术手套：你在地球打开左手套发现是"右手的"，你就立刻知道月球上的那只一定是"左手的"——但量子纠缠更强，因为测量前它们并不存在各自的确定状态。

爱因斯坦称其为"鬼魅般的超距作用"，但实验反复证明这是真实的。目前无法用它超光速传递信息（因为结果随机），但可以用于量子隐形传态和密码学。

【可靠度】纠缠的存在被无数实验证实（贝尔不等式违反），可靠度极高。
