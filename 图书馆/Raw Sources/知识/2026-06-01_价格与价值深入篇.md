# 价格与价值 深入篇

## 深入一：价值到底是什么？——三大哲学立场

### 1. 客观价值论：价值凝结在劳动里
代表：亚当·斯密、大卫·李嘉图、卡尔·马克思

> 商品价值 = c + v + m
- c（不变资本）：原材料、机器磨损等转移的过去劳动
- v（可变资本）：工人劳动力的价值，即维持工人及其家庭生活所需的生活资料价值（工资）
- m（剩余价值）：工人创造的总价值中，超出劳动力价值而被资本家无偿占有的部分。利润、利息、地租都由此转化

**商品拜物教**：人们崇拜商品如同崇拜偶像，价格可以在供求下严重偏离价值，但无法消除价值的劳动根基。

### 2. 主观价值论：价值生在人心里
代表：边际三杰（杰文斯、门格尔、瓦尔拉斯），奥地利学派

- 水总效用极大，但因日常可得数量多，最后一杯水的边际效用极低，所以价格低
- 钻石总效用很小，但数量极少，最后一颗的边际效用极高，所以价格高
- 决定价格的不是"总有用性"，而是**边际效用**

精明的消费者无意识地在做边际效用/价格的比较：
> MU_苹果 / P_苹果 = MU_橙子 / P_橙子
即最后一元钱在不同商品上带来的边际效用相等。

### 3. 新古典综合：价格是供求的婚礼
马歇尔把客观成本（供给）和主观效用（需求）结合：
- 需求曲线向下（边际效用递减）
- 供给曲线向上（边际成本递增）
- 两线相交处 = **均衡价格**
- 完全竞争下：价格 = 边际成本，社会总福利最大化

## 深入二：价格形成的精密机器

### 弹性
> 需求价格弹性 = 需求量变动的百分比 / 价格变动的百分比

必需品弹性小，奢侈品弹性大。商家用它制定最大化收入的定价策略。

### 垄断者的操作
垄断者寻找 **MR = MC**（边际收益 = 边际成本）的产量点，然后按需求曲线定价，产生**无谓损失**（社会福利损失）。

常用**价格歧视**：学生票、早鸟票、高峰加价——把买家"最高愿意支付的价格"都收割干净。

### 金融资产的内在价值
资产价值 = 未来现金流的折现总和：
> NPV = CF₀ + CF₁/(1+r)¹ + CF₂/[(1+r)²] + ... + CFₙ/(1+r)ⁿ

资本资产定价模型（CAPM）：
> 预期收益率 = 无风险利率 + β × (市场平均收益率 - 无风险利率)

## 深入三：背离、泡沫与心理暗流

### 明斯基时刻：泡沫的内在剧本
市场从"稳健融资"（能还本付息）→"投机融资"（只够付息）→"庞氏融资"（靠资产涨价滚动）。在庞氏阶段，价格完全由"相信有下一个接盘侠"支撑，信心逆转即崩溃。

### 行为经济学：你的大脑不是机器
- **锚定效应**：原价999现价299，让999成为锚点，即使只值150仍觉得划算
- **羊群效应**：别人买就跟着买，把价格推离理性

### 安全边际
> 安全边际 = 估算的内在价值 - 买入价格

买得足够便宜，就是把犯错的容错空间设计进策略。

## 深入四：前沿颠覆——旧理论的裂痕

### 数字经济：零价格与巨大价值并存
- 谷歌搜索、微信、维基百科价格是零，但创造了难以度量的效率价值
- 你支付的是"注意力和数据"
- 估算方式：**消费者剩余** = 愿意支付的最高价格 - 实际支付的价格

### AI与零边际成本
- AI生成文章/音乐的复制成本几乎为零
- 按"价格 = 边际成本"，数字产品价格趋向于零
- 催生：全民基本收入（UBI）、数据贡献证明等新议题

### 给自然写一张价格牌
- 红树林提供护岸、净水、固碳等功能，没有市场价
- "碳排放的社会成本"：每多排一吨碳，未来损失折现至今是多少钱
- 用价格语言迫使决策者正视环境价值

### 加密货币：纯粹的共识价值？
- 比特币不产生现金流，无国家背书
- 价值完全来自去中心化信任网络
- 提出终极问题：**如果足够多人坚信某物有价值，它是否就有了价值？**
