# 知识库

## 架构

```
知识库/
├── Raw_Sources/     # 第一层：原始资料（用户拥有，我只读）
├── Wiki/            # 第二层：知识库（由我维护）
└── Schema/          # 第三层：规则文件（共同维护）
```

## 三层说明

### 第一层：Raw Sources
只属于用户的原始资料库，我只能读取，不能修改。

### 第二层：Wiki
由我负责的知识库。从 Raw Sources 中提取、对比、验证后记录。

### 第三层：Schema
配置规范，包含结构规范、命名约定、工作流程。
