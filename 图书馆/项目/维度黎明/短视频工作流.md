# 《维度黎明》短视频制作工作流

## 目标
用本地 ComfyUI 工作流，制作与《维度黎明》世界观相关的简短视觉内容（概念展示、场景卡、角色动效等）。

---

## 工作流设计

### 阶段一：素材规划（做之前）
1. **确定主题** — 例如「临江市夜景」「太初钥」「守夜人」
2. **收集参考** — 文字描述 → 转译为画面关键词
3. **输出分镜脚本** — 每秒/每镜的画面描述

### 阶段二：图像生成（ComfyUI）
**A. 概念图工作流**
```
[世界观描述] → [ComfyUI 提示词优化节点] → [FLUX/SDXL 生成] → [高清放大] → [输出]
```

**B. 批次生成流程**
1. 设定「维度黎明」基础风格模板
2. 批量生成同一场景的不同角度/氛围版本
3. 挑选最优 → 进入视频阶段

### 阶段三：视频生成
**方案一：本地工具**
- Sadaharaku / DynamiCrafter（本地视频生成）
- 图像 → 短片段（3-5秒）

**方案二：即梦/可灵**
- 用本地生成的图像作为关键帧
- AI 扩展为视频片段

## 《维度黎明》风格提示词模板

### 基础风格设定
```
Sci-fi cyberpunk, dark city night, neon lights, eldritch magic glow, 
cinematic lighting, 8K detail, --ar 16:9 --style raw
```

### 具体场景模板

**临江市夜景**

rain-slicked streets, towering megastructures with eldritch glowing runes,
holographic advertisements floating above, [情绪: mysterious/dangerous/grand],
cinematic atmosphere, volumetric lighting, 8K

**太初钥**

**零（女主角）**

## 制作优先级（简单优先）

### 第一批尝试（最容易出效果）
1. **临江市天际线** — 城市全景，赛博朋克氛围
2. **单件零号遗物特写** — 太初钥/不周杖
3. **角色概念姿态** — 守夜人剪影

### 输出规格
- 分辨率：1280×720 或 1920×1080
- 格式：PNG序列 → MP4
- 时长：5-15秒短视频
