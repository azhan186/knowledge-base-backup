# 基金净值（NAV）

**Net Asset Value（净资产价值）**：单位基金份额对应的**净资产价值**，是基金的**核心定价机制**。

## 计算公式

```
NAV = (基金总资产 - 基金总负债) / 总份额
```

## 关键特征

- **每日公布**（开放式公募基金，每个交易日结束后）
- **申购/赎回价 = NAV**（未知价原则：交易时不知当日 NAV，收盘后按 NAV 结算）
- 净值 = 1.00 起始（基金成立时面值）
- 净值增长 = 投资收益（不含分红再投资）

## 实际例子

```
基金总资产 = 14,639.74 元
基金总负债 = 0
总份额 = 9,066.93 份

NAV = 14,639.74 / 9,066.93 = 1.614630 元/份
```

## NAV 与投资的几个要点

### 1. NAV 高低 ≠ 投资价值
- 净值 2.00 的基金不一定比 1.00 的贵
- 关键看**未来收益能力**（基金持仓、行业前景）
- 类比：股票股价 100 元不一定比 10 元贵

### 2. NAV 与分红
- 基金分红后 NAV 下降（把利润分配给投资者）
- 例如 NAV 1.6 → 分红 0.005 → 新 NAV 1.595
- **分红不是"额外收益"**，而是投资者自己钱的一部分

### 3. 净值增长 vs 累计净值
- **单位净值**：当前 NAV
- **累计净值**：单位净值 + 历次分红（更能反映真实收益）

## 关键洞察

> **NAV 是基金的"温度计"——告诉你"现在值多少"，但不告诉你"未来值多少"**
