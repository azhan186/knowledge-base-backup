# 行星

## 一句话定义

行星是围绕恒星运行，自身足够大、呈球形，并且"清扫"了自己轨道附近区域的天体。

## 核心概念

### 三要素

1. **绕恒星转** —— 与恒星（自己发光发热）的最大区别
2. **足够圆** —— 自身引力足以把自己"捏"成近似球体
3. **清空轨道** —— 引力把周围差不多大小的东西都吸引或弹开了，是所在区域的"老大"

> 正是第三条，让冥王星在2006年被"降级"为矮行星——它位于满是"邻居"的柯伊伯带，没能清空自己的轨道。

### 旋转木马比喻

| 天体类型 | 比喻 |
|----------|------|
| **行星** | 个头很大的大木马，自身大而圆，在一条干净的专属跑道上跑，跑道上的小石子早就被扫干净了 |
| **矮行星**（如冥王星） | 个头小一点的木马，所在跑道是和其他小马、石块混在一起的拥挤外圈——自己够圆，但不是跑道的主人 |
| **小行星等小天体** | 跑道上没被清理的碎石子，形状不规则，也绕中心转，但不够大、没能力变圆，也不是任何区域的老大 |

**决定地位的关键**：除了"绕谁转"和"长什么样"，更重要的是"有没有搞定自己的地盘"。

### 历史演变

| 时间 | 事件 |
|------|------|
| 古代 | "行星"（planet）源于希腊语"流浪者"——水金火木土+日月的七颗"流浪星" |
| 哥白尼革命 | 地球"降级"为普通行星，太阳成为中心 |
| 1781 | 赫歇尔意外发现天王星——第一颗被望远镜发现的行星 |
| 1846 | 轨道计算预测海王星存在，被发现——牛顿力学伟大胜利 |
| 1930 | 冥王星被发现，成为"第九行星" |
| 20世纪90年代起 | 柯伊伯带发现越来越多和冥王星差不多大的天体 |
| 2006 | IAU历史性投票：通过三要素定义，冥王星被划入矮行星，太阳系行星从9变8 |

### 两大家族：类地 vs 类木

详见 [[类地行星与类木行星]]：

- **类地行星**（水星、金星、地球、火星）：岩石金属质地，固体表面，较小
- **类木行星**（木星、土星、天王星、海王星）：气态或冰巨星，无固体表面，较大

### 研究行星的价值

- **理解地球的过去与未来**：金星是失控温室效应，火星是死去的气候
- **寻找生命**：宜居带（温度允许液态水存在的区域）是寻找"下一个地球"的指南针
- **资源与保护**：了解小行星轨道预防撞击；小行星矿藏是人类太空文明潜在目标
- **导航与计时**：日历、GPS的精确运行依赖对地球运行规律的精确掌握

### 系外行星前沿

- **已确认超过5000颗**系外行星
- **超级地球之谜**：银河系最常见的行星类型，在太阳系却没有——为什么我们如此特殊？
- **直接成像挑战**：恒星太亮，需开发遮星板等新技术
- **大气分析**：凌星法分析穿过行星大气的星光，寻找水、甲烷、氧气等"生命迹象"

### 定义争议（未解决）

- IAU 2006定义对远距离行星不公平（地球放冥王星轨道也无法清空）
- 系外行星（围绕其他恒星的行星）需要一个更普适的定义
- 科学定义与公众情感的拉锯：冥王星在公众情感中仍是"第九行星"

## 交叉引用

> [!note]
> 行星 vs 恒星：行星不自己发光，靠反射恒星的光。如果一颗天体能自己发光（核聚变），那它就是恒星而非行星。
