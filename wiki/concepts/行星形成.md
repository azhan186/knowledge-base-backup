# 行星形成（ Planet Formation ）

> 核心定义：行星形成是年轻恒星周围的一盘气体和尘埃（原行星盘），在几千万年内，通过微观粘附、引力聚集和剧烈碰撞，逐步从微米级的尘埃成长为数千公里级的天体的过程。

## 三步走：吸积、碰撞、清空

| 阶段 | 说明 |
|------|------|
| **吸积** | 极小的尘埃先粘在一起，像滚雪球一样变大 |
| **碰撞** | 大到一定程度后，通过引力吸引彼此，发生大规模碰撞，合并成行星胚胎 |
| **清空** | 最终的行星胚胎扫荡自己轨道，清空附近所有物质，成为真正独立的行星 |

## 四步详细分解

### 第一阶段：尘埃凝聚（0.1微米 → 1公里）
- 原行星盘：99%气体 + 1%微米级尘埃
- 尘埃因静电力或化学粘性粘在一起，慢慢长成毫米、厘米大小的团块
- **最大谜题"径向漂移障碍"**：厘米到米级团块因气体阻力快速螺旋坠入恒星，时间来不及继续长大
- 必须快速跨越这个尺寸区间，直接形成公里级的星子

### 第二阶段：星子形成与吸积（1公里 → 1000公里）
- 直径约1公里的天体叫**星子**
- 较大星子开始"**引力聚焦**"，像小型真空吸尘器
- 进入"**失控式吸积**"，迅速长到月球到火星大小
- 这个大小的天体叫**行星胚胎**

### 第三阶段：巨型撞击
- 剩下几十个月球到火星大小的胚胎，轨道变得混乱
- 极其暴力的巨型撞击时代
- **月球形成**：火星大小的忒伊亚与原始地球巨大斜撞，溅出物质形成月球
- 水星高密度：可能是剥掉外层的一次大撞击
- 天王星躺着自转：可能被侧撞了一次

### 第四阶段：清空轨道与两条岔路

| 岔路 | 位置 | 结果 |
|------|------|------|
| **类地行星（岩石行星）** | 雪线之内（靠近恒星） | 太热，水和甲烷只能以气体存在，抓不住 → 只形成岩石金属小个头行星 |
| **类木行星（气态巨行星）** | 雪线之外（远离恒星） | 水、甲烷、氨凝结成冰，固体原材料翻几倍 → 长到地球质量10倍左右 → 引力吸引氢氦气体，急剧膨胀成气态巨行星 |
| **冰巨星** | 形成得晚 | 盘中气体已大量消散，只裹上较少大气 |

> **雪线**：越过它，水冰才能稳定存在。

## 类比：静电吸附的杂粮粉

| 阶段 | 场景 | 对应 |
|------|------|------|
| 第一阶段 | 静电吸附，面粉糖霜粘成微小絮状团块 | 微米尘埃粘附成毫米/厘米级团块 |
| 第二阶段 | 团块碰撞粘住，变成小米球、花生米 | 星子阶段（公里级） |
| 第三阶段 | "花生米"长大，用引力拉向自己，长成葡萄 | **行星胚胎**，失控式吸积 |
| 第四阶段 | 几颗葡萄碰撞合并成大核桃 | **巨型撞击**，行星胚胎合并 |
| 最后清场 | 大核桃吃掉碎屑或弹飞到边缘/中心 | **清空轨道**，行星形成 |

## 历史演变

| 时代 | 理论 | 说明 |
|------|------|------|
| **18世纪** | **康德-拉普拉斯星云假说** | 哲学雏形正确，具体机制错误 |
| **20世纪初** | **遭遇危机** | 星云假说角动量不对，"潮汐假说"有严重缺陷 |
| **1940年代** | **魏茨泽克涡旋模型** | 重新拉回星云盘正确道路 |
| **1960-70年代** | **萨夫龙诺夫星子假说** | 现代行星形成理论的**真正奠基人** |
| **1990年代至今** | **系外行星革命** | 1995年飞马座51b（热木星）迫使加入"行星迁移"机制 |

## 前沿挑战与争议

### 1️⃣ 米级障碍（Meter-size barrier）
- 尘埃如何跨越从厘米到公里星子的"死亡谷"？
- 可能机制：在湍流"**压力陷阱**"里集体快速引力坍缩，直接跨过
- **ALMA**已在原行星盘中拍到疑似环和旋涡结构

### 2️⃣ 巨行星的迁移与秩序
- 热木星不应该存在，却存在 → 巨行星会大规模迁移
- 迁移会摧毁沿途小行星，类地行星还有机会吗？
- 尼斯模型：太阳系巨行星大幅度外迁，**时间和幅度仍有争议**

### 3️⃣ 超级地球的形成之谜
- **超级地球**（地球和海王星之间）是银河系最常见行星类型
- 太阳系里根本没有
- 形成超级地球的典型盘环境，太阳系为何没有？

### 4️⃣ 直接观测行星形成中的过程
- 原行星盘中看到环和缝隙（行星胚胎撬开）
- 直接拍到嵌在盘中正在吸积的热"行星候选体"
- 把观测和模型一一对应，**是当前最令人激动的领域**
