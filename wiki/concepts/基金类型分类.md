# 基金类型分类

按**投资标的**和**风险等级**划分的基金分类体系。

## 六大主要类型

| 类型 | 投资标的 | 风险 | 预期年化 | 适合 |
|------|----------|------|----------|------|
| **货币基金** | 短期银行存款、国债、央行票据 | 极低 | 2-3% | 现金管理 |
| **债券基金** | 国债、企业债、可转债 | 低-中 | 4-6% | 保守型 |
| **股票基金** | 股票（80%+）| 中-高 | 8-15% | 长期投资 |
| **混合基金** | 股票+债券+现金（比例灵活）| 中 | 5-10% | 平衡型 |
| **指数基金** | 跟踪特定指数 | 跟随市场 | 接近指数 | 长期投资者 |
| **ETF** | 同上（可在交易所买卖）| 跟随市场 | 接近指数 | 灵活投资者 |

## 风险-收益递进

```
低风险 ▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁ 高风险
货币 → 债券 → 混合 → 股票 → 主动股票
                        ↓
                    行业基金 / 主题基金
```

## 投资风格维度（股票基金细分）

| 维度 | 类型 | 特征 |
|------|------|------|
| **市值** | 大盘/中盘/小盘 | 流通市值 |
| **风格** | 价值/平衡/成长 | 估值/质量/增长 |
| **行业** | 行业基金/主题基金 | 集中特定行业 |
| **地区** | A股/港股/全球 | 地理范围 |
| **策略** | 量化/对冲/多空 | 投资方法 |

## 特殊类型

| 类型 | 特点 |
|------|------|
| **QDII** | 投资境外市场 |
| **FOF（基金中的基金）| 投资其他基金 |
| **MOM（管理人的管理人）| 投资其他基金经理 |
| **REITs** | 投资不动产 |
| **量化基金** | 算法/模型驱动 |
| **对冲基金** | 私募，多空策略 |

## 选择建议

| 投资目标 | 推荐类型 |
|----------|----------|
| 现金管理 | 货币基金 |
| 短期稳健 | 债券基金、混合偏债 |
| 长期增值 | 指数基金 / ETF / 优质主动基金 |
| 行业机会 | 行业基金 / 主题基金 |
| 全球化 | QDII / 全球配置基金 |
