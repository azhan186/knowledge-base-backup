# 马科维茨投资组合理论

**Markowitz Portfolio Theory (MPT)**：1952 年由哈里·马科维茨提出的**现代金融学奠基理论**，**用数学方法**解决了"**如何在风险与收益之间取得最优平衡**"的问题。**1990 年诺贝尔经济学奖**。

## 核心思想

> **不要把鸡蛋放在一个篮子里——但具体怎么分散？**
> 马科维茨给出了**数学最优解**。

## 均值-方差模型

### 输入
- 多个资产的**预期收益**（均值 μ）
- 多个资产的**风险**（方差 σ²）
- 资产之间的**相关性**（协方差 σ_ij）

### 输出
- **最优投资组合**：在给定预期收益下**方差最小**，或给定方差下**收益最大**
- 求解方法：**二次规划**（数学上严格可解）

## 有效边界（Efficient Frontier）

在均值-方差平面上，**所有最优投资组合**构成一条曲线：

```
收益
  ↑
  │            ╱ 有效边界
  │          ╱
  │        ╱
  │      ╱ ← 切点（M 点，全市场组合）
  │    ╱
  │  ╱
  └────────→ 风险（σ）
```

- **有效边界上的组合**：在同样风险下收益最高，或同样收益下风险最低
- **边界下方**：次优组合（同样风险下收益更低）
- **边界不存在**：次优区域
- **CML（资本市场线）**：从无风险利率出发，与有效边界相切的直线

## 关键洞察

### 1. 分散化的数学价值
- **不是简单"多买几只"**
- 而是找到**相关性低甚至负相关**的资产
- 让组合波动 < 单只加权平均波动
- **关键**：组合波动 ∝ √(权重 × 协方差矩阵)

### 2. 系统性 vs 非系统性风险
- **非系统性风险**：单只股票/债券的特有风险 → **可分散**
- **系统性风险**：整个市场的风险 → **不可分散**
- 即使充分分散，**仍有系统性风险敞口**

## 局限

| 局限 | 说明 |
|------|------|
| **输入难以估计** | 预期收益、协方差都用历史数据估计，**未来未必重复** |
| **正态分布假设** | 真实市场有"肥尾"（极端事件）|
| **静态视角** | 实际投资是动态调整的 |
| **忽略交易成本** | 频繁调仓成本不低 |
| **不处理税收** | 税后收益才真实 |

## 在基金中的应用

- **FOF（基金中的基金）**：用 MPT 思想配置不同基金
- **资产配置策略**：股票/债券/商品的权重优化
- **风险平价（Risk Parity）**：MPT 的变种，让各资产风险贡献相等
- **Black-Litterman 模型**：在 MPT 基础上加入主观判断

> ——马科维茨
