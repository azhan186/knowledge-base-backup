# Alpha-Beta 归因

**Alpha-Beta Attribution**：把基金的**总收益**拆解为**市场风险贡献（Beta）**和**主动管理贡献（Alpha）**的分析框架。**是评估基金经理能力的最重要工具**。

## 经典公式

```
R_fund = R_f + α + β × (R_market - R_f) + ε
```

| 符号 | 含义 |
|------|------|
| R_fund | 基金实际收益 |
| R_f | 无风险收益（如同业拆借利率）|
| α (Alpha) | **超额收益**——基金经理的"真正能力" |
| β (Beta) | **市场风险敞口**——基金对市场的敏感度 |
| R_market | 基准市场收益 |
| ε (Epsilon) | 残差（特异风险）|

## Beta 的含义

- β = 1：基金与市场同涨同跌
- β > 1：基金波动大于市场（激进型）
- β < 1：基金波动小于市场（稳健型）
- β < 0：基金与市场反向（罕见，如反向基金）

**关键**：**Beta 贡献 = 基金经理靠承担系统性风险获得的收益**——这部分**不值得付费**（市场已开放购买）。

## Alpha 的含义

- α > 0：基金跑赢市场（剔除风险因素后仍有超额收益）
- α = 0：基金收益完全来自市场暴露
- α < 0：基金跑输市场（付费反而损失价值）

**关键**：**Alpha = 主动管理的"价值"**——这是投资者付管理费**唯一合理的理由**。

## 选股 vs 择时（Alpha 进一步拆解）

```
α = α_选股 + α_择时 + α_其他
```

| 来源 | 含义 | 评估方法 |
|------|------|----------|
| **α_选股** | 选对股票产生的超额收益 | 持仓分析 |
| **α_择时** | 调对仓位产生的超额收益 | 现金比例时间序列 |
| **α_其他** | 行业暴露、因子暴露 | Brinson 模型 |

## 实证数据

- **绝大多数主动基金**：α < 0（扣除费用后）
- **少数"明星"基金**：α > 0，但**很难持续**
- **平均**：α ≈ -0.5% 到 -1%/年（被费用侵蚀）

## 经典案例

**A 基金**：年化 15%，β = 1.2，市场收益 10%
- Beta 贡献 = 1.2 × 10% = 12%
- Alpha 贡献 = 15% - 12% = **3%**（优秀！）

**B 基金**：年化 10%，β = 1.0，市场收益 10%
- Beta 贡献 = 1.0 × 10% = 10%
- Alpha 贡献 = 10% - 10% = **0%**（市场收益，不值管理费）

**C 基金**：年化 8%，β = 1.0，市场收益 10%
- Beta 贡献 = 1.0 × 10% = 10%
- Alpha 贡献 = 8% - 10% = **-2%**（跑输市场 2%）

## 关键洞察

> **Beta 容易，Alpha 难**
> 多数基金经理的"能力"其实是 Beta——市场涨就涨，市场跌就跌
> 真正的 Alpha 来自**信息优势、独特视角、纪律执行**——这极其稀缺
