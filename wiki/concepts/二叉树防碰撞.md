# 二叉树防碰撞

**Binary Tree Anti-collision Algorithm**：通过**逐步分裂标签集合**来识别唯一标签的确定性防碰撞算法。**EPC Gen2 同时使用时隙 Aloha，二叉树常见于 ISO 14443 等 HF 协议**。

## 算法思想

```
阅读器：序列号以 '0' 开头的请回答
  → 仍有碰撞
  
阅读器：序列号以 '00' 开头的请回答
  → 仍有碰撞
  
阅读器：序列号以 '000' 开头的请回答
  → 唯一标签应答
  → 识别完成
  
阅读器：回退到上一层 '001' 分支继续搜索
```

## 关键参数

- **分支方向**：通常 0 / 1
- **深度限制**：避免无限递归
- **回溯策略**：深度优先（DFS）vs 广度优先（BFS）

## 与时隙 Aloha 对比

| 维度 | 时隙 Aloha | 二叉树 |
|------|------------|--------|
| **性质** | 概率性（随机选时隙）| 确定性（按位分裂）|
| **复杂度** | O(N) | O(N log N) |
| **识别率** | 受 36.8% 极限约束 | 理论 100% |
| **速度** | 极快（毫秒级）| 较慢（需多轮）|
| **适用** | UHF 大量标签 | HF 少数标签 |
| **协议** | EPC Gen2 | ISO 14443 Type A/B |

## 实际应用

- **HF 协议**（ISO 14443/15693）：常用二叉树
- **NFC 点对点模式**：可用二叉树快速识别
- **EPC Gen2**：混合使用（盘活用 Aloha，细查用树）
