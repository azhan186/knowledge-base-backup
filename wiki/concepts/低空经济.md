# 低空经济（Low-Altitude Economy）

## 一句话定义

以低空空域（通常垂直高度1000米以下，根据不同管理需要可延伸至3000米）为依托，以通用航空为基础，融合无人机、eVTOL等飞行器的综合性经济形态——是"十五五"期间我国的新兴支柱产业。

##四大产业板块

|板块 | 内容 |
|------|------|
| 低空制造 |无人机、eVTOL、轻型飞机、直升机的研发和生产 |
| 低空飞行 |物流配送、载人交通、农林喷洒、测绘巡检 |
| 低空保障 | 空域管理、通信导航、起降场地、充电设施 |
| 综合服务 |驾驶培训、融资租赁、保险、会展 |

## 为什么是现在？

1. **电动化**：分布式推进（十几个小电机 → 单点失效仍可安全降落）+简化机械结构
2. **无人机普及**：硬件成本压到极低，操作极大简化
3. **eVTOL兴起**：让安全的、可规模化运营的城市载人空中交通成为可能

## 当前应用

-急救转运 /特种消防救援 / 点对点高端接驳
-未来10 年：叫车软件"打飞的"成可能

##关键技术：固态电池

-能量密度是常规三元锂电的 **1.5 倍**
-实验室向量产冲刺阶段：寿命 +良品率是当前挑战
- 已成功飞越琼州海峡

##根本性挑战（三约束三角形）

1. **能量密度**（物理学）：与载重、航程、安全裕度构成"不可能三角"
2. **安全可认证性**（系统工程学）：避障算法若用深度学习，则无法追溯因果 →适航认证死结
3. **空域容量与流量管理**（运筹学）：万架级调度组合爆炸，必须"宏观战略解耦+微观战术自治"

##监管分歧

- **集中式**：统一平台+预审批航路（安全清晰，灵活不足）
- **分布式**：动态协商+实时授权（响应灵活，复杂度暴增）
- **中国**：多城市试点，摸索中间路线

##市场规模

- 有望牵引**万亿级至十万亿级**产业集群
-早期市场必然是高端商务+特定摆渡，非日常通勤
