# 股票

**股票（Stock / Equity）**：代表对公司的**部分所有权** + **未来现金流**的索取权。

## 核心概念

- **公司所有权**：1 股 = 1/N 投票权 + 1/N 利润分享权
- **两类收益**：股息 + 资本利得
- **风险**：经营 + 市场 + 行业 + 汇率 + 政策

|------|------|
