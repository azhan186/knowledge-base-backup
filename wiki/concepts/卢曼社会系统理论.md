# 卢曼社会系统理论

**Niklas Luhmann's Social Systems Theory**（1984 起）：用系统论的视角重新理解社会——社会不是由共识凝聚的，而是由**成功的沟通本身**凝聚的。

## 核心思想

- 社会是**自创生系统（autopoietic system）**——通过自身运作实现自我再生产
- 社会的基本元素不是人或行动，而是**沟通**
- 子系统通过**二元符码（binary code）**独立运作

## 二元符码表

| 子系统 | 二元符码 |
|--------|---------|
| 法律 | 合法 / 非法 |
| 政治 | 执政 / 在野 |
| 经济 | 支付 / 不支付 |
| 媒体 | 信息 / 非信息 |
| 科学 | 真 / 假 |
| 教育 | 教化 / 不教化 |
| 宗教 | 信仰 / 不信仰 |
| 艺术 | 美 / 丑 |

## 结构耦合（Structural Coupling）

子系统之间通过**相互观察、相互扰动**形成耦合，但**保持各自的符码独立**。

例如：政治系统与媒体系统的耦合：

- 政治系统可以制造媒体系统的"非信息"
- 媒体系统可以生产政治系统的"权力冲突"信息

→ 形成**双向短路**，后真相生态由此产生。

## 对后真相的重新解读

> 后真相**不是信息生态的崩坏**，而是**一种新的耦合形态正在形成**：
> 大众媒体系统的"信息/非信息"符码，与政治系统的"权力/无权力"符码，发生了深度的相互渗透和短路。

## 专家洞察

> 🔑 在功能分化的复杂社会中，试图通过"事实核查"消除所有信息矛盾，就像试图**消除免疫系统的自我非我识别**——不仅不可能，而且**破坏系统自身的复杂性处理机制**。

**真正的问题不是矛盾的存在**，而是**哪些社会子系统失去了处理矛盾并将之转化为稳定结构的能力**。

当一个系统只能承受**极低程度**的内部矛盾时，反而会**极其脆弱**，小的扰动就会导致其**解耦或崩溃**。

## 学术地位

- 社会学的高阶宏大理论
- 富有解释力但高度抽象
- 可靠度属于**理论范式级别**，非经验定律
- 存在实质性争议
