# 宇宙

## 一句话定义

宇宙是所有空间、时间、物质和能量的总和。

## 核心概念

### 什么是宇宙

> 宇宙就是一切：你、我、地球、太阳、所有的星星、星系，以及它们之间的"空"的部分，还有过去、现在和未来的全部时间。

**特别强调**：宇宙大爆炸不是在一个原本就有的"空房间"里发生的爆炸——空间本身也是宇宙的一部分，并且是从无到有、一直膨胀到今天。

### 葡萄干面包：理解宇宙膨胀

想象一块正在烤箱里膨胀的葡萄干面包：
- **面团** = 宇宙空间本身
- **葡萄干** = 一个个星系（如银河系）
- 面团膨胀 → 所有葡萄干相互远离

你站在其中一颗葡萄干上，会看到其他所有葡萄干都在离你远去——这和你这颗葡萄干有没有动没关系，是面团本身在膨胀。

### 宇宙组成（精确比例）

| 成分 | 占比 | 说明 |
|------|------|------|
| 可见物质 | <5% | 恒星、行星、气体、尘埃 |
| 暗物质 | ~27% | 不发光但参与引力作用，支撑星系结构 |
| 暗能量 | ~68% | 推动宇宙加速膨胀，本质未知 |

### 关键时间线

1. **大爆炸**（138亿年前）：空间从奇点开始膨胀，产生时间和物质
2. **最初3分钟**：合成氢和氦，比例与理论精确吻合
3. **宇宙微波背景辐射**（1964年发现）：大爆炸的"余晖"，关键证据
4. **哈勃发现宇宙膨胀**：红移→星系都在远离，距离越远速度越快
5. **暗能量发现**（1998年）：宇宙膨胀正在加速，而非减速

### 宇宙对我们的影响（实用层面）

| 应用 | 来源 |
|------|------|
| GPS导航 | 需同时考虑相对论效应：卫星移动快时间变慢、离地球远引力弱时间变快。不考虑的话一天偏差十几公里 |
| 天气预报 | 气象卫星、遥感技术源自天体物理和太空探索 |
| Wi-Fi | 诞生于澳大利亚科学家对黑洞射电信号的研究 |
| 手机相机CMOS | 最早为太空望远镜研发 |
| 医用CT/MRI | 与天体粒子探测和信号处理技术同源 |

### 未解之谜

- **暗物质**：占27%，究竟是什么粒子？我们还不清楚
- **暗能量**：占68%，本质是物理学最大谜题
- **大爆炸之前**：时间概念可能不存在，"之前"这个问题本身可能无意义
- **宇宙终极命运**：持续加速膨胀（大撕裂）还是转为收缩（大挤压）？
- **多重宇宙、量子引力**：前沿理论

### 冷知识

**我们都是"星尘"**：身体里的碳、氧、铁等元素，都是在恒星内部通过核聚变锻造，在恒星爆炸时散入太空。血液里的铁，可能是几十亿年前一颗遥远恒星爆炸的遗物。
