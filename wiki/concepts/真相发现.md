# 真相发现

**Truth Discovery**：在多源矛盾数据下，**迭代逼近**"统计上共识真实"的算法范式——矛盾信息工程处置的最高层级。

## 核心算法

1. 给**所有来源**和**所有事实**赋予初始置信度
2. **迭代更新**：
   - 置信度高的来源支持的事实 → 提分
   - 能被多数高置信度来源支持的事实 → 其来源也提分
   - 反之亦然
3. **反复迭代**，直到收敛

## 典型算法

- **HITS / PageRank 类**：来源 ↔ 事实之间的迭代互证
- **Sums**、**AverageLog**、**Investment**、**PooledInvestment**
- **基于贝叶斯**：把来源可靠性作为先验

## 与 D-S 证据理论的关系

- D-S 理论提供**单次冲突度量**
- 真相发现提供**多轮迭代**机制
- 两者互补：D-S 量化 → 真相发现收敛

## 在矛盾处置 5 层中的位置

```
1. 表面裁决
2. 可靠性建模
3. 语义消解
4. 延迟裁决 + TMS
5. 真相发现 ← 本条
```

## 应用

- 众包数据真伪
- 知识图谱构建
- 假新闻识别
- 在线评价真实性
- 多源医疗证据整合
