# 夏普比率

**Sharpe Ratio**：衡量投资组合**每承担一单位总风险**所能获得的**超额收益**的经典指标。**由诺贝尔奖得主威廉·夏普提出**，是基金业绩评估的**最广泛使用指标**。

## 计算公式

```
Sharpe = (R_p - R_f) / σ_p
```

| 符号 | 含义 |
|------|------|
| R_p | 投资组合的预期收益 |
| R_f | 无风险收益 |
| σ_p | 投资组合收益的标准差（总风险）|

## 关键解读

### 1. "性价比"
- **每承担一单位风险，能拿到多少超额收益**
- 夏普 1.0 = 每承担 1% 风险，拿 1% 超额收益
- 夏普 2.0 = 每承担 1% 风险，拿 2% 超额收益（**优秀**）
- 夏普 0.5 = 每承担 1% 风险，只拿 0.5% 超额收益（**不值得**）

### 2. 实际数据参考

| 资产 | 长期夏普比率 |
|------|--------------|
| 美国大盘股 | 0.4-0.5 |
| 美国小盘股 | 0.5-0.6 |
| 美国长期国债 | 0.3-0.4 |
| 黄金 | 0.2-0.3 |
| 优秀对冲基金 | 0.8-1.5 |
| 顶级量化基金 | 1.5-2.5 |
| 传奇交易员（巴菲特等）| 0.7-0.8 |

### 3. 关键计算对比

| 基金 | 收益 | 标准差 | 夏普 |
|------|------|--------|------|
| A 激进基金 | 25% | 30% | (25-3)/30 = **0.73** |
| B 稳健基金 | 12% | 5% | (12-3)/5 = **1.80** |
| C 普通基金 | 15% | 15% | (15-3)/15 = **0.80** |
| D 神话基金 | 40% | 40% | (40-3)/40 = **0.93** |

**关键洞察**：
- A 基金收益高，但风险更高 → 夏普 0.73
- B 基金收益低，但风险更低 → 夏普 1.80（**最值**）
- **夏普率高 ≠ 收益高，而是风险调整后的"性价比"高**

## 夏普比率的变种

### 索提诺比率（Sortino Ratio）
```
Sortino = (R_p - R_f) / σ_downside
```
- **只计算下行风险**（亏损波动）
- 上行波动不算"风险"
- 更符合投资者直觉

### 卡玛比率（Calmar Ratio）
```
Calmar = 年化收益 / 最大回撤
```
- 用**最大回撤**代替标准差
- 直观反映"最坏情况下的痛苦"
- 适合厌恶大亏损的投资者

### 信息比率（Information Ratio, IR）
```
IR = α / Tracking Error
```
- 主动基金相对基准的**超额收益稳定性**
- **IR > 0.5** 算优秀主动管理
- 衡量"主动管理能力"

## 夏普比率的局限

| 局限 | 说明 |
|------|------|
| **正态分布假设** | 真实市场有肥尾 |
| **上下波动对称** | 实际亏损波动比盈利波动更"危险" |
| **标准差不稳定** | 极端事件会扭曲 |
| **不区分策略** | 同样的夏普可能来自不同策略 |
| **历史数据依赖** | 未来未必重复 |

## 关键洞察

> **夏普比率 = 风险的"价格"**——告诉你在给定风险下能拿多少收益
> **夏普比率的提升空间**：
> 1) 提高收益（难） 2) 降低风险（更可行，如分散化） 3) 用期权对冲（成本高）
> **优秀基金的夏普 = 1.0+**；**顶级量化 = 2.0+**；**传奇 = 3.0+**（索罗斯 1985-2010 约 2.5）
