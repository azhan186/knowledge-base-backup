# 形式系统

**Formal System**：由**公理**和**推理规则**构成的逻辑系统——哥德尔不完备定理的研究对象。

## 组成

- **形式语言**：符号、形成规则（合法公式）
- **公理集合**：不需要证明的基本命题
- **推理规则**：从前提推出结论的规则（如分离规则 modus ponens）
- **证明**：从公理出发、应用推理规则得到的公式序列

## 关键性质

| 性质 | 含义 |
|------|------|
| **可靠性（Soundness）** | 推出的都是真命题 |
| **完备性（Completeness）** | 所有真命题都可推出 |
| **一致性（Consistency）** | 不能同时推出 P 和 ¬P |
| **可判定性（Decidability）** | 存在算法判断任意命题是否可证 |

## 哥德尔的冲击

> 任何**足够强**（能编码基本算术）且**一致**的形式系统 F：
> 1. 必**不完备**——存在 F 内无法证明的真命题
> 2. F **无法在自身内部证明自身的一致性**

## 经典例子

| 形式系统 | 范围 |
|---------|------|
| **PA（皮亚诺算术）** | 自然数算术 |
| **ZFC（集合论）** | 现代数学基础 |
| **Propositional Logic** | 命题逻辑 |
| **Predicate Logic (FOL)** | 一阶谓词逻辑 |

## 在矛盾信息中的地位

形式系统是矛盾逻辑学研究的**载体**——逻辑矛盾的定义、一致性的判定、超一致逻辑的提出，都依赖形式系统框架。
