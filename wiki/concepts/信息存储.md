# 信息存储

**Information Storage**：把信息（文字、声音、图像、指令、数据等）转换成某种**物理状态或符号**，固定在一种**载体**上，使它能在以后被**读取、还原或使用**。

## 一句话原理

**编码（怎么写）+ 载体（写在哪）+ 存取方式（怎么读）** 三大要素决定存储方式。

## 历史四阶段

1. **手工时代**：结绳、岩画、羊皮纸
2. **机械时代**：印刷术、打孔卡
3. **磁时代**：磁带、磁盘（1950s）
4. **半导体时代**：闪存、SSD（2000s+）
5. **前沿**：DNA、量子、全息

## 存储金字塔

```
寄存器 (ns)  ← 最快
缓存 (10ns)
内存 (100ns)  DRAM
闪存 (μs)  SSD
磁盘 (ms)  HDD
磁带 (s)  ← 容量大
云存储
```

## 物理核心约束

- **兰道尔原理**：擦 1 比特至少耗能 k_B·T·ln 2（室温 2.8×10⁻²¹ J）
- **信息熵**：无损压缩极限 = 信息熵
- **三难**：速度 / 容量 / 成本不可兼得

## 关键子领域

- **机械硬盘（HDD）**——磁介质代表
- **SSD**——闪存
- **RAID**——多盘组合管理
- **DNA 存储**——生物介质
- **量子存储**——前沿
