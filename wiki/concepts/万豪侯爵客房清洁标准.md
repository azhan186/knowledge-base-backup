# 万豪侯爵客房清洁标准

> 上海万豪侯爵酒店客房清洁的完整标准作业体系（基于 Floor 系列 43 份 SOP 提炼）。

## 完整清洁流程（11 步）

参考 Floor_026（房间清洁）+ Floor_027（铺床）+ Floor_028（浴室清洁）：

1. **领取任务表，检查房态**（Floor_026, Floor_029 优先打扫）
2. **进入前**：
   - 敲门三次，每次隔 2 秒（Floor_005）
   - 自我介绍："您好，客房服务员"
   - 视觉检查房间是否有可疑摄像头（Floor_014）
   - 遇 DND（请勿打扰）不得强行进入（Floor_030）
3. **打开所有窗帘** + 测试所有灯
4. **撤布草**（床单、枕套、被套、毛巾等）
5. **除尘 + 擦桌 + 吸尘**（Floor_020 吸尘器规范）
6. **浴室清洁**（Floor_028）：
   - 马桶消毒
   - 淋浴/浴缸去污
   - 洗手台/镜面
   - 瓶具清洁（Floor_041）
7. **铺床**（Floor_027 标准步骤）
8. **补充客用品**：
   - 浴室：沐浴用品（Floor_042）、瓶具（Floor_041）
   - 迷你吧（Floor_037）
   - 拖鞋、擦鞋布、洗衣袋等
9. **自检**（Floor_013 关门打扫标准）
10. **关闭房门**（Floor_013）
11. **上报**：
    - 上午/下午差异房报告（Floor_025）
    - 损坏/遗失报告（Floor_033, Floor_035）

## 房态类型与处理

| 房态 | 缩写 | 处理 |
|------|------|------|
| 住客房 | OD (Occupied Dirty) | 按 DND 状态决定 |
| 退客房 | VD (Vacant Dirty) | 完整 11 步清洁 |
| 空房 | VC (Vacant Clean) | 仅检查 + 补品 |
| 长住房 | LS (Long Staying) | 周一全面清洁 |
| 维修房 | OOO (Out of Order) | Floor_043 程序 |
| 贵宾房 | VIP | Floor_039 + Floor_023 鲜花 |
| 参观房 | | Floor_038 |
| 自放房 | | Floor_044 资格政策 |

## 优先打扫顺序（Floor_029）

按前台指令和 VIP 等级：

1. **VIP 房**（先于一切）
2. **续住房**（保证客人在房时干净）
3. **提前退房**（前台优先放房）
4. **常规退房**
5. **空房检查**

## 夜床服务（Turn-down Service, Floor_031）

- **时间**：每晚 19:00 后开始
- **开床方式**：
  - 折角 / 开一边 / 开两边（按房型）
- **补充**：晚安卡、水、拖鞋
- **降灯光、拉窗帘**

## 关键用品

- 床品：被单、枕套、被套（布草由 LDY 部门负责洗涤）
- 浴室用品：洗发水、沐浴露、润肤露、肥皂（Floor_042 补充标准）
- 迷你吧：酒水、零食（Floor_037 每日检查）
