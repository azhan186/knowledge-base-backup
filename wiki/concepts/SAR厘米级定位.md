# SAR 厘米级定位

**Synthetic Aperture Radar (SAR) for RFID Positioning**：利用**移动中阅读器的高精度相位信息**，通过合成孔径原理，将标签定位精确到**厘米级**的研究前沿。

## 物理原理

### 相位-距离敏感性
- UHF 频段（≈ 900 MHz）波长 ≈ 33 cm
- **距离变化几厘米 → 相位旋转 360°**
- 也就是说，**相位是距离的"放大镜"**

### 合成孔径
- 单个阅读器孔径小，分辨率有限
- **移动阅读器** → 在不同位置多次测量 → 等效于大孔径
- 类似于雷达中的 SAR 技术

### 算法流程

```
阅读器位置 1：测相位 φ₁
阅读器位置 2：测相位 φ₂
阅读器位置 3：测相位 φ₃
...
应用 SAR 算法 → 重建标签位置
```

## 系统极限

### 多径反射畸变
- 在工业货架、仓库等复杂环境
- 电磁波经过**多次反射**到达标签
- 多径分量**严重畸变相位**
- 简单的"相位 vs 距离"线性关系**失效**

### 解决方案

| 方案 | 原理 | 现状 |
|------|------|------|
| **深度学习剥离** | 用神经网络学习多径特征 | 学术研究热点 |
| **MIMO 多天线** | 多角度测量增强鲁棒性 | 商业化中 |
| **超宽带（UWB）| 短脉冲天然抗多径 | 已是商业方案（DecaWave 等）|
| **环境指纹** | 预先采集环境，定位时匹配 | 部署复杂 |

## 应用场景

- 仓储盘点（精确定位到货架某层某列）
- 零售防盗（顾客取走商品的精确位置）
- 医院资产管理（移动设备精确追踪）
- 智能制造（生产线物料精确位置）

## 关键洞察

> **相位 = RFID 定位的"金矿"**——分辨率远超 RSSI（信号强度）
> **多径 = RFID 定位的"毒药"**——实际工业环境致命
> **深度学习 = 当前最有希望的解药**——但尚未完全成熟
