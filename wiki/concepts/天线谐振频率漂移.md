# 天线谐振频率漂移

**Antenna Resonance Frequency Drift**：当天线周围环境介电常数改变时（如水接近），天线的谐振频率偏移，导致**标签在原设计频率上失谐**的现象。

## 物理原理

- 天线谐振频率：f = c / (λ × √ε_eff)
- ε_eff = 有效介电常数（周围介质的贡献）
- **当 ε_eff 增大（介电常数变高）→ f 下降 → 漂出原设计频段**

## 水的破坏（具体场景）

水的相对介电常数 ≈ 80（远高于空气的 1）
- 标签天线周围有水 → 有效介电常数提高
- 谐振频率从 UHF 860-960 MHz 向低频漂
- 严重时漂出 100+ MHz → **彻底失谐**

## 后果

即使有足够能量：
- 标签在阅读器工作频率上不能有效吸收
- 反射调制深度大幅降低
- 标签"看到"阅读器，阅读器却"看不到"标签

## 应对方案

| 方案 | 原理 |
|------|------|
| 抗水涂层 | 隔离水分子与天线直接接触 |
| 平台天线 | 利用间隔层缓冲介电变化 |
| 宽带天线 | 容忍一定频率漂移 |
| 频率补偿 | 主动检测环境并调整 |
| 选 LF/HF | 磁场对介电变化不敏感 |

## 与介电损耗的关系

两者都是"水破坏"机制，但：
- **介电损耗**：能量被水吸收
- **频率漂移**：能量被反射但频率不匹配

需要**同时**应对——单纯增强功率或选 LF/HF 都不能完全解决 UHF 在水环境的根本问题。
