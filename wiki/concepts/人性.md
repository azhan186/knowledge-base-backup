# 人性（Human Nature）

## 核心定义

"人之所以为人"的那些与生俱来、又会被后天塑造的思维、情感和行为倾向的总和。

## 三层理解

- **本能层**：饿了想吃、困了想睡、遇危险会害怕 —— 生物性，求生存
- **情感与社会层**：渴望被爱、害怕孤独、会嫉妒、会同情、在意公平
- **理性与意义层**：问"为什么活着"、克制冲动、会内疚、为长远目标牺牲眼前快乐

## 智能手机"操作系统"类比

- **出厂设置** = 本能（求生欲、情绪反应、归属需求）
- **安装应用** = 后天塑造（文化、家庭、教育）
- **系统限制** = 物种边界（语言能力是预装模块）
- **更新与兼容** = 可塑但核心架构不变

## 历史演变

1. **古代善恶之争**：孟子（性善）、荀子（性恶）、告子（流水）、柏拉图灵魂三分、基督教原罪
2. **近代理性与白纸**：霍布斯（自私/丛林）、洛克（白纸）、卢梭（高贵的野蛮人/伊甸园）
3. **现代科学登场**：达尔文进化适应器、弗洛伊德潜意识、人本主义自我实现、进化心理学遗产品

## 人性三问：霍布斯 vs 卢梭 vs 情境论

- **霍布斯丛林**：无社会约束 → 人性自私 → "所有人反对所有人"
- **卢梭伊甸园**：原始人是高贵的野蛮人，社会才催生恶
- **现代情境论**：人性更像点唱机，情境是点播的按钮

**斯坦福监狱实验**：普通人，在允许虐待的匿名情境和权力结构下，会做出难以置信的事。恶是情境诱导出的人性倾向。

## 共情的双重起源

- **进化硬件**：镜像神经元（内部模拟对方体验）、情绪传染（婴儿自发共情）、亲缘利他与互惠利他
- **文化软件**：文化决定共情的方向盘（对谁）和油门（表达方式）

共情 = 大脑预装的"同情心引擎"，文化决定方向盘往哪转、油门踩多大。

## 普遍人性结构的证据：乔姆斯基普遍语法

所有人类语言底层共享深层结构原则（名词/动词范畴、结构依赖、递归思想），说明人类大脑天生预装了一套"语言习得装置"。这是"普遍人性结构"在认知上的明确证据。

## 性别心理差异的互为因果

两性差异 = 用同一盒乐高（共同人性），在不同图纸（社会角色）下，拼出带有性别特色但重叠巨大的两个模型。

## 理性人假设的四大违背

1. **损失厌恶**：失100元比得100元痛苦两倍 → 禀赋效应、捂着亏损股不卖
2. **框架效应**：说法不同选择就变 → "成功率90%" vs "死亡率10%"
3. **心理账户**：钱在脑里分不同账户 → 工资省着花，横财挥霍
4. **双曲贴现**：对眼前诱惑毫无抵抗力 → 拖延症、储蓄不足的根

## AI融合四层次的人性挑战

1. **外部增强**：记忆存云，"我的知识"边界模糊
2. **赛博格化**：编辑心理反应，"未经修饰的情绪自我"真实性成疑
3. **心智上传**：摧毁生物基础，不再有饥饿恐惧性欲
4. **共生伦理**：AI伴侣让人撤退到完美共情的虚拟茧房，丧失真实亲密能力

**关键挑战**：人性诞生于匮乏、脆弱与必死躯壳。当技术移除这些根基，"我们是谁"的坐标轴可能消失。
