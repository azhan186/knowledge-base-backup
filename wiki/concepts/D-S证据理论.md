# D-S 证据理论

**Dempster-Shafer Theory of Evidence**：用于多源信息融合的信度分配框架，区别于经典概率论。

## 核心创新

- 信任度（mass function）可分配给**复合假设**（如 {A,B}），不限于单事件
- 能区分**不确定性（ignorance）**和**概率均等**——概率论做不到
- 总和永远 = 1

## 关键概念

| 概念 | 含义 |
|------|------|
| **框架（Frame of Discernment）** | 所有可能假设的集合 Ω |
| **mass 函数 m(X)** | 分给假设 X 的信任度，X ⊆ Ω |
| **Belief (Bel)** | 信任函数，所有包含 X 的子集信任度之和 |
| **Plausibility (Pl)** | 似然函数，不与 X 矛盾的信任度上限 |
| **冲突系数 K** | 量化两条信息矛盾程度的统计量，0-1 |

## 冲突系数 K

> **K = Σ m₁(X) × m₂(Y)**，对所有满足 **X ∩ Y = ∅** 的集合求和

含义：信息 1 信任的 X 与信息 2 信任的 Y **完全互斥**时，视为冲突。

## Zadeh 反例（鲁棒性问题）

经典反例：
- 医生 1：0.99 → {X}, 0.01 → {Y}
- 医生 2：0.99 → {Z}, 0.01 → {Y}

X-Z 冲突贡献 0.99×0.99 ≈ 0.98 → "几乎完全矛盾"

**反直觉**：诊断**针对不同疾病**，应视为"信息针对不同对象"而非"激烈冲突"。

→ 学术界称 **"证据冲突度量鲁棒性问题"**；改进方法：冲突比、组合后信度分配的距离度量等。

## 应用领域

- 多传感器融合
- 医学诊断
- 情报分析
- 自动驾驶感知
- 法律证据评估
