# 卫星

## 一句话定义

卫星是围绕行星或矮行星运行的天体，本身不直接绕恒星公转。

## 核心概念

### 引力层级结构

宇宙天体的"引力层级游戏"：
- **恒星**：中央国王（如太阳），自己发光发热
- **行星**：直接效忠恒星的领主（如地球），绕恒星公转
- **卫星**：效忠行星/矮行星的随从，绕行星转，随行星一起绕恒星转

> 关键：无论大小，只要绕行中心是行星或矮行星，就是卫星。忠诚对象是领主而非国王。

### 历史里程碑

| 时间 | 事件 |
|------|------|
| 伽利略之前 | 人类只知道月球一颗卫星 |
| 1610 | 伽利略发现木星四颗卫星（伽利略卫星），敲响地心说丧钟 |
| 开普勒命名 | 拉丁语"satellite"=侍从/护卫 |
| 17-19世纪 | 发现土卫六、天王星卫星等 |
| 旅行者号时代 | 发现木卫一活火山、木卫二冰下海洋、土卫六浓密大气 |
| 至今 | 太阳系确认200+颗卫星，数量持续刷新 |

### 代表性卫星

- **月球**：地球唯一卫星，潮汐成因，稳定地球自转轴
- **木卫二（欧罗巴）**：冰下海洋，地外生命最热门候选地之一
- **土卫二（恩克拉多斯）**：冰下海洋，羽状喷流含水汽和有机分子
- **土卫六（泰坦）**：唯一有稠密大气和甲烷湖泊的卫星，早期地球化学实验室
- **卡戎**：冥王星最大卫星，冥王星降级为矮行星后依然拥有随从

### 卫星的价值

1. **潮汐与生命起源**：潮间带被认为是生命从海洋走向陆地的关键试验场
2. **气候稳定器**：大卫星（如月球）防止行星自转轴剧烈摆动，维持规律季节
3. **太空实验室**：木卫二/土卫二冰下海洋是新药、化学反应的独特环境
4. **深空跳板**：月球是人类迈向火星乃至更远深空的第一块垫脚石

### 前沿挑战

- **海洋世界生命之谜**：欧罗巴快船等任务将分析木卫二羽状喷流，寻找化学指纹
- **重新定义"可居住"**：卫星可以靠母行星潮汐加热，在宜居带外维持液态水
- **卫星边界争议**：土星环与卫星相互转化；临时卫星（2020 CD3）绕转数年就离开
- **系外卫星（Exomoon）**：下一代望远镜的重要目标
