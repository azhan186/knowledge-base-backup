# EPC Gen2 协议

**EPC Class-1 Generation-2（也称 ISO/IEC 18000-63）**：UHF 频段 RFID 的**事实标准**，主导全球供应链和零售业的标签通信协议。

## 协议地位

- 2004 年由 EPCglobal 发布
- 2006 年成为 ISO/IEC 18000-63 国际标准
- 至今仍是 UHF RFID 的**唯一主流协议**
- 全球数十亿张标签遵守此协议

## 命令集（"最少承诺"哲学）

| 命令 | 功能 |
|------|------|
| Select | 选择特定标志位的标签子集 |
| Query | 开始一轮盘存（含 Q 值参数）|
| QueryAdjust | 调整 Q 值（碰撞/空闲时）|
| QueryRep | 重复上次盘存 |
| ACK | 确认收到标签响应 |
| NAK | 否定确认 |
| Req_RN | 请求新随机数（句柄）|
| Read | 读存储区 |
| Write | 写存储区 |
| Kill | 灭活标签（32位密码）|
| Lock | 锁定存储区 |
| Access | 进入特权状态 |

**核心观察**：几乎所有命令都是**无状态宏指令**，阅读器不记忆标签细节。

## 协议栈分层

| 层 | 职责 |
|----|------|
| 物理层 | PIE 编码（下行）/ FM0/Miller（上行）/ Tari/Blf |
| 标签识别层 | 防碰撞（Q 算法 + 时隙 Aloha）|
| 标签存储层 | 4 个存储区（Reserved/EPC/TID/User）|
| 安全层 | 访问/灭活密码、Lock 位 |
| 会话层 | 4 个会话标志（持久性梯度）|

## 关键设计原则

- **复杂度留给阅读器，简单留给标签**
- 阅读器可升级、可插电、可连接算力
- 标签一旦部署就难更改（成本/环境限制）
- 这是**摩尔定律 + 成本非对称性**的深刻洞察
