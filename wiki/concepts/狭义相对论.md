# 狭义相对论

**Special Relativity (1905)**：爱因斯坦关于**匀速运动**的相对性理论。

## 一句话原理

> 物理定律在所有惯性参考系中都相同 + **光速不变** → 时间膨胀 + 长度收缩 + 质能等价

## 两大基本假设

- **相对性原理**：所有惯性系等价
- **光速不变**：c = 299,792,458 m/s 恒定

## 三大推论

| 推论 | 公式 |
|------|------|
| **时间膨胀** | Δt' = γΔt, γ = 1/√(1-v²/c²) |
| **长度收缩** | L' = L/γ |
| **质能等价** | **E = mc²** |

## 实验验证

- μ 子寿命延长（宇宙射线）
- 粒子加速器（LHC）
- **GPS 原子钟**（每天 +38 μs，必须修正！）
- 核磁共振

## 与广义相对论关系

- 狭义：**惯性参考系**（无引力）
- 广义：**任意参考系**（含引力）
- 广义 ≈ 狭义 + 等效原理 + 弯曲时空
