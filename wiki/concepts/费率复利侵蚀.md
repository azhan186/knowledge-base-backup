# 费率复利侵蚀

**Fee Compounding Erosion**：长期投资中，**看似微小的费率差异**通过复利效应造成**巨大财富差距**的现象。**这是主动基金长期跑输指数的核心机制之一**。

## 关键计算

**同样 6% 毛收益，40 年投资 100 万**：

| 费率 | 终值 | 差距 |
|------|------|------|
| **0.15%**（指数基金）| **205 万** | - |
| **0.5%**（低费率主动）| **178 万** | -27 万 |
| **1.0%**（平均主动）| **149 万** | -56 万 |
| **1.35%**（高费率主动）| **124 万** | **-81 万** |

**结论**：40 年累计，1.35% vs 0.15% 费率差 = **损失 81 万**（差近一倍）

## 复利机制

```
年末价值 = 上一年末价值 × (1 + 毛收益 - 费率)
```

- 费率每年从收益中扣除
- 被扣除部分**永远无法再产生收益**
- 复利效应下，差距指数级扩大

## 完整费用构成

| 费用 | 显性/隐性 | 典型范围 |
|------|----------|----------|
| **管理费** | 显性 | 0.5-1.5% |
| **托管费** | 显性 | 0.1-0.2% |
| **销售服务费** | 显性 | 0-0.8% |
| **申购费**（前端）| 显性 | 0-1.5% |
| **赎回费** | 显性 | 0-1.5% |
| **交易佣金** | 隐性 | 0.1-0.5% |
| **买卖价差** | 隐性 | 0.05-0.2% |
| **市场冲击** | 隐性 | 取决于规模 |
| **软美元** | 隐性 | 灰区 |

## 关键洞察

> **费率 = 你为"试图跑赢市场"付出的"保险费"**
> 问题是：**80%+ 的主动基金实际跑输了**——保险费白交

## 实证数据

| 资产规模 | 平均费率 |
|----------|----------|
| < 10 亿 | 1.30% |
| 10-50 亿 | 1.10% |
| 50-100 亿 | 0.95% |
| > 100 亿 | 0.85% |
| 指数基金（任意）| 0.15% |

**结论**：**基金规模越大，费率越低**——规模效应
