# 分镜脚本标准 v2.0

> **版本**：v2.0｜**日期**：2026-05-15｜**状态**：✅ 确立
> **来源**：用户提供的AI短剧分镜框架 + 专业分镜15要素标准

---

## 一、基础规格（每部分镜必填）

| 要素 | 选项 | 说明 |
|------|------|------|
| **画面比例** | 16:9（横屏）/ 9:16（竖屏） | 决定发布平台 |
| **视觉风格** | 电影写实/国风动漫/皮克斯3D/赛博朋克 | 决定全剧基调 |
| **光照色调** | 温馨暖黄/悬疑冷蓝/胶片质感/霓虹赛博 | 决定情绪底色 |
| **画质标准** | 8K/4K/电影感 | --ar 16:9 --style raw |

---

## 二、15个核心要素（每镜必填）

| # | 要素 | 说明 |
|----|------|------|
| 1 | **镜头编号** | 场号—镜头号（如 1A-01） |
| 2 | **画面草图/构图描述** | 主体 + 动作/表情 + 环境/背景 |
| 3 | **景别** | ELS/FS/MS/MSU/CU/ECU |
| 4 | **摄影机角度与高度** | 平视/俯视/仰视/荷兰角 |
| 5 | **镜头运动** | 推/拉/摇/移/手持/固定 |
| 6 | **镜头时长** | 秒（如 5s） |
| 7 | **构图和框架** | 三分法/黄金分割/前景/背景 |
| 8 | **对白/台词** | 角色对话内容 |
| 9 | **旁白/画外音** | 内心独白/画外音 |
| 10 | **音乐/音效** | BGM + 环境音 |
| 11 | **情绪/氛围** | 紧张/孤独/压抑/悬疑 |
| 12 | **色调/灯光** | 主色调 + 光源方向 |
| 13 | **特效/Chest** | VFX + 视觉特效说明 |
| 14 | **参考画面/提示词** | AI生图提示词（见下方公式） |
| 15 | **备注/现场注意** | 拍摄注意事项/负面提示词 |

---

## 三、AI提示词万能公式

```
主体 + 动作/表情 + 环境/背景 + 景别/构图 + 光线/色彩 + 风格/画质
```

### 公式拆解

| 模块 | 示例 |
|------|------|
| **主体** | 一位25岁的亚洲女性，穿深色风衣 |
| **动作/表情** | 泪水滑落脸颊，表情隐忍而悲伤 |
| **环境/背景** | 雨天的落地窗，远处霓虹灯闪烁 |
| **景别/构图** | 侧面特写，三分法构图 |
| **光线/色彩** | 柔和的冷蓝侧光，赛博朋克氛围 |
| **风格/画质** | cinematic lighting, photorealistic, 8K, --ar 16:9 --style raw |

### 示例对比

| 等级 | 提示词 |
|------|--------|
| ❌ 基础版 | 一个悲伤的女人 |
| ✅ 进阶版 | 一位25岁的亚洲女性，侧面特写，泪水滑落脸颊，表情隐忍而悲伤，背景是雨天的落地窗，柔和的冷蓝侧光，电影感色调，高细节，4K |

## 四、景别说明

| 代码 | 名称 | 说明 | AI提示词 |
|------|------|------|----------|
| ELS | 远景/Establishing Shot | 展示大环境，交代地点 | wide establishing shot |
| FS | 全景/Full Shot | 角色全身 | full shot |
| MS | 中景/Medium Shot | 腰部以上 | medium shot |
| MCU | 近景/Medium Close-Up | 胸部以上 | medium close-up |
| CU | 特写/Close-Up | 头部 | close-up |
| ECU | 大特写/Extreme Close-Up | 眼睛/手部细节 | extreme close-up |

---

## 五、摄影机角度

| 类型 | 说明 | 情绪效果 |
|------|------|----------|
| 平视 | 与角色眼睛同高 | 客观、中立 |
| 俯视 | 居高临下 | 渺小、无力感 |
| 仰视 | 低位向上 | 权威、压迫感 |
| 荷兰角 | 倾斜镜头 | 不安、诡异 |

---

## 六、镜头运动

| 符号 | 运动 | 说明 |
|------|------|------|
| 🔒 | 固定 | 无运动 |
| ➡️ | 推 | 靠近角色，情绪递增 |
| ⬅️ | 拉 | 后撤，空间放大 |
| ↔️ | 摇 | 左右横摇展示环境 |
| 🎥 | 移/跟 | 随角色移动 |
| 📷 | 手持 | 不稳定，真实感 |

---

## 七、镜头时长参考

| 场景类型 | 时长 |
|----------|------|
| 大全景（信息量大） | 8-12秒 |
| 中近景对话 | 4-6秒 |
| 动作快切 | 2-4秒 |
| 特写/情绪镜头 | 3-5秒 |

---

## 八、框架构图原则

- **三分法**：将画面横竖各分三等份，主体放在交叉点上
- **黄金分割**：0.618比例分割画面
- **前景/背景**：利用纵深增加层次
- **视线留白**：给角色视线方向留出空间

## 九、AI工具工作流

### 推荐工作流

```
1. 文生图定基调 → 2. 图生视频动起来 → 3. 剪辑组装 → 4. AI配音配乐 → 5. 最终合成
```

### 主流工具对比

| 工具 | 优势 | 适合场景 |
|------|------|----------|
| **可灵 (Kling)** | 写实风格强，语义理解好 | 追求真实感的短剧 |
| **Runway Gen-3** | 艺术创作自由度大，控制选项精细 | 创意、风格化强的片段 |
| **Luma Dream Machine** | 生成速度快，电影感镜头流畅 | 快速产出第一手素材 |
| **Sora / Veo 2** | 画质顶级，时长和一致性有突破潜力 | 追求高质量长镜头的项目 |

### 工具选择建议

- **写实短剧** → 可灵
- **风格化/创意** → Runway
- **快速原型** → Luma
- **高质量长镜头** → Sora/Veo 2

## 十、负面提示词（避坑用）

在AI生图时加入负面提示词，避免常见问题：

```
不要扭曲的手，不要多余的手指，不要变形的脸，不要动画感，不要低分辨率
```

英文版：
```
distorted hands, extra fingers, deformed face, cartoonish, low resolution, blurry
```

---

## 十一、角色一致性

固定角色形象，确保多张图中主角长相、服装一致：

- **Midjourney** → 使用 `--cref`（角色参考）+ `--seed`（种子）
- **可灵/Runway** → 上传参考图绑定角色
