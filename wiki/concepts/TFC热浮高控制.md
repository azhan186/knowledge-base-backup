# TFC 热浮高控制

**Thermal Fly-Height Control（热浮高控制）**：用磁头内置微型加热器动态调节读/写元件飞行高度的精密控制技术。

## 背景

现代 HDD 磁头与盘面距离仅约 **1-3 纳米**（头发丝直径的 1/1000）。这个极小间隙是密度提升的关键——飞得越低 → 读出信号越强、密度越高。但同时也带来挑战：

- **碰撞风险**：任何凸起/污染都可能"撞头"
- **写/读需求矛盾**：
  - 写入：需要更强场 → 较远飞行距离
  - 读出：需要极高 SNR → 必须飞得更低

TFC 是用"热"解这道题。

## 机制

### 物理原理

- 磁头底部的**读/写元件本身就是微型电阻加热器**
- 通入电流 → 元件局部热膨胀 → 读/写元件向盘面"凸起"零点几纳米
- **飞行高度 ≈ 静态基础高度 − TFC 凸起量**

### 写/读双模式

- **写入模式**：低 TFC 功率 → 飞行较高 → 写磁场强、不易撞头
- **读出模式**：高 TFC 功率 → 飞行极低 → 读出信号强、SNR 高
- **切换时间**：**微秒级**动态切换

## 校准挑战

- **每个磁头、每个半径区域单独校准**
- 错误 TFC 功率 →
  - 功率不足：读信号过弱
  - 功率过大：磁头"撞地"

## 热衰减（Thermal Decay）——正反馈风险

**正反馈循环**：

1. 介质上微小凸起或污染物
2. 磁头以最小间隙飞行 → 局部摩擦加热
3. 磁头局部热膨胀 → 飞行高度进一步降低
4. 凸起/污染点更接近磁头 → 摩擦加热更剧烈
5. 持续正反馈 → **瞬态热扰动甚至头盘接触**

**后果**：磁头与介质损坏，是 HDD 物理性失效的重要原因之一。

## 工程意义

- 是 HDD 在皮米/纳米尺度控制的**关键技术**
- 让读写模式物理参数解耦（用温度而非机械移动）
- 体现"用热解决机械难题"的工程哲学
