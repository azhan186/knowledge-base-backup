# 介电损耗

**Dielectric Loss**：水分子（强极性分子）在高频电场下随场旋转摩擦，将电磁能量不可逆转化为热量的物理现象。**这是 RFID 标签"怕水"的根本原因**。

## 物理机制

1. **水分子是强极性分子**：在外电场作用下，正负电荷中心会分离
2. **UHF 频段的高频电场**：水分子随电场剧烈旋转（每秒数十亿次）
3. **摩擦产热**：分子间碰撞将电磁能不可逆转化为热能
4. **趋肤深度极浅**：电磁波还没穿透水珠，能量就被消耗殆尽

## 两个具体后果

### 1. 能量衰减
- UHF 频段，水的介电损耗是空气的数百倍
- 标签在水中/含水物体表面 → 接收的能量远小于设计值
- 严重时标签"失明"，完全不应答

### 2. 频率漂移
- 标签天线周围的水会**提高有效介电常数**
- 谐振频率向低频漂移（UHF 标签可能漂出 860-960 MHz 频段）
- **标签在目标频率上彻底失谐** → 即使有足够能量也无法有效通信

## 实际影响

| 场景 | 后果 |
|------|------|
| 湿布草洗涤 | UHF 标签失明 → 必须用近场 + 平台天线 |
| 饮料瓶/罐头 | 必须设计成"悬浮架空"或带吸波层 |
| 活体动物标签 | 选 LF/HF（磁场对水穿透损耗小）|
| 食品溯源 | 冷冻/含水食品选 HF 或带抗水涂层 |
