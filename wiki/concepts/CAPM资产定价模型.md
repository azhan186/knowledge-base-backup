# CAPM 资产定价模型

**Capital Asset Pricing Model (CAPM)**：1964 年由威廉·夏普（William Sharpe）基于马科维茨投资组合理论发展的**单因子资产定价模型**。**它回答了一个核心问题：资产的"合理预期收益"应该是多少？**

## 核心公式

```
E(R_i) = R_f + β_i × (E(R_m) - R_f)
```

| 符号 | 含义 |
|------|------|
| E(R_i) | 资产 i 的预期收益 |
| R_f | 无风险收益（如同业拆借利率）|
| β_i | 资产 i 相对市场的 Beta（系统性风险）|
| E(R_m) | 市场组合的预期收益 |
| (E(R_m) - R_f) | 市场风险溢价（MRP）|

## 关键解读

### 1. 资产的"应有"收益
- 投资者承担**系统性风险**应得的补偿
- 承担多少市场风险（β）→ 拿多少风险溢价
- **承担非系统性风险不获得额外收益**（理论上可分散掉）

### 2. Beta 的几何意义
- 资产收益对市场收益的**回归系数**
- β = 1：与市场同涨同跌
- β > 1：比市场更激进（如行业 ETF）
- β < 1：比市场更稳健（如公用事业）
- β = 0：与市场无关（如现金）
- β < 0：与市场反向（如反向 ETF）

### 3. 证券市场线（SML）
```
收益
  ↑
  │           ╱ SML
  │         ╱
  │       ╱
  │     ╱
  │   ╱
  │ ╱
  └─────→ β
       R_f
```

- 在 SML **上方**：α > 0（被低估）
- 在 SML **下方**：α < 0（被高估）
- 在 SML **线上**：α = 0（公允定价）

## 关键假设

| 假设 | 现实偏离 |
|------|----------|
| 投资者理性、风险厌恶 | 实际有行为偏差 |
| 单一持有期 | 实际多期投资 |
| 可以无风险利率借贷 | 实际有借贷限制 |
| 没有税收和交易成本 | 实际有 |
| 所有投资者信息相同 | 实际信息不对称 |
| 资产可无限分割 | 实际有最小交易单位 |

## CAPM 的实际应用

### 1. 计算合理预期收益
- 给定公司 β = 1.3，无风险利率 3%，市场风险溢价 6%
- E(R) = 3% + 1.3 × 6% = **10.8%**
- → 公司合理的预期年化收益是 10.8%

### 2. 业绩评估
- 实际收益 - CAPM 预期收益 = **Alpha**
- Alpha > 0：基金经理创造价值
- Alpha < 0：基金经理毁灭价值

### 3. 资本预算
- 企业评估投资项目时，用 CAPM 算**权益资本成本**

## 局限

| 局限 | 后果 |
|------|------|
| **单因子** | 不能解释价值股 vs 成长股的差异 |
| **β 不稳定** | 同一资产不同时期 β 差异大 |
| **市场组合不可观测** | 实际用指数代替 |
| **忽略其他风险** | 流动性风险、通胀风险等 |

## 后续发展

- **Fama-French 三因子**（1993）：市场 + 市值 + 价值
- **Carhart 四因子**（1997）：再加动量
- **Fama-French 五因子**（2015）：再加盈利 + 投资
- **多因子模型**成为现代投资的主流

## 关键洞察

> **CAPM 是"金融学的爱因斯坦方程"**——简洁、优美、不完全准确
> 即使不完美，**仍然是思考"风险-收益关系"的起点**
