# 反向散射调制

**Backscatter Modulation**：无源 RFID 标签不主动发射电磁波，而是通过**改变天线匹配阻抗**来**调制入射波的反射强度**，从而向阅读器传回数据。

## 物理本质

- 标签芯片通过切换两种反射状态：让反射波**一会儿强、一会儿弱**
- 这就像手电筒照射一面小镜子，用手快速晃动镜子，反射光一闪一闪
- **闪动本身就携带了信息**

## 雷达方程（UHF 频段）

```
P_recv ∝ (P_trans · G_reader² · λ² · σ_tag) / d⁴
```

| 符号 | 含义 |
|------|------|
| P_recv | 阅读器接收信号功率 |
| P_trans | 阅读器发射功率 |
| G_reader | 阅读器天线增益 |
| λ | 波长 |
| σ_tag | 标签能调节的雷达截面积变化量 |
| d | 距离 |

**关键洞察**：分母是 **d⁴**——信号衰减极为剧烈。
- 手机通信通常只与距离平方（d²）成反比
- **UHF RFID 几米到十几米的距离极限，根源就在这 d⁴**

## 双向用途

1. **传统**：被视作"信号"用
2. **前沿**（专家篇）：整流倍压电路的强非线性产生高次谐波 → 利用谐波相位对负载变化的极高灵敏度 → **在谐波频率反向散射**，避开基波强自干扰和多径
