# 骨折机制

> 骨科基础知识，骨头的完整性或连续性被破坏

## 核心定义

骨折 = 骨头的完整性或连续性被破坏

外力超过骨头承受极限 → 裂缝或完全断开，骨膜撕裂 → 剧烈疼痛、肿胀、淤血。

## 核心特征

- **骨膜**：富含血管和神经，骨折时被撕裂是疼痛根源
- **自愈能力**：骨头是活的，能长出"骨痂"自我修复
- **愈合条件**：稳定的环境，让自愈顺利进行

## 骨折分类（按形态）

| 类型 | 说明 |
|------|------|
| **完全骨折** | 断端完全断开，错位 |
| **青枝骨折** | 折而不断，骨膜完整（小兒常见）|
| **骨裂/不全骨折** | 仅一道微小裂纹 |
| **开放骨折** | 骨头戳出皮肤 |
| **不愈合** | 约5%-10%，愈合困难 |

## 历史发展

1. **古代**：手摸心会 + 夹板固定（中医正骨千年传统）
2. **近代**：石膏绷带（马蒂森）+ X射线（伦琴，1895）
3. **现代**：手术内固定（无菌技术+麻醉，钢板螺钉、髓内钉）
4. **当代**：微创接骨板、3D打印定制植入物、生物制剂（干细胞、生长因子）

## 处理原则

> **复位 → 固定 → 功能锻炼**

### 院前急救
- 不移动伤肢，就地取材简易固定
- 脊柱骨折可疑 → 严禁随意搬动

### 诊断选择
- **X光**：首选，适合多数骨折
- **CT**：复杂部位（关节内、脊柱、骨盆）
- **MRI**：观察软组织（韧带、半月板）

### 治疗选择

| 方案 | 适用情况 |
|------|----------|
| 保守治疗 | 无移位/稳定型骨折，手法复位+石膏固定 |
| 手术治疗 | 不稳定骨折、关节内骨折、开放骨折 |

### 康复
- "三分治疗，七分练"
- 循序渐进：肌肉收缩 → 关节活动度 → 负重行走

## 前沿挑战

- **保守 vs. 手术边界**：锁骨、脊椎压缩骨折等灰色地带
- **骨质疏松性骨折**：老人"人生最后一次骨折"，1年死亡率20%-30%，需骨科-老年科联动
- **骨不愈/骨缺损**：BMP生物制剂、自体骨移植、3D生物打印人工骨
- **AI辅助诊断**：X光/CT识别准确率不亚于资深医生，但责任归属争议
