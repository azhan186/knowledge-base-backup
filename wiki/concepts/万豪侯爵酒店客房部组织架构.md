# 万豪侯爵酒店客房部组织架构

> 上海万豪侯爵酒店（JW Marriott Marquis Hotel Shanghai Pudong）客房部（Rooms Division → Housekeeping）的内部分工与汇报关系。

## 顶层结构

```
总经理 (General Manager)
  └─ 房务部 (Rooms Division)
      └─ 客房部 (Housekeeping)  ← 本知识库覆盖范围
          ├─ 楼层 (Floor / Guest Room Housekeeping)        Floor_001–044
          ├─ 公共区域 (Public Area, PA)                      PA_001–022
          ├─ 洗衣房 (Laundry, LDY)                           LDY_001–026
          └─ 健身中心 (Fitness Center, FC)                  FC_001–005
```

## 关键岗位

| 岗位 | 角色 | 直接关联 SOP |
|------|------|-------------|
| **服务总监 (Director of Service)** | 起草人/部门主管 | 所有 96 份 SOP |
| **总经理 (General Manager)** | 批准人 | 所有 96 份 SOP |
| **当班文员 (Coordinator on Duty)** | 失物招领登记 | Floor_004 / PA_022 |
| **客房服务员 (Room Attendant)** | 客房清洁 | Floor 系列 |
| **PA 清洁员 (PA Attendant)** | 公共区域清洁 | PA 系列 |
| **洗衣房员工 (Laundry Attendant)** | 布草/客衣/制服处理 | LDY 系列 |
| **健身中心员工 (FC Attendant)** | 健身/泳池运营 | FC 系列 |
| **制服房主管 (Uniform Room Supervisor)** | 制服发放 | LDY_001 |
| **外包员工** | 后勤/PA 部分岗位 | 行政人事/外包员工入职注意事项 |

## 跨部门协作

客房部与以下部门有强联动：

- **前厅部 (Front Office)** — 房态、客需、VIP 信息、迷你吧账单
- **安保部 (Security)** — 失物招领、可疑设备、火警应急
- **人力资源部 (HR)** — 新员工制服领用、岗位培训
- **工程部 (Engineering)** — 设备维修、虫害处理（外协艺康）
- **财务部 (Finance)** — 客衣收费、迷你吧账单、营收核算
- **餐饮部 (F&B)** — 送餐托盘回收、餐厅清洁协调

## 关键日期

- **2019-05**：主人们入职开荒期
- **2019-08-08**：酒店正式开业（所有 SOP 默认生效日期）
