# 微笑曲线定投

**Smile Curve DCA**：在**先跌后涨**的市场中，定期定额投资产生的**U 形（微笑）成本曲线**。**是普通投资者应对市场波动的经典策略**。

## 工作原理

### 固定金额定投
- 每月投入固定金额（如 1000 元）
- 净值低时 → 买更多份额
- 净值高时 → 买更少份额
- **长期平均成本 = 加权平均价**

### 微笑曲线形成

```
价格
  ↑
  │     ╭─────╮
  │   ╱        ╲
  │ ╱            ╲
  │╱              ╲
  └────────────────────→ 时间
   跌    谷底    涨
   ↑              ↑
  起点           终点（高）
```

- **起点和终点价格相同**（U 形）
- 期间经历先跌后涨
- **定投平均成本 < 起点价格**（更便宜）

## 数学原理

假设 4 个月定投，每月 1000 元：

| 月份 | 价格 | 买入份额 |
|------|------|----------|
| 1 | 2.00 | 500 |
| 2 | 1.00 | 1000 |
| 3 | 1.00 | 1000 |
| 4 | 2.00 | 500 |
| **合计** | - | **3000** |
| **平均成本** | - | **4000/3000 = 1.333** |

**平均成本 1.333 < 起点价格 2.00**（**便宜 33%**）

## 适用条件

### 适合
- 长期看好某类资产（股票/股票基金）
- 市场短期波动但长期向上
- 没有大额一次性投入的资金
- 心理上无法择时

### 不适合
- **单边上涨市场**：一次性投入更优
- **单边下跌市场**：定投越买越亏
- **高费率产品**：复利侵蚀超过微笑曲线优势

## 与"越跌越买"的区别

- **微笑曲线**：固定金额自动调节
- **越跌越买**：主动判断+加倍投入
- 后者**收益可能更高**，但**需要择时能力**

## 心理价值

> **定投的最大价值不是"数学最优"，而是"心理可执行"**
> 避免追涨杀跌，让普通人**真正长期持有**
