# 五星级酒店部门架构与岗位设置

> **来源**：用户提供｜**日期**：2026-05-15｜**类型**：酒店管理知识
> **说明**：只保留部门架构与岗位设置，不含具体人数

---

## 一、核心管理层

- **总经理**：酒店最高决策者，对业主和品牌管理方负责
- **驻店经理/房务总监**：分管酒店日常运营，常直接管理房务部
- **各部门总监/经理**：形成扁平化高效管理

---

## 二、一线运营部门（直接对客）

### 1. 前厅部
**下属岗位**：前台、礼宾、宾客关系、总机、商务中心、行政酒廊、GRO（宾客关系主任）、大堂副理

**职责**：预订，入住，退房、问询、VIP接待、宾客投诉处理

**特点**：
- 24小时运转，需三班倒
- 大堂副理是前厅部经理的副手，直接向前厅部经理汇报
- GRO日常工作中向大堂副理负责，虚线联动前厅部经理；重大事项同时知会前厅部经理

---

### 2. 客房部
**下属岗位**：楼层服务、公共区域保洁（PA）、洗衣房、布草房

**职责**：保障客房品质与卫生、床品管理、布草洗涤

**完整汇报链**：
- 楼层服务员 → 楼层领班 → 楼层主管 → 客房部经理
- PA员工 → PA领班 → PA主管 → 客房部经理
- 洗衣房员工/布草房员工 → 洗衣房主管/布草房领班 → 客房部经理
- 客房部经理 → 房务总监/驻店经理 → 总经理

**特点**：洗衣房和布草房可合并管理，也可分开设置

---

### 3. 餐饮部（最大部门）
**下属岗位**：
- **服务线**：各餐厅，大堂吧、宴会厅，送餐、迎宾
- **厨房线**：中厨，西厨、饼房、员工餐厅
- **管事部**：餐具清洗、厨房卫生

**职责**：餐饮服务、宴会运营、客房送餐、员工餐厅

**完整厨房汇报链（从低到高）**：
- 厨工/帮厨/学徒 → 各档口厨师（热菜/冷菜/点心/烧腊等） → 档口主管 → 副厨师长（中厨）/副厨师长（西厨） → 行政总厨 → 餐饮部总监 → 总经理

**服务线汇报链**：
- 服务员/传菜员/迎宾/吧员 → 餐厅领班 → 餐厅主管 → 餐厅经理（中/西/宴会/送餐经理） → 餐饮部总监 → 总经理

**特点**：
- 管事部（洗碗工等）→ 管事部主管 → 餐饮部总监 → 总经理
- 行政总厨统筹整个厨房，副厨师长代表不同板块（中厨/西厨）
- 餐厅经理与行政总厨平行协作，共同对餐饮部总监汇报

---

### 4. 康乐部（五星必备）
**下属岗位**：健身中心、游泳池，SPA水疗（可外包）、KTV/棋牌室

**职责**：康体娱乐服务

**完整汇报链**：
- 服务员/救生员/技师 → 康乐主管 → 康乐部经理 → 总经理

**特点**：部分设施（如SPA）可能外包经营

---

### 5. 保安部
**下属岗位**：监控中心、各岗点保安、消防

**职责**：安全管控、消防安全、监控值班，出入管理

**完整汇报链**：
- 保安员/监控员 → 保安领班 → 保安主管 → 保安部经理 → 总经理

**特点**：
- 24小时运转
- 按执勤点位数量配置，四班三运转

---

## 三，二线支撑部门（后台）

### 1. 市场销售部
**下属岗位**：客房销售、宴会销售，数字营销、公关美工、预订部

**职责**：营收，品牌推广、媒体关系、预订管理

**完整汇报链**：
- 销售代表/预订员/美工 → 销售经理/预订主管 → 高级销售经理/市场传讯经理 → 销售总监/市场销售总监 → 总经理

**特点**：
- 销售团队分层级设置（总监→副总监→高级经理→经理→代表）
- 公关部负责品牌推广与媒体关系
- 预订部通常归属市场销售部

---

### 2. 财务部
**下属岗位**：日审/夜审、收银，成本会计、总账会计、仓管

**职责**：日审/夜审、收银监管，成本控制、采购仓管

**完整汇报链**：
- 收银员/仓管员/审计员/会计 → 主管/会计主管（如总账会计） → 财务经理/副经理 → 财务总监 → 总经理
- **业务上虚线汇报给管理集团的区域财务总监**（双线监管）

**关键角色**：
- 日夜审：负责每日营收稽核
- 仓管员：分管总库、餐厅、客房等不同仓库

---

### 3. 人力资源部
**下属岗位**：招聘、培训、薪酬福利、员工餐厅、员工宿舍、车队

**职责**：人员招聘、培训、薪酬福利、后勤保障

**完整汇报链**：
- 员工餐厅厨师/宿管/司机/文员 → 对应主管 → 人事经理 → 人力资源总监 → 总经理

**特点**：
- 大型酒店人事行政部还管辖员工餐厅、员工宿舍、车队
- 中型酒店这些职能可能外包或合并

---

### 4. 工程部
**下属岗位**：强弱电、暖通（HVAC）、给排水、装修维修、锅炉房值班

**职责**：设施设备运维、24小时值班

**完整汇报链**：
- 技工（强弱电/暖通/装修工） → 值班工程师/领班 → 专业主管 → 副总工程师 → 工程部总监/总工程师 → 总经理

**特点**：按客房间数的8%左右配比，需24小时值班

---

### 5. 采购部
**下属岗位**：食材采购、物料采购、工程采购

**职责**：酒店物资采购

**特点**：部分酒店采购部归属财务部管理，不单独设部门

---

## 四，五星级酒店架构的特点

| 特点 | 说明 |
|------|------|
| **服务细分** | 设专职"金钥匙"礼宾司、管家服务团队、GRO等 |
| **营收多元** | 餐饮、宴会和SPA收入占比高，架构相应匹配专职团队 |
| **品牌标准** | 国际连锁常设"品牌标准/质量经理"，监督软硬件达标 |
| **24小时运转** | 前厅、客房、保安，工程必须全天候在岗 |
| **管理扁平化** | 一般按部门经理→主管→领班→员工四级或经理→主管→员工三级设置 |
| **一线向倾斜** | 一线部门编制应足额并留有余地（流失率较高） |

---

## 五、汇报关系（双线原则）

### 核心决策层汇报链

- 部门总监/经理（房务、餐饮，市场销售、财务、人事，工程等）**实线汇报**给总经理
- 驻店经理/副总经理通常分管特定模块，相关部门经理可能**虚线汇报**给分管副总，但实线仍归总经理
- 总经理向酒店业主或酒店管理集团区域总裁汇报

---

### 汇报关系速查

| 部门 | 汇报链（从低到高） |
|------|------------------|
| **客房部** | 楼层/PA/洗衣房/布草房各班组 → 各主管 → 客房部经理 → 房务总监 → 总经理 |
| **厨房** | 厨工 → 档口厨师 → 档口主管 → 副厨师长 → 行政总厨 → 餐饮总监 → 总经理 |
| **前厅** | 各岗位员工 → 领班 → 主管；GRO → 大堂副理 → 前厅部经理；大堂副理为前厅部经理副手 |
| **财务** | 收银/仓管/审计/会计 → 主管 → 财务经理 → 总经理；业务虚线向集团汇报 |
| **工程** | 技工 → 领班/值班工程师 → 主管 → 副总工程师 → 工程总监 → 总经理 |
| **市场销售** | 销售/预订/美工 → 经理/主管 → 高级经理 → 销售总监 → 总经理 |
| **人力资源** | 各岗位员工 → 主管 → 人事经理 → 人力资源总监 → 总经理 |
| **康乐** | 服务员/救生员/技师 → 主管 → 康乐经理 → 总经理 |
| **保安** | 保安员/监控员 → 领班 → 主管 → 保安经理 → 总经理 |

---

### 关键汇报原则

- **越级申诉，不可越级指挥**：指令尽量层层下达；员工有特殊重大事件，可越级向经理或人事部反映
- **前厅双线**：GRO日常向大堂副理汇报，重大事项同时知会前厅部经理；大堂副理是经理副手，不是夹在领班和主管之间
- **厨房平行协作**：餐厅经理与行政总厨平行，共同向餐饮总监汇报，不互相从属
- **财务双线**：行政上归酒店总经理管，财务准则和审计业务上必须向集团虚线汇报
