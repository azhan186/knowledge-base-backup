# 捕鲸的原因：鲸是"游动的化工厂"——3 大驱动力

> 视角：工业捕鲸时代**为什么疯狂**——鲸能提供**3 大类无可替代产品**。

## 核心洞察

> 在**石油化工成熟**之前，鲸（尤其**大型须鲸**）能提供**好几种**现代人已经用**石油、塑料或植物油替代**的**关键原料**。

鲸 = **"游动的化工厂"**

## 3 大产品

### 1. 鲸油——最主要的"万能油"

#### 用途 1：照明与取暖

- **19 世纪**：城市街道、家庭灯具、**灯塔**
- **优势**：燃烧**火焰明亮稳定**、**少烟少味**
- **对比**：比**菜籽油**、**牛油蜡烛**好得多

#### 用途 2：工业润滑剂

- **行业**：纺织机械、**钟表**、精密仪器
- **要求**：**轻薄** + **不易凝固**
- **明星产品**：
  - **抹香鲸脑油**（sperm whale head oil）
  - **抹香鲸油**
- 物理性质**极适合**——**一度无可替代**

#### 用途 3：制皂 + 日化

- **鲸油硬化处理** → **人造黄油**（早期）
- 后被**植物氢化油**取代
- **甘油** = **肥皂 + 化妆品** 重要原料

#### 用途 4：战略物资

- **甘油** → **硝化甘油**（炸药）
- **两次世界大战** = **各国争夺**的重要**战略油脂**
- **平民生计** + **战争机器** = **双重需求**

### 2. 鲸须——塑料出现前的"弹性材料"

**鲸须板**（长在须鲸嘴里过滤食物的**角质片**）：
- **坚韧** + **有弹性** + **可热弯定型**

#### 用途

| 用途 | 说明 |
|------|------|
| **裙撑** | **女士**裙撑、紧身胸衣的**支撑条** |
| **雨伞骨架** | 鲸须可**弯曲定型** |
| **鞭子** | 弹韧 |
| **钓鱼竿梢** | 鱼竿顶端 |
| **马车弹簧** | 座椅弹簧 |
| **牙刷毛** | 早期牙刷 |

**关键驱动**：**19 世纪时尚**对**裙撑**的**巨大需求** → **弓头鲸**等**须较长**种类被**集中屠杀**。

### 3. 鲸肉与其他副产品

| 产品 | 用途 |
|------|------|
| **鲸肉** | 传统蛋白质（日/挪/冰）/ **狗粮猫粮** / 家畜饲料 |
| **龙涎香** | 抹香鲸肠道 → **高级香水定香剂**（极度昂贵，产量极低）|
| **骨粉** | 磨粉 → **肥料 + 饲料添加剂** |

> 注：龙涎香（ambergris）是**附带高价值**产品，**非主要捕鲸目标**。

## 类比：万能油脂小镇

> 想象一个叫"**清水镇**"的北方小镇，**还没通电**，**也没有塑料**。
>
> 附近的深湖里有一种巨大的"**脂鱼**"，从它身上可以得到：
> - **点灯**用的**脂鱼油**（比松明子**亮十倍**）
> - 处理后的**鱼骨弹片** → 梳子、伞骨、撑起**新娘婚纱裙摆**
> - **鱼肉**可以吃，**内脏**能做肥皂

### 故事

- **起初**：镇民**划小船**捕几条 → **补贴家用**
- **后来**：**工厂主**发现 → 把**油卖到缺灯油的大城市** → **利润比种地高百倍**
- **结果**：投资**蒸汽渔船** + **鱼叉炮** + **船上加工厂** → **一年猎杀的脂鱼比过去百年还多** → **湖里的脂鱼快绝迹**

> 清水镇 = 工业捕鲸时代**人类社会**的**缩影**

## 为什么是"工业时代"才疯狂？

### 4 大历史耦合

| 因素 | 详情 |
|------|------|
| **1. 技术突破** | 蒸汽船 + 爆炸捕鲸叉 + 海上工厂船 → **追捕大型鲸** + **当场炼油** |
| **2. 市场膨胀** | **城市化** + **工业革命** → 对**照明油 + 润滑油 + 日化**的**海量需求** |
| **3. 高额利润** | **南极捕鲸** = "**浮动的金矿**"；一头**蓝鲸** → **30 吨油**；回报率**极高** |
| **4. 资源错觉** | 人们普遍认为**海洋浩瀚** + **鲸无穷无尽** → 直至**崩溃**才知 |

### 关键时点

- **1859 年**：**石油大规模商业化**前
- **鲸油** = **优质油脂主要来源**
- 各国企业（**挪威、英国、荷兰、日本**）竞相投入

### 不确定性

> 关于早期人类是否**纯粹**因捕鲸导致某些种群崩溃，还是**气候、洋流变化**也有**叠加作用**，学术界对**具体群系**的崩溃**比例**有**细微争论**。
>
> 但**工业捕鲸是主因**这一结论，在生物学界是**高度共识**。

## 后来为什么不捕了？

**反向验证**原因 = 不再捕鲸 = 因为**便宜替代品** + **鲸快没了**

### 1. 石油衍生物

- **煤油** + **矿物润滑油** + **合成甘油** + **塑料**
- **全面取代**鲸油 + 鲸须

### 2. 化学硬化油

- **植物油加氢硬化** → 人造黄油 + 肥皂
- 取代鲸油硬化油

### 3. 保护意识 + 禁令

- 部分鲸类**已到极危边缘**
- **全球环保运动**压力
- **1982 年** IWC **暂停商业捕鲸决议**
