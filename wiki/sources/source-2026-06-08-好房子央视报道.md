# 未来五年，"好房子"这样建！——住建部详解《城市更新"十五五"规划》

> 住建部详解"**好房子、好小区、好社区、好城区**"四个好，配套老旧小区 **11.5 万个**、完整社区 **5000 个**硬指标。这是同一天上午住建部吹风会"摘要版"的"详细版"。

- **发布时间**：2026-06-08 晚
- **发布渠道**：央视财经

- **联合发布部门**：
  - 住房城乡建设部

---

## 二、核心框架："四个好"

| 维度 | 定位 |
|------|------|
| **好房子** | 住房标准、设计、材料、建造、运维全链条提升 |
| **好小区** | 老旧小区改造、加装电梯、基础设施升级 |
| **好社区** | 养老、托育等公共服务设施完善 |
| **好城区** | 路网结构优化、停车设施补齐、存量资源利用 |

**配套规划指标**：《城市更新"十五五"规划》设置 **10 项规划指标**，部署一批**民生工程、发展工程、安全工程**。

---

## 三、好房子建设

### 新建房屋
- **全链条提升**住房标准、设计、材料、建造、运维水平

### 老旧房屋
- 抓好老旧房屋**更新改造**
- 鼓励居民对住房进行**装修和改造**
- 推广**装配式装修** 和 **适老化、无障碍改造**
- 支持老旧住房**自主更新、原拆原建**

---

## 四、好小区建设（"十五五"目标）

| 指标 | 数量 |
|------|------|
| 新开工改造城镇老旧小区 | **11.5 万个** |
| 改造内容 | 燃气、供水、热力、道路等基础设施 |
| 配套 | 支持**具备条件的楼栋加装电梯** |

**吹风会发言人**：住房城乡建设部副部长 [[秦海翔]]

---

## 五、好社区建设（"十五五"目标）

| 指标 | 数量 |
|------|------|
| 建设改造社区 | **5000 个** |
| 重点完善 | 社区**养老、托育**等公共服务设施 |

---

## 六、好城区建设

- 充分利用**存量资源**
- 补齐**公共服务设施**短板
- 优化**路网结构**
- 补齐**停车设施**短板

## 九、关键术语

| 术语 | 解释 |
|------|------|
| **四个好** | 好房子/好小区/好社区/好城区 |
| **装配式装修** | 工厂预制部品 + 现场组装的装修方式 |
| **适老化改造** | 针对老年人安全/便利需求的住宅改造 |
| **原拆原建** | 保留原住户、原位置、原规模的拆建模式 |
| **完整社区** | 配齐养老/托育/文化/体育/智能化设施的社区 |
