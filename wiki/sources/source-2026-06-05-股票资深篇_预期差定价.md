# 股票资深篇：预期差定价与高阶思维

## 核心认知

**超额收益 = 你的认知 – 市场共识**

- 市场共识已100%price-in到当前股价
- 你的认知必须是非共识且正确的
- 研究股票的最高境界 = 研究"市场共识中哪里有裂缝"

## DCF现金流折现模型

**企业价值 = 未来所有年份自由现金流按折现率换算的现值**

| 要点 | 含义 |
|------|------|
| 价值载体是现金流 | 利润≠现金流，积压库存≠价值 |
| 增长只在于超额增长 | 投入回报>资本成本，增长才创造价值 |
| 久期决定估值 | 现金流越远，对利率越敏感（加息→高估值成长股灾难） |

## 市场微观结构

- **流动性海市蜃楼**：极端时刻流动性瞬间蒸发
- **两种对手盘**：知情交易者（有信息优势） vs 噪音交易者（被迫交易）
- 核心技能：分辨价格波动是信息驱动还是资金流动驱动

## 风险本质：脆弱性 ≠ 波动

| 类型 | 特征 |
|------|------|
| 脆弱 | 高杠杆+现金流紧绷，任何冲击可能归零 |
| 反脆弱 | 手握净现金+能反向扩张，市场波动让它更强 |

**尾部风险对冲**：稳健资产+极少比例布局极端事件期权（非线性收益）

## 两大思维框架

1. **叙事经济学**：市场交易叙事而非事实。分析叙事阶段——无人信→共识形成→狂热流行（最危险）
2. **边际定价**：股价由最迫切成交的小撮人决定。在足够安全位置接手被边际定价者扭曲的资产
