# 万豪洗涤衣物缺陷通知（2026-01 绿发版）

> 视角：客房**洗涤问题**反馈卡——客人权益保护。

## 基础信息

- **创建时间**：2026-01-14
- **类型**：客房**洗涤问题反馈卡**
- **使用方**：万豪品牌客房

## 使用场景

客人发现**送回的衣物有问题**：
- 缩水
- 褪色
- 变形
- 损坏
- 没洗干净
- 异味

## 反馈流程

1. 客人填卡 + **保留物证**
2. 24 小时内送**礼宾部**
3. 礼宾 + 洗衣房**联合调查**
4. 给出**处理方案**：
   - **重洗**（免费）
   - **赔偿**（按万豪标准）
   - **保险理赔**（损坏）

## 关键提示

- **第一时间**反馈（不要过夜）
- **保留原物**（不要自行处理）
- **拍照留证**
- **温和沟通**（礼宾部优先）

## 万豪《洗衣赔偿标准》

| 损坏程度 | 赔偿 |
|---------|------|
| 轻微（可修复）| 免费修复 + 5x 洗衣费 |
| 中等（部分损坏）| 按原价的 **30-50%** |
| 严重（不可修复）| 按原价的 **50-80%** |
| 灭失 | 按**原价**赔 + 10% 慰问金 |
