# 信息矛盾资深篇

> 系列：信息矛盾（入门 → 深入 → **资深** → 专家）

---

## 一、形式系统的终极困境

**哥德尔不完备定理**：任何足够复杂的、一致的系统都**天生不完备**，存在无法证明的真命题，且**无法在自身内部证明自身一致性**。

**虚构自指案例**：自动化法官系统 + 输入 S："系统永远不会承认 S 是真的"
- 承认 S → 违反 S
- 不承认 S → 错过真命题

→ **永远无法从内部完全检测和消除所有潜在矛盾**，除非系统本身就不一致。

---

## 二、现实世界矛盾信息的工程学处置（5 层）

| 层级 | 机制 |
|------|------|
| 1. 表面裁决 | 多数投票 / 最新时间戳 / 优先级 |
| 2. 可靠性建模 | 为来源建立**随时间变化的可靠性档案**，计算后验可信度 |
| 3. 语义消解 | 识别**信息粒度差异**（不是非黑即白冲突） |
| 4. 延迟裁决 + TMS | 记录**裁决理由**，支持回溯撤销——**基于理由的信念修正** |
| 5. 真相发现 | **迭代置信度**：来源和事实互证，逼近"共识真实" |

---

## 三、认知科学视角

### 认知失调（费斯廷格）

心理不一致 → 不适驱力 → 选阻力最小的路径：

| 难度 | 路径 |
|------|------|
| 最难 | 改变行为（"我戒烟了"） |
| 中 | 改变认知（"危害被夸大"） |
| **最易** | 增加协调性认知（"我爷爷抽烟活到 99"） |

### 双加工理论（卡尼曼）

| 系统 | 特征 | 任务 |
|------|------|------|
| **系统 1**（直觉） | 快、自动、不费力 | **心理一致性** |
| **系统 2**（分析） | 慢、刻意、费力 | **逻辑一致性** |

→ **后真相本质**：信息轰炸激活系统 1 绕过系统 2

---

## 四、叙事的战争

### 框架理论（Entman）

同一事件用不同框架 → 建构截然不同的矛盾叙事：

| 框架 | 叙事 | 核心概念 |
|------|------|---------|
| "执法" | 嫌疑人暴力拒捕，警察被迫用武 | 公共安全、英雄、法律授权 |
| "人权" | 手无寸铁公民被国家暴力机器处决 | 公民权利、国家暴力、程序正义 |

### 信息战的终极目的

不是让你相信某个叙事，而是让你对"**存在任何可知真相**"本身产生虚无感 → 公众**犬儒主义** → 社会共识根基瓦解
