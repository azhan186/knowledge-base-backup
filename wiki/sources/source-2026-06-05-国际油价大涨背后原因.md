# 国际油价大涨背后原因

## 核心驱动因素

| 因素 | 说明 |
|------|------|
| 霍尔木兹海峡通行受阻 | 持续受阻对海湾国家石油产能造成冲击 |
| 全球原油储备下降 | 显著减少 |
| 美国商业原油库存 | 环比减少近800万桶，降幅约为预期两倍 |

## 原油市场数据

- **纽约轻质原油期货（7月）**：96.02美元/桶，涨幅 **+2.41%**
- **伦敦布伦特原油期货（8月）**：97.81美元/桶，涨幅 **+1.89%**

## 美股市场

中东局势紧张 + 就业数据超预期 → 市场担忧美联储在稳就业和控通胀之间倾向后者。利率互换市场显示年底前加息一次概率攀升至85%。

| 指数 | 涨跌幅 |
|------|--------|
| 道指 | -1.21% |
| 标普500 | -0.74% |
| 纳指 | -0.89% |

科技股领跌（金融、公用事业、通信服务等大面积下跌）。

### 英伟达
完成对一家硅谷初创AI公司的收购，股价收跌 **3.62%**。

### 博通
2026财年二季度主要业绩优于预期，但三季度AI芯片业务收入指引低于预期，盘后一度跌超 **12%**。

## 欧洲股市

英国跌0.40%，法国跌0.71%，德国跌1.31%。中东局势趋紧 + 美国可能对欧盟加征关税双重压力。

## 贵金属

美元走强令贵金属承压：

- **国际金价**跌破4500美元/盎司（黄金期价跌1.17%）
- **白银期价**下跌近2.5%

## 要点

> 霍尔木兹海峡是全球石油运输关键通道，其通行受阻直接推升国际油价。库存降幅超预期加剧市场紧张，同时美联储加息预期升温抑制风险资产。
