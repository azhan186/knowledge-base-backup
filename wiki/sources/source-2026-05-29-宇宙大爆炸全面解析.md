# 宇宙大爆炸全面解析

> 视角：关于宇宙起源的主流理论——138 亿年前的一次"奇点爆炸"。

## 核心定义

**宇宙大爆炸** = 关于宇宙起源的主流理论，认为宇宙起始于约 **138 亿年前**的一次极端高温高密度的"奇点"爆炸，从那时起宇宙一直在膨胀和冷却。

## 三大支柱证据

| 证据 | 说明 |
|------|------|
| **哈勃膨胀** | 1929 年哈勃发现几乎所有星系都在远离我们，且距离越远退行越快（v = H₀·d）|
| **宇宙微波背景辐射（CMB）** | 1965 年彭齐亚斯和威尔逊发现，宇宙大爆炸约 38 万年后释放的辐射，至今仍弥漫宇宙，温度约 **2.725 K** |
| **元素丰度** | 大爆炸核合成理论预言的氢、氦、锂比例与观测完美吻合 |

## 时间线

```
T=0：奇点（大爆炸）
T=10⁻⁴³ 秒：普朗克时期
T=10⁻³⁶ 秒：暴胀时期（宇宙瞬间膨胀 10²⁶ 倍）
T=3 分钟：原初核合成（形成氢、氦）
T=38 万年：复合时期（原子形成，宇宙变透明）
T=2 亿年：第一代恒星形成
T=92 亿年：太阳系形成
T=138 亿年：现在
```

## 关键参数

- **哈勃常数 H₀**：67-74 km/s/Mpc（不同方法有争议——哈勃张力）
- **宇宙年龄**：138 亿年
- **宇宙组成**：5% 普通物质 + 27% 暗物质 + 68% 暗能量

## 未解之谜

- 暴胀的物理机制
- 暗物质/暗能量的本质
- 哈勃张力（CMB vs Ia 型超新星测得的 H₀ 不一致）
