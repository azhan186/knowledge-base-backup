# 中方禁止新西兰议员入境

## 核心事件

新西兰个别议员窜访中国台湾地区，违反一个中国原则、干涉中国内政，中方决定对有关人员采取**禁止入境**等措施。

| 要点 | 说明 |
|------|------|
| 一个中国原则 | 国际社会的普遍共识和国际关系基本准则 |

| 台湾问题底线 | 踩线越界必将付出代价 |

## 背景

- 台湾问题是中国的核心利益，不容任何外部势力干涉
- 新西兰作为主权国家，其议员行为违反国际法和国际关系基本准则
- 中方此次采取禁止入境措施，属于外交反制手段
