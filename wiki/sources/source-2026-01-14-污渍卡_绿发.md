# 万豪污渍卡（2026-01 绿发版）

> 视角：客房**污渍应急处理**——客人意外弄脏地毯/床品。

## 基础信息

- **创建时间**：2026-01-14
- **类型**：客房**污渍处理卡**
- **使用方**：万豪品牌客房

## 卡片内容

- 污渍类型（红酒/咖啡/油渍等）
- 位置（地毯/床品/沙发）
- 通知方式（电话/微信）
- 处理时间（30 分钟内）

## 应急处理流程

1. 客人**发现污渍** → 立即联系前台
2. 楼层**紧急处理**（30 分钟内）
3. 严重 → 申请**专业清洗**
4. 记录 → 责任判定

## 责任判定

| 情况 | 责任 |
|------|------|
| **客人原因** | 客人**付费**清洁 |
| **酒店原因** | 酒店**免费**处理 |
| **不可抗力**（天气/疾病）| 协商 |
| **第三方** | 第三方负责 |

## 关键提示

- **不要**自行用纸巾/抹布擦（会扩大污渍）
- 立即**通知前台**
- **保留证据**（拍照）

## 2026 绿发版

- **环保清洁剂**
- 极简卡片
- 体现 ESG 主题
