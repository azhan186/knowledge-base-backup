# 量子计算机深入篇

> 视角：量子算法 + 量子门

## 经典量子算法

### 1. Shor 算法（1994）
- **整数分解**：多项式时间（vs 经典指数时间）
- 应用：**破解 RSA 加密**
- 意义：现代公钥密码学的**潜在终结者**

### 2. Grover 算法（1996）
- **无序搜索**：O(√N)（vs 经典 O(N)）
- 应用：数据库查询、密码破解（对称密码减半）
- 意义：**二次方加速**

### 3. Shor 变体
- 离散对数
- 椭圆曲线密码破解

## 量子门

| 门 | 功能 |
|----|------|
| **Hadamard (H)** | 创建叠加态 |
| **CNOT** | 纠缠两 qubit |
| **Pauli (X/Y/Z)** | 比特/相位翻转 |
| **Toffoli (CCX)** | 三 qubit 通用门 |
| **Phase (S/T)** | 相位控制 |

## 量子电路

任何量子算法 = **量子门序列**（电路）+ 测量

## 关键挑战

- **退相干**：qubit 状态极易被环境破坏
- **错误率**：~0.1%-1% per gate
- **需要纠错码**（surface code 等）
