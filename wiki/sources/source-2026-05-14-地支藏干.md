# 地支藏干：人元之秘

> 视角：每个地支藏**1-3 个天干**——天元（天干）、地元（地支）、人元（藏干）。

## 基础概念

- **天干为阳为天** → **天元**（表面的）
- **地支为阴为地** → **地元**（内里的）
- **支中藏干** → **人元**（天地之间）

每个地支藏 1 至 3 个天干，按力量分：
- **本气**（最旺）
- **中气**（次之）
- **余气**（最弱）

## 关键洞察

### 子午卯酉（桃花四正）

- **子**（癸）→ 纯阴水 → 桃花代表感情
- **午**（丁己）→ 火土 → 激情
- **卯**（乙）→ 纯阴木 → 优雅
- **酉**（辛）→ 纯阴金 → 收敛

### 寅申巳亥（四生四驿马）

- 四个"动"的地支
- 藏干最多（3 个）→ **多才多艺**
- 是**驿马**所在 → 奔波

### 辰戌丑未（四库四墓）

- 四个"库"的地支
- 藏干多但力量弱 → **储藏**
- 是**墓库**所在 → 收藏

## 应用

| 应用 | 方法 |
|------|------|
| **判断五行力量** | 算每个地支的本气 + 中气 + 余气 |
| **找十神** | 用日干对比藏干 |
| **看格局** | 月令的本气决定主格局 |
| **找用神** | 看四柱最弱的那一行 |
