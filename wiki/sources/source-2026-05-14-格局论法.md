# 格局论法：子平术的核心

> 视角：八字命理"职业化路径"的核心——**月令**定格局。

## 核心认知

> 格局论法是**子平术的核心**，神煞重"象"，格局重"理"。

可以把八字理解为一家公司：
- **格局** = 你的"**工种**"（做什么工作）
- **神煞** = 你的"**奇遇**"（意外事件）
- **身命三垣** = 你的"**创业方向**"（内心向往）

## 第一步：定格局的核心是"月令"

格局立在哪里，以**月令为主**。月支是主宰人生职业化路径的最强气。

### 普通格局（8 种）

| 月令 | 格局 |
|------|------|
| 寅卯辰月 | 杂气格 |
| 巳午未月 | 炎上格 |
| 申酉戌月 | 从革格 |
| 亥子丑月 | 润下格 |

### 特殊格局

| 格局 | 触发条件 |
|------|---------|
| **正官格** | 月支藏正官，且不被破坏 |
| **七杀格** | 月支藏七杀 |
| **食神格** | 月支藏食神 |
| **伤官格** | 月支藏伤官 |
| **正印格** | 月支藏正印 |
| **偏印格** | 月支藏偏印 |
| **正财格** | 月支藏正财 |
| **偏财格** | 月支藏偏财 |
| **比肩格** | 月支藏比肩 |
| **劫财格** | 月支藏劫财 |

## 第二步：定身强身弱

> **身强** = 日主得令、得地、得生、得助
> **身弱** = 日主失令、失地、失生、失助

### 四得 vs 四失

| 维度 | 身强 | 身弱 |
|------|------|------|
| **得令** | 日干生于旺月 | 日干生于休囚月 |
| **得地** | 日干坐临官/帝旺 | 日干坐死/绝 |
| **得生** | 印星生扶 | 财官克泄 |
| **得助** | 比劫帮身 | 比劫无力 |

## 第三步：取用神

| 格局 | 用神方向 |
|------|---------|
| 身强 | **克泄耗**（官杀、食伤、财星）|
| 身弱 | **生扶**（印星、比劫）|
| 调候 | 平衡五行（寒热燥湿）|

> **调候用神** 往往比格局更重要——尤其对普通人。

## 第四步：判断吉凶

- 用神**得力** → 吉
- 用神**被克** → 凶
- **忌神**（与用神相反）出现 → 凶
