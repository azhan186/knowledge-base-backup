# 接触传染率与行为心理学

## 核心公式

**有效接触率：** `β = c × p`
- c = 接触率（单位时间接触易感者数）
- p = 单次接触传播概率

**基本再生数：** `R₀ = β / γ`
- R₀ > 1 → 疫情扩散；R_t < 1 → 疫情受控

---

## 两种传播模式

| 模式 | 假设 | 适用场景 |
|------|------|----------|
| 密度依赖 | c ∝ N（人口密度） | 学校、大城市流感 |
| 频率依赖 | c 为常数（固定社交圈） | 性传播疾病、稳定社区 |

---

## 心理学视角：β 是动态参数

### 风险感知 → c/p 变化

- **健康信念模型**：易感性感知↓ → c↑
- **乐观偏差**：低估自身风险 → c↑
- **心理距离**：危机感弱 → c↑

### 群体心理

- **从众效应**：多数人戴口罩 → p↓
- **心理抗拒**：强制措施 → c↑（反向效应）
- **疫情疲劳**：习惯化 → c/p 恢复高位

### 认知捷径

- **习惯**：握手/拥抱等默认动作 → c↑
- **风险补偿**：高防护 → 更多风险行为 → c↑
- **行为免疫**：对可见病原体回避，但对无症状感染者失效
- **污名化**：隐瞒病情 → 隐性传染链扩大

---

## 有效干预思路

1. **精准风险沟通**：拉近心理距离，提升效能感
2. **正向社会规范**："保护他人"作为身份信号
3. **行为助推**：环境设计让安全行为成为默认选项
