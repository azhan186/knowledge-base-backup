# vlog设备对比

> 来源：公开参数整理（2026-05-14）

## DJI Pocket 3 vs Pocket 4 vs GoPro Action 5 Pro

| 参数 | Pocket 3 | Pocket 4 | Action 5 Pro |
|------|----------|----------|---------------|
| **参考价格** | ¥2,499 | ¥3,499 | ¥2,998 |
| **传感器** | 1英寸 CMOS | 1英寸 CMOS（升级） | 1/1.9英寸 |
| **光圈** | f/2.0 | f/2.0（优化） | f/2.5 |
| **分辨率** | 4K/120fps | 4K/120fps（升级） | 4K/120fps |
| **防抖** | 三轴云台 | 三轴云台（更强） | HyperSmooth 6.0 |
| **屏幕** | 2英寸旋转屏 | 2英寸旋转屏（更亮） | 前后双屏 |
| **续航** | 约2.5小时 | 约3小时 | 约2小时 |
| **防水** | 需防水壳 | 需防水壳 | 裸机10米 |
| **重量** | 179g | 约180g | 154g |

## 选购建议

| 场景 | 推荐 |
|------|------|
| 日常vlog/人像 | Pocket 4 > Pocket 3 > Action 5 Pro |
| 运动/户外/潜水 | Action 5 Pro > Pocket 4 > Pocket 3 |
| 性价比首选 | Pocket 3（降价后最划算） |
| 最新最强 | Pocket 4（AI芯片、续航更强） |

> 注：价格以电商平台实时为准，仅供参考。
