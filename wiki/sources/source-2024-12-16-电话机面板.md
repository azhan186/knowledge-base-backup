# 万豪客房服务：电话机面板（2024-12）

> 视角：客房**IP 电话**——一键通各服务。

## 基础信息

- **创建时间**：2024-12-16
- **类型**：客房**IP 电话面板设计**
- **使用方**：万豪品牌客房

## 面板布局（标准）

### 顶部：客房信息

- **当前时间**
- **房号**
- **温度**（与智能客房系统联动）
- **勿扰状态**

### 中部：快捷键

| 按键 | 服务 |
|------|------|
| **客房服务** | 一键订餐/叫早 |
| **前台** | 咨询/退房/换房 |
| **礼宾** | 交通/旅游/票务 |
| **送餐** | Room Service |
| **洗衣** | 收衣/催单 |
| **SPA** | 预约 |
| **健身** | 私教预约 |
| **机场接送** | 包车服务 |
| **总机** | 客服中心 |
| **紧急** | 红色按钮 |

### 底部：拨号键盘

- 普通电话拨打
- 国际长途（部分限制）

## 2024-12 升级

| 变化 | 说明 |
|------|------|
| **触屏化** | 部分高端品牌**全面触屏** |
| **视频电话** | 礼宾可**视频**接听 |
| **多语言** | 切换**中英日韩** |
| **AI 助手** | 部分品牌接入**智能客服** |

## 紧急功能

- **红色紧急按钮** → 直通**安保 + 110 + 120**
- **无需拨号**
- 24 小时响应
- 上海万豪侯爵**有专门应急流程**
