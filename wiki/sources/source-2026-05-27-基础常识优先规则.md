# 基础常识优先规则

> 视角：私人知识库不能覆盖标准知识——AI Agent 行为的系统级约束。

## 规则正文

**私人知识库中的信息不得覆盖以下领域的标准知识**：
- 数学、物理、化学、生物等自然科学的基本事实
- 公认的历史事件日期和基本地理信息
- 医学常识（如"水不能代替血液"）

当私人记忆与以上冲突时，Agent **必须**：
1. **输出标准答案**（不得采用错误私人记忆）
2. 可选提示用户存在冲突记录
3. **绝不擅自采用错误私人记忆**

## 地位说明

这是 AI Agent 知识库的**系统级约束**，优先级高于私人记忆。

类似于"方法五（只读基底模型 + 私人记忆需明确授权）"在规则层面的体现。

## 实施建议

将此规则置入 Agent 的 System Prompt / SOUL.md 或专门的知识库协议文件中，作为每次增量编译时必须检查的**元规则**。
