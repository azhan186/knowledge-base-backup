# 境外追逃：蔡某国因组织偷渡被引渡回国

> 视角：境外追逃的**典型案例**——5 年+ 跨部门合作，**6-8 押解回国**。

## 事件

**2026-06-08**，一名**长期潜藏境外**、**组织他人偷越国（边）境**的犯罪嫌疑人**蔡某国**被从境外**成功引渡回国**。

## 案件时间线

| 时间 | 事件 |
|------|------|
| **2023-2024** | 蔡某国等人**勾结境内外"蛇头"**，组织中国公民**转道偷渡**境外从事不法活动 |
| **2024-02** | **江苏省镇江市公安出入境管理部门**工作中发现跨国犯罪团伙 |
| **同期** | 国家移民管理局列为**挂牌督办案件** |
| **同期** | 蔡某国列为**追逃目标逃犯** |
| **2026-03** | 某国警方将蔡某国**抓获**并**通报**我方 |
| **2026-03 起** | 公安部 + 国家移民管理局指导办案单位**向该国提出引渡请求** |
| **同期** | 外交部门积极发挥**职能作用** |
| **2026-05 月底** | 该国**作出引渡裁定** |
| **2026-06-08** | 蔡某国**押解回国** |
| **2026-06-11（今日）** | 国家移民管理局**正式公布** |

## 关键术语

### "蛇头"

- **含义**：组织**偷越国（边）境**的**犯罪分子**
- **特点**：
  - **境内外**勾结
  - **跨境**作案
  - **高利润**（每人几万到几十万）
- **罪名**：组织他人偷越国（边）境罪

### 偷渡形式

- **"转道"偷渡**：
  - 借道**第三国**（如东南亚 → 美/欧）
  - 伪造**旅行证件**
  - 隐瞒**真实目的**
- 蔡某国案中："**组织多名中国公民转道偷渡境外从事不法活动**"

### 引渡

- **法律依据**：《中华人民共和国和某国引渡条约》或**互惠原则**
- **流程**：
  1. 中方提出**引渡请求**
  2. 对方国**审查**
  3. 对方国**司法裁决**
  4. **执行引渡**（押解回国）
- **耗时**：通常 1-2 年（蔡某国案 3 月提出 → 5 月底裁定 = 2 个月，**较快**）

## 国家移民管理局正告

> "**法网森严，罪责难逃**。**境外绝非避罪天堂**，**更不是法外之地**。任何**负隅顽抗、逃避打击**的行为都必将受到**法律严惩**，**唯一正确出路**是**主动回国投案自首**，**争取宽大处理**。"

## "天网行动"背景

- **全称**："天网"行动
- **启动**：2015 年 3 月
- **主导**：中央反腐败协调小组国际追逃追赃工作办公室
- **协作**：中央纪委、最高法、最高检、外交部、公安部、国家监委等
- **成果**：已追回**数千名**外逃人员
