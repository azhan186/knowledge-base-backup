# 6/11 全球金融市场综合：油价涨 / 股市跌 / 金价跌

> 视角：**中东冲突 + 美国通胀**双轮驱动——油价涨、股市跌、金价跌。

## 核心结论

| 资产 | 表现 | 主因 |
|------|------|------|
| **WTI 原油** | **+2.07%** 至 $90.03 | 中东冲突 + 库存大降 |
| **布伦特原油** | **+1.80%** 至 $93.10 | 同上 |
| **道指** | **-1.87%** | 中东 + 通胀双杀 |
| **标普 500** | **-1.62%** | 同上 |
| **纳指** | **-1.98%** | AI 概念股 + 半导体撤离 |
| **英国 FTSE** | **+0.27%** | 能源股拉动 |
| **法国 CAC** | **-0.51%** | 奢侈品股下跌 |
| **德国 DAX** | **-0.97%** | 同上 |
| **黄金** | **-3.57%** 至 $4133.30 | 美债收益率上行 + 流动性出清 |
| **白银** | **-0.77%** 至 $64.740 | 同上 |

## 一、原油：WTI 重回 $90

### 上涨三因素

| 因素 | 详情 |
|------|------|
| **中东冲突升级** | 市场看淡美伊达成协议前景 |
| **霍尔木兹海峡** | 通航**持续受阻** |
| **美国库存大降** | 上周商业原油库存**-723 万桶**（远超预期）|
| **战略储备消耗** | 上周 SPR **-793 万桶**（继续高速消耗）|

### 收盘价

- **WTI 7 月期货**：**$90.03/桶**（+2.07%）
- **布伦特 8 月期货**：**$93.10/桶**（+1.80%）

## 二、美股：全线收跌

### 主因

| 因素 | 影响 |
|------|------|
| **中东冲突升级** | 抛售**风险资产** |
| **通胀压力** | **AI 概念 + 半导体**资金撤离 |
| **加息预期** | **资金借贷成本**↑ → 产能扩张↓ |
| **消费支出预期** | 工业/材料/非必需消费品**板块下跌** |

### 三大指数

- **道指**：**-1.87%**
- **标普 500**：**-1.62%**
- **纳指**：**-1.98%**（AI + 半导体最受伤）

### 重点个股：英伟达

- **股价**：**-3.73%**
- **从高点回落**：约 **15%**
- **创一个多月新低**
- **市值**：**跌破 5 万亿美元**关口

## 三、美国 5 月通胀：3 年新高

### 关键数据

| 指标 | 数值 | 同比 |
|------|------|------|
| **CPI** | **+4.2%** | 2023-05 以来新高 |
| **核心 CPI**（剔除食品能源）| +2.9% | — |

### 美联储预期

- **本月议息会议**：**"按兵不动"**（大概率）
- **年底前加息 25bp 概率**：**67%**（芝商所）
- 市场预期从"降息"转向"加息"

## 四、欧洲：分化明显

### 主因

- 中东冲突 → 资金流出**估值偏高科技股**
- 通胀上行 → **奢侈品/非必需消费品**股下跌
- 油价上涨 → **英国能源股**上涨（FTSE 收涨）

### 表现

- **英国 FTSE 100**：**+0.27%**（能源股拉动）
- **法国 CAC 40**：**-0.51%**（奢侈品）
- **德国 DAX**：**-0.97%**（同上）

## 五、贵金属：黄金暴跌 3.5%

### 下跌三因素

| 因素 | 机制 |
|------|------|
| **美债收益率上行** | 持有黄金**机会成本**↑ |
| **美元指数小幅上涨** | 以美元计价的黄金**承压** |
| **中东冲突持续** | 油价高位 → 通胀持续 → **加息压力** → 黄金不利 |
| **流动性出清** | 部分投资者**保证金追缴** → **集中平仓抛售** |

### 关键点位

- 黄金盘中**接连跌破 $4200** 和 **$4100**
- 创 **2025-11 下旬**以来新低
- 白银跌破 **$65/盎司**

### 收盘价

- **黄金 8 月期货**：**$4133.30/盎司**（-3.57%）
- **白银 7 月期货**：**$64.740/盎司**（-0.77%）

## 六、关键观察

### 资产联动

```
中东冲突升级
    ↓
油价涨（WTI $90, 布伦特 $93）
    ↓
通胀压力（美国 CPI 4.2%）
    ↓
美联储加息预期（67%）
    ↓
美元 / 美债收益率 ↑ ↑ ↑
    ↓
美股跌（科技 + 半导体最重）
    ↓
金价跌（流动性出清 + 加息预期）
    ↓
欧洲股分化（能源涨 / 奢侈品跌）
```

### 历史类比

- **2022-2023 加息周期**：类似环境
- **2025 末黄金大涨**：反向
- **2026 中**：从"避险"转向"通胀-加息"主导

### 后续关注

| 节点 | 影响 |
|------|------|
| **6 月美联储议息** | 决定"按兵"还是"加息" |
| **7 月初 OPEC+ 会议** | 减产是否延长 |
| **中东冲突演变** | 油价核心驱动 |
| **AI 半导体业绩** | 纳指反弹关键 |

## 七、对中国影响

### A 股

- **能源股**（中石油/中海油）→ 利好
- **半导体股** → 利空（美股科技拖累）
- **黄金股** → 利空（黄金跌）

### 人民币

- **美元/人民币**：压力（美元↑）
- **跨境资本**：可能**外流压力**
- **出口**：可能**回暖**（人民币贬值）

### 央行政策

- **中国央行** 政策**空间**较大（不加息）
- 可能**反向操作**（降息/降准）
