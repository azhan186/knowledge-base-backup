# 双子星全面解析

> 视角：银河系过半恒星生活在双星系统中——太阳这样的"单身汉"反而是少数派。

## 核心定义

**双子星** = 由两颗恒星组成的引力束缚系统，它们围绕共同的质量中心运行。

## 关键特征

| 特征 | 说明 |
|------|------|
| **引力绑定** | 同一片星际云坍缩诞生，或演化中捕获对方，被引力锁在一起 |
| **绕共同质心** | 不绕对方，而是一起绕系统共同的质心旋转 |
| **不是视双星** | 碰巧同一视线方向但实际相距遥远的"光学双星"不是双子星 |

## 重要事实

> 银河系里，**超过一半的恒星**都生活在双星或多星系统中。

## 质量与轨道的关系

- 两颗星质量相当 → 质心在两人正中间 → 明显互绕
- 质量悬殊 → 质心靠近大块头 → 看起来像小孩绕着大块头转

## 分类

| 类型 | 说明 |
|------|------|
| **目视双星** | 望远镜可直接分辨两颗星 |
| **分光双星** | 通过光谱多普勒效应分辨 |
| **食双星** | 轨道平面侧向地球，互相遮挡导致亮度变化 |
| **脉冲双星** | 含脉冲星的系统（如 Hulse-Taylor 双星，引力波间接证据）|

## 演化意义

- 质量交换：双星间可发生物质吸积，产生 X 射线双星、Ia 型超新星
- 引力波源：脉冲双星（如双中子星）是引力波主要来源
- 引力理论验证：脉冲双星轨道衰减率与广义相对论预言完美吻合（1993 年诺贝尔物理学奖）
