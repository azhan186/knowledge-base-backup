# 西方题材短故事小课堂1期：海外特色文化介绍

## 狼人文化核心设定

### 狼人阶级体系

| 阶级 | 说明 | 角色定位 |
|------|------|----------|
| 狼王 | 狼群绝对大佬，手握最高权威 | 皇帝/霸总型主角 |
| 狼辅 | 大佬左膀右臂 | 助理/副手角色 |
| 庶裔 | 底层群体，弱小易被排挤 | 前期受虐/后期逆袭 |
| 狼后 | 首领命运伴侣，共同管理狼群 | 女主/女配 |

### 基础设定

- **内在狼**：独立意识，影响宿主情绪、判断和力量
- **月亮女神**：唯一信仰，掌控命运与伴侣分配
- **变身**：16-18岁解锁，可后期觉醒
- **无狼**：弱势者设定，前期虐点+后期逆袭

### 情感核心：命运伴侣

| 设定 | 说明 | 剧情用途 |
|------|------|----------|
| 命运伴侣 | 一生唯一灵魂伴侣，被背叛承受极致痛苦 | 虐点核心 |
| 拒绝缔结 | 单方/双方拒绝遭彼此反噬 | 极致虐心 |
| 二次机会伴侣 | 被拒绝后极小概率重新缔结 | 破镜重圆 |
| 伴侣舞会 | 成年狼人寻找确认命运伴侣 | 相遇情节 |
| 伴侣仪式 | 狼人版婚礼，正式绑定 | 仪式感高潮 |
| 意识联结 | 靠意念交流 | 甜宠/虐心 |

## 吸血鬼文化核心设定

### 外型与能力

- 面容苍白，尖锐獠牙，体温冰冷
- 没有影子，镜子无法成像
- 瞳孔在进食或情绪激动时改变（变红、竖瞳）
- 永生自愈、变身（蝙蝠/狼/老鼠/薄雾）、心灵感应、精神控制、超凡体能

### 弱点

- 惧怕阳光、惧怕圣物
- 木桩刺心、斩首、火焚
- 大蒜驱赶（东欧民间传说）
- 无法进入未经邀请的人类住宅

### 社会等级

| 等级 | 头衔 |
|------|------|
| 高等 | 亲王、公爵、伯爵 |
| 中等 | 子爵、男爵 |
| 低等 | 末代血族 |

### 情感设定

| 设定 | 说明 | 剧情用途 |
|------|------|----------|
| 初拥 | 创造者与被初拥者形成特殊血缘、精神纽带 | 甜宠/虐心 |
| 血契 | 交换血液形成的契约，信任、忠诚、保护 | 建立羁绊 |
| 解除血契 | 特定仪式或魔法，可能导致力量丧失或精神痛苦 | 虐点/情感拉扯 |
