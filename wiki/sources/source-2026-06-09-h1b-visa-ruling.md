# 美法官裁定：特朗普政府 H-1B 签证 10 万美元费用违法

> **马萨诸塞州联邦地区法院**认定 10 万美元 H-1B 收费属"税款性质"，超越总统权限；同日美联社披露**750 美元加急签证面试**服务将于 7/1 试点。

---

## 一、核心裁决

### 裁决方
- **马萨诸塞州联邦地区法院**（法官姓名未披露）

### 被告
- **特朗普政府**
- 被告机构：**美国国务院** + **美国公民及移民服务局（USCIS）**

### 裁决要点

| 项目 | 内容 |
|------|------|
| 政策内容 | H-1B 签证申请费从数千美元 → **10 万美元** |
| 法院认定 | 此款项"无论叫什么名字，都是税款的一种"，具有**税收性质** |
| 法律依据 | 联邦移民法赋予总统限制外国人入境的权力，但**未授权总统征税** |
| 处置 | 国务院 + USCIS **不得执行**该政策 |

### 案件背景

- **原告**：20 名**民主党籍**州检察长
- **诉讼理由**：H-1B 政策超越行政权力，影响各州雇用高技能外国专业人才
- **涉及签证**：H-1B 是美国临时工作签证（非移民签证），允许美国公司雇用外籍专业技术人员，主要被**美国科技企业**用于吸引高学历外籍人才

## 二、配套政策：750 美元加急签证面试服务

### 消息来源
- **美联社** 2026-06-08 报道
- **未官方公布**（消息人士透露）

### 政策内容

| 项目 | 内容 |
|------|------|
| 名称 | "高级加急服务"（Premium Rush Service）试点项目 |
| 加价 | 额外 **750 美元** |
| 适用 | 申请美国**商务或旅游签证**（B-1/B-2）的外国公民 |
| 启动 | **2026-07-01** |
| 结束 | **2026-12-31** |
| 形式 | 加快签证**面试预约**流程 |

## 四、影响分析

### H-1B 政策被否的连锁反应
- **美国科技企业**（FAANG 等）外籍员工招聘**预期重置**
- **印度外包公司**（Infosys、Tata、Wipro）业务模式承压
- 特朗普政府可能**上诉**到联邦巡回上诉法院

### "限制"vs"便利化"的政策反差
- 一边以**百万级门槛**限制技术移民
- 一边以**750 美元付费**加速非移民签证
- 反映美国签证体系正分化出"**双轨制**"信号

### "百日加急"反映的签证积压
- 美国非移民签证预约**长期积压**（数月等待）
- 750 美元定价反映"**付费优先**"思路
- 与 H-1B 政策被否形成对比
