# 大熊猫百科

> 视角：食肉目熊科——选择吃素却保留肉食装备的中国"国宝"。

## 核心定义

**大熊猫**（学名：Ailuropoda melanoleuca）= 食肉目、熊科，中国特有物种，主要生活在四川、陕西、甘肃的冷凉山林。食谱 99%+ 是竹子。黑白分明、体态圆润。被称为"活化石"和"国宝"。

## 生活类比：爱吃沙拉的胖厨师

天生配置肉食肠胃的人，偏偏只吃蔬菜沙拉：
- 为靠低热量食物维生，必须**每天 14 小时不停吃**
- 为不消耗多余能量，行动慢悠悠，能躺着不坐着
- 为抓稳滑溜菜叶，专门进化出**"假拇指"**（第六指）

这就是大熊猫——选择吃素却保留肉食装备的熊。

## 演化路径

| 时期 | 阶段 |
|------|------|
| 800 万年前 | 禄丰始熊猫，体型像胖狐狸，杂食偏肉食 |
| 200 多万年前 | 小种大熊猫，体型变大，竹子渐成主食 |
| 极盛时期 | 巴氏大熊猫，比现在大 1/10，分布至东南亚 |
| 今世 | 栖息地退缩至西南高山竹林，竹子专一素食者 |

## 现实应用

- **伞护种**：保护大熊猫 = 保护整片原始森林和竹林
  - 金丝猴、羚牛、小熊猫等均受益
- **生态旅游**：四川卧龙、佛坪等保护区带动经济
- **科研价值**：低代谢、黑白皮毛形成机制、生殖繁育难题
- **文化输出**：世界野生动物保护形象大使、中国软实力

## 主要挑战

| 挑战 | 现状 |
|------|------|
| **栖息地碎片化** | 已有 33 个保护区，总面积约 2.7 万 km² |
| **繁殖率低** | 每年仅发情 2-3 天，野外幼崽成活率低 |
| **食物单一** | 99% 竹子，若竹子大面积开花死亡会危机 |
| **气候变化** | 竹子分布北移，影响栖息地 |

## 中国保护成就

- 1980 年代：野生约 1100 只，濒临灭绝
- 2024 年：野生约 1900 只 + 圈养 700+ 只
- **IUCN 红色名录**从"濒危"降为"易危"（2016 年）
