# 八字论命体系·七条暗礁（完结篇）

> 视角：排盘和实战时极易出错的"暗礁"——掌握这七条，体系即闭环。

## 核心认知

排盘和实战时极易出错的"暗礁"，掌握这七条，体系即闭环。

## 一、必须确定"日主"才能定全局喜忌

**日主是八字的主人**，它不出来，所有吉凶都落不了地。

- **身强身弱**：格局是月令给的**职位**，日主是你的**能力**
  - 格局好、日主弱 → 身兼要职但身体垮了，反而累死
  - 格局差、日主强 → 闲不住总想折腾

## 二、调候用神 > 格局用神

- **调候**：看出生**季节**决定（冬生要火、夏生要水）
- **格局**：看月令十神决定
- **实战原则**：调候用神**比格局更重要**，尤其对普通人

## 三、勿以"用神"反推格局

- 用神是**结果**，格局是**原因**
- 不可"用神缺失就改格局"——这会循环论证
- 应该**先定格局 → 再看喜忌 → 最后用神**

## 四、神煞不能脱离宫位

- 单看**神煞**毫无意义
- 必须看它在**哪个柱**（年/月/日/时）
- 例：桃花在时柱 = 老年桃花（可能再婚）
- 桃花在月柱 = 青年桃花（多恋爱）

## 五、大运流年不可分割

- 大运是**主旋律**（10 年）
- 流年是**变奏**（1 年）
- 必须**大运 + 流年**一起看，不能只流年

## 六、空亡不是真"空"

- 空亡是**旬头空**（10 天干轮空）
- 实际意义：能量暂时"**不显**"，不是"**消失**"
- 大运/流年填实 → 能量显现 → 应事

## 七、四柱的"层次"不同

| 柱 | 含义 | 优先级 |
|----|------|--------|
| **年柱** | 祖宗、出身、0-15 岁 | ⭐⭐ |
| **月柱** | 父母、事业、15-30 岁 | ⭐⭐⭐⭐⭐ |
| **日柱** | 自己 + 配偶、30-60 岁 | ⭐⭐⭐⭐⭐ |
| **时柱** | 子女、晚年、60+ 岁 | ⭐⭐⭐ |

> **日主最重**，日支为"家"。

## 完整流程

```
1. 排盘（年/月/日/时）
2. 定日主 → 五行
3. 算身强/身弱
4. 取用神（调候 > 格局）
5. 看格局
6. 查神煞 + 宫位
7. 配大运 + 流年
8. 应期推断
```
