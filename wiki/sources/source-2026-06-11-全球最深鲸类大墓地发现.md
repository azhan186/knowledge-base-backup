# 全球最深鲸类"大墓地"：迪亚曼蒂纳深渊 1000 万头鲸长眠

> 视角：中国主导"**全球深渊探索计划**"——**最深最古老最壮观**的鲸类化石群。

## 重大发现

**中国科学院深海科学与工程研究所**主导的 **"全球深渊探索计划"** 取得**重大发现**：

- **位置**：**东南印度洋迪亚曼蒂纳深渊**（Diamantina Trench）
- **观察内容**：大量**鲸类化石** + **完整鲸落生态系统**
- **意义**：**全球已知深度最深、规模最大**的鲸类化石群与鲸落分布区

## 关键数据

| 指标 | 数据 |
|------|------|
| **鲸类长眠总数** | **1000 万头+** |
| **时间跨度** | **530 万年前至今** |
| **深渊长度** | 1200 公里 |
| **下潜次数** | 32 次 |
| **"活着"的鲸落** | 5 处 |
| **鲸类化石堆积** | **476 处** |
| **密度** | **每平方公里 700+ 具** |
| **水深** | 近 **7000 米** |
| **之前最深记录** | 4200 米 |

## 关键发现

### 1. 食骨蠕虫

> "食骨蠕虫伸出**根状结构**钻入骨骼内部，吸收**骨髓中的脂质**。"

### 2. 海蛇尾（棘皮动物）

> "棘皮动物**海蛇尾**，如**藤蔓**缠绕在鲸骨上。"

### 3. 化能合成双壳类

> "化能合成双壳类**簇拥**在鲸骨四周，利用鲸骨释放的**硫化氢**维持生命。"

### 4. "海菊花"（木栖海星）

> 科研团队还在鲸落中**首次记录**到**迄今发现最深**的木栖海星"**海菊花**"。

## 为什么这么多鲸？

### 地理结构

- 迪亚曼蒂纳深渊是 **V 字形超大峡谷**
- 像**巨大漏斗**
- **陡峭侧壁**把鲸类残骸**汇聚**到谷底
- 堆积成**深海奇观**

### 生物学解释

- 巨鲸可能**本来就是这里居民**或**迁徙至此**
- 深海**觅食**
- 年老/力竭时**无力浮出水面**
- 死亡后**缓缓沉入谷底**

## 鲸落生态系统

### 4 阶段

| 阶段 | 时间 | 主要生物 |
|------|------|---------|
| **1. 移动清道夫** | 数月 | 鲨鱼、盲鳗 |
| **2. 机会主义者** | 数年 | 多毛类、甲壳类 |
| **3. 化能自养阶段** | 10-50 年 | **化能合成细菌** |
| **4. 礁岩阶段** | 100+ 年 | **珊瑚、海绵** |

### "一鲸落，万物生"

> 巨鲸陨落，化作**深海花园**——让科普书中的"**一鲸落，万物生**"的故事在**深渊有了具象化的诠释**。
