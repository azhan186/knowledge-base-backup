# 股票专家篇：投资的哲学实践

## 核心认知

**我们从未真正拥有"股票"，我们交易的是时间的预期报酬。**

股票的三层致命属性：
1. **剩余的**：员工/供应商/债主/税务先满足，残渣归股东
2. **抽象的**：只存在于法律条文和集体想象中
3. **存在于时间之外**：DCF告诉你，现在的价格换未来所有现金总和

## 市场：复杂适应系统

- **自反性**：价格会反过来改变基本面（股价涨→公司变好；股价跌→公司变差）
  - 无客观价值，只有自我强化的正反馈
- **非遍历性**：1%破产概率重复足够多次=必然破产
  - 终极风控 = 确保永不被迫退出游戏

## 认知三层圈

| 圈层 | 说明 |
|------|------|
| 已知的已知 | 财报数据等 |
| 已知的未知 | 承认自己不知道的 |
| **未知的未知** | 风险全部来自这里 |

## 决策哲学

- **核心是放弃**：99个机会不在认知圈→无动于衷；只做确定性+赔率+时机+心态全部共振的非对称机会
- **逆向 = 与部落本能反向**：否决生理性恐惧；唤起泡沫史记忆以对冲贪婪

## 五大终极之问

1. 存在之问：知识全错时，信任建立在哪？
2. 时间之问：如何设计几十年持续运行的持有结构？
3. 洞察之问：好公司vs价值陷阱的临界点在哪？
4. 空性之问：如何做到无我不被仓位/成本/观点定义？
5. 巅峰之问：巴菲特/索罗斯/西蒙斯手法不同，那个共同的"一"是什么？
