# DNA数据储存原理

## 核心原理

数字信号 → DNA字母（00=A, 01=C, 10=G, 11=T）→ DNA合成仪"打印" → 读取时测序转回0和1

## 优势

| 优势 | 数据 |
|------|------|
| 存储密度 | 1克DNA ≈ 2.15亿GB |
| 保存期限 | 可保存成千上万年 |
| 格式兼容性 | 只要有生物学就有测序仪 |

## 四大瓶颈

1. **成本极高**：读写电影成本天文数字
2. **速度极慢**：合成和测序耗时
3. **无法随机存取**：必须顺序测序整管DNA
4. **错误率**：合成和测序过程产生字母错误

## 现状

已验证案例：微软在DNA中储存《OK Computer》专辑
短期定位：归档存储（国家档案、文化遗产）
