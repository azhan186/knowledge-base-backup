# Cron 定时任务核心定义

> 视角：电脑里的"24 小时不睡觉的定时小秘书"。

## 核心定义

**Cron** = 计算机系统里负责**按时间表自动执行任务**的服务程序。

- 你写好任务清单（crontab）
- 告诉它"什么时候做什么"
- 它像闹钟一样在后台准时执行，无需人手

## 类比：管家

主人写纸条贴冰箱：
- 每天 7:00 煮咖啡
- 每周一、三、五 20:00 推垃圾桶
- 每月 1 号 0:00 发开销报告

管家**一直盯着时钟**，到点就执行——即使主人在睡觉/旅行。

## Cron 表达式

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期几 (0-7, 0/7=周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

**例子**：
| 表达式 | 含义 |
|--------|------|
| `0 9 * * 1-5` | 周一-五 9:00 |
| `*/15 * * * *` | 每 15 分钟 |
| `0 0 1 * *` | 每月 1 日 0:00 |
| `30 2 * * 0` | 每周日 2:30 |

## 三种任务类型

| 类型 | 说明 |
|------|------|
| **系统级** | `/etc/crontab`（系统维护任务）|
| **用户级** | `crontab -e`（当前用户任务）|
| **Anacron** | 适用于不 24 小时开机的设备 |
