# 八字分析流程（方法论）

> 视角：从排盘到吉凶判断的**完整 8 步流程**。

## 分析前的准备：理解与排盘

**生辰八字** = 出生时的**干支历日期**，由年、月、日、时共八个字组成。

## 排盘步骤

1. **年柱**：根据出生年份，对照**干支纪年法**确定
   - 例：2026 年为丙午年
2. **月柱**：根据出生月份和年干，依据"**五虎遁**"口诀推算月干
3. **日柱**：通过**公式计算**或查阅**万年历**，直接获取
4. **时柱**：根据出生时辰和日干，依据"**五鼠遁**"口诀推算时干

## 核心分析步骤（8 步）

### 第一步：审视日主强弱

- **确定日主**：以日柱的天干代表命主本人，称为"**日主**"或"**日元**"
- 算四柱对日主的**生扶/克泄**
  - 印星 + 比劫 = 生扶
  - 财官 + 食伤 = 克泄

### 第二步：取用神

- 身强 → **克泄耗**（官杀/食伤/财）
- 身弱 → **生扶**（印星/比劫）
- 调候 → 平衡寒热燥湿

### 第三步：判断格局

- 看**月令**（月支）是什么
- 月支十神决定主格局
- 例：月支藏正官 → 正官格

### 第四步：看神煞

- 查关键神煞（天乙/桃花/驿马/华盖/七杀/羊刃）
- 看神煞在**哪个柱**（意义不同）

### 第五步：配大运

- 算起运岁数（3 天 = 1 岁）
- 排大运干支
- 看大运对原局的影响

### 第六步：配流年

- 当前流年（如 2026 丙午年）
- 看流年对原局 + 大运的引发

### 第七步：看命宫、身宫、胎元

- 命宫 = 性格倾向
- 身宫 = 人生归宿
- 胎元 = 先天禀赋

### 第八步：综合判断

- 用神得力 → 吉
- 忌神得力 → 凶
- 大运流年引发 → 应期

## 关键原则

| 原则 | 说明 |
|------|------|
| **先大后小** | 先看格局 → 再看大运 → 最后流年 |
| **先静后动** | 先看原局 → 再看大运流年 |
| **三才合一** | 天地人合一（天=四柱/地=宫位/人=神煞）|
| **多维验证** | 单一结论不立，需多维印证 |

## 完整流程图

```
排盘（年/月/日/时）
    ↓
定日主 + 算身强身弱
    ↓
取用神（调候 > 格局）
    ↓
定格局（月令）
    ↓
看神煞（位置）
    ↓
配大运（10 年主题）
    ↓
配流年（1 年事件）
    ↓
综合判断 + 应期
```
