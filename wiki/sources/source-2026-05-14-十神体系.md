# 八字十神体系

> 视角：以**日干**为中心，按五行生克关系划分的 10 个专有名词。

## 十神定义

**核心原则**：以日干为我，其他七字相对日干的关系命名。

| 十神 | 五行关系 | 阴阳 |
|------|----------|------|
| **比肩** | 同我 | 偏（同性） |
| **劫财** | 同我 | 正（异性） |
| **食神** | 我生 | 偏（同性） |
| **伤官** | 我生 | 正（异性） |
| **偏财** | 我克 | 偏（同性） |
| **正财** | 我克 | 正（异性） |
| **七杀** | 克我 | 偏（同性） |
| **正官** | 克我 | 正（异性） |
| **偏印** | 生我 | 偏（同性） |
| **正印** | 生我 | 正（异性） |

> **同性为偏，异性为正**——这是判断偏正的核心原则。

## 阴阳判断法

| 阴阳 | 关系 |
|------|------|
| **阳干** | 甲丙戊庚壬 |
| **阴干** | 乙丁己辛癸 |

**例子**：日干为甲（阳）
- 见甲（阳，同性）→ **比肩**
- 见乙（阴，异性）→ **劫财**
- 见丙（阳，同性）→ **食神**
- 见丁（阴，异性）→ **伤官**

## 十神类象

| 十神 | 代表 |
|------|------|
| **比肩** | 兄弟、同事、朋友 |
| **劫财** | 竞争、争夺 |
| **食神** | 才艺、口福、子女 |
| **伤官** | 创意、叛逆、表达 |
| **正财** | 稳定收入、妻子（男命）|
| **偏财** | 意外财、父亲（男命）|
| **正官** | 事业、地位、丈夫（女命）|
| **七杀** | 权力、压力、竞争 |
| **正印** | 母亲、贵人、学业 |
| **偏印** | 继母、玄学、孤独 |
