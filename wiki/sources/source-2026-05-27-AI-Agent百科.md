# AI Agent百科：从定义到应用与争议

> 视角：AI Agent（人工智能智能体）的完整知识图谱——从定义到五大历史阶段、现实应用与前沿争议。

## 核心定义

**AI Agent** = 能自主感知环境、做出决策并采取行动以达成目标的程序或系统。

**三个核心能力**：
- **感知**（Perception）：获取环境信息
- **决策**（Decision）：基于目标规划行动
- **行动**（Action）：执行具体操作

**vs 大语言模型**：纯大模型只会"对话"；Agent = 大模型 + 手和脚（可查资料、调用工具、执行任务、反复试错）。

## 五大历史阶段

| 阶段 | 时期 | 特点 | 例子 |
|------|------|------|------|
| 规则式 | 1950s-1980s | 人写死"如果...就..."规则 | 恒温器 |
| 反应式 | 1980s-1990s | 即时反应，无记忆 | Roomba 扫地机器人 |
| 基于模型 | 1990s-2000s | 内部世界模型+推理 | 自动驾驶 |
| 目标驱动 | 2000s-2010s | 比较路径选最优 | AlphaGo |
| 学习型（现代）| 2010s-今 | RL + LLM | ChatGPT + 插件 |

**关键转折：2023 年后**，LLM 让 Agent 有了"常识"和"语言理解"，可处理模糊指令。

## 现实应用

- **智能客服**：自动冻结信用卡、引导补办
- **自动驾驶**：感知+决策+加速/刹车/变道
- **推荐系统**：主动试探用户偏好
- **科研自动化**：查阅论文→假设→实验→分析
- **个人助手**：整理邮件、筛选会议、比价购物

## 主要争议与挑战

| 挑战 | 本质 |
|------|------|
| **安全与控制** | Agent 自主行动与人类价值观冲突（对齐问题）|
| **黑箱决策** | 深度学习驱动 Agent 决策不可解释 |
| **失业冲击** | 客服、翻译、数据录入可被替代 |
| **隐私** | 超级 Agent 需大量个人数据 |
| **技术瓶颈** | 长期记忆差、多步计划易出错、被对抗攻击欺骗 |
