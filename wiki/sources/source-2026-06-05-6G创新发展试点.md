# 工信部启动6G部省协同试点

## 核心结论

工信部正式启动6G创新发展部省协同试点专项行动，**2029年目标**形成自主技术方案、培育应用场景、打造产业集群，为6G商用全面落地铺路。

## 四大重点任务

| 方向 | 内容 |
|------|------|
| 融合技术 | 通信+AI、卫星互联网、无线感知，支撑标准研制 |
| 通信设备 | 基站、核心网、承载网、专用仪器仪表 |
| 关联产业 | 新型终端、芯片器件、操作系统、商业航天 |
| 应用场景 | 沉浸式通信、工业制造、低空经济、具身智能、智慧海洋 |

## 产业机遇

6G商用落地将全面带动：
- 基站/核心网/承载网建设
- 卫星互联网与地面通信融合
- AI驱动的通信架构变革
- 新型终端（AR/VR/全息设备）
- 低空经济（无人机通信感知）
