# 万豪客用品设计稿：JW 雀巢咖啡口味介绍（2024-04）

> 视角：JW 万豪客房**迷你吧咖啡**——雀巢品牌介绍。

## 基础信息

- **创建时间**：2024-04-18
- **类型**：客房**咖啡介绍**
- **品牌合作**：JW 万豪 × 雀巢（Nestle）
- **使用方**：JW 万豪品牌客房

## 合作背景

- 雀巢是**全球最大食品公司**
- JW 万豪（万豪旗下**高奢品牌**）
- 合作由来已久
- 体现 JW 的"**细致入微的奢华**"

## 提供的咖啡

| 类型 | 说明 |
|------|------|
| **意式浓缩** | Nespresso 胶囊（标准）|
| **美式** | 滴滤咖啡 |
| **拿铁** | 浓缩 + 牛奶 |
| **卡布奇诺** | 浓缩 + 奶泡 |
| **摩卡** | 浓缩 + 巧克力 |
| **低咖啡因** | 满足特殊客人 |

## 设备配置

- **Nespresso 胶囊咖啡机**（标准）
- **电热水壶**
- **马克杯**（×2）
- **奶球**（mini 装）
- **糖包**（黄糖/白糖/红糖）

## 风味介绍

- **浓烈**：深度烘焙，黑巧克力余味
- **平衡**：中度烘焙，坚果香
- **柔和**：轻度烘焙，果香

## 健康提示

- 每日咖啡因 ≤ 400 mg（健康成人）
- 孕妇 ≤ 200 mg
- 部分胶囊含**乳糖**（过敏者注意）
- 胶囊**不可回收**（铝壳）→ 部分酒店有回收计划
