# 叙事经济学入门

## 核心定义

**叙事经济学**（罗伯特·希勒，诺贝尔经济学奖得主）：像病毒一样传播的故事/观念，会大规模改变人们情感和行为，从而驱动经济波动。

核心类比：把"故事"当作会传染的"思想病毒"

## R₀公式（流行病学模型）

**R₀ = c × p × d**

| 变量 | 含义 |
|------|------|
| c | 每人每天分享次数 |
| p | 每次分享被接受并再传播的概率 |
| d | 叙事保持传染力的天数 |

R₀ >> 1 → 叙事野火蔓延，成为经济"流行病"

## 历史演变

1. 古老直觉：凯恩斯"动物精神"
2. 被边缘化：理性预期/有效市场假说主导
3. 行为经济学铺垫：卡尼曼/特沃斯基
4. 希勒系统化：2000《非理性繁荣》→2017正式提出→2019同名著作
5. 数字化加速：社交媒体+算法推荐

## 现实应用

- 加密货币叙事变种驱动价格波动
- "买房永远涨"→自我实现预言
- 游戏驿站：散户抱团叙事病毒式传播
- 央行前瞻指引 = 管理经济叙事
- 品牌叙事可让销量暴涨/谣言可摧毁行业

## 争议与挑战

- 叙事太模糊，难精确量化
- 因果方向不清
- 预测能力弱
- 与理性模型冲突
- 前沿：NLP量化/跨文化传播/嵌入宏观模型
