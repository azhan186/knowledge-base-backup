# 客房清洁中标价格表

> 视角：万豪侯爵酒店客房清洁外包的**中标价格**——供楼层主管参考。

## 核心信息

- **用途**：客房清洁外包**招标 + 中标价格记录**
- **使用人**：楼层主管
- **作用**：控制清洁成本 + 评估供应商

## 价格表结构（典型）

| 项目 | 单价 | 备注 |
|------|------|------|
| **标准间清洁** | XX 元/间 | 含床铺/卫生间/吸尘 |
| **退房深度清洁** | XX 元/间 | 含家具擦拭/换布草 |
| **住客续住** | XX 元/间 | 简清（不动床铺）|
| **空房维护** | XX 元/间 | 每周一次 |
| **大堂 PA** | XX 元/月 | 公共区域 |
| **玻璃清洁** | XX 元/m² | 外窗 + 内窗 |
| **地毯深度清洁** | XX 元/m² | 每季度 |
| **沙发清洁** | XX 元/位 | 按座位 |
| **窗帘拆洗** | XX 元/幅 | 半年一次 |
| **特殊污渍处理** | XX 元/处 | 临时报价 |

## 招标流程（标准）

1. **需求提出**：楼层主管根据客房数量 + 清洁标准
2. **询价**：至少 3 家供应商
3. **评标**：价格 + 资质 + 业绩 + 配合度
4. **中标公示**：内部审批
5. **签订合同**：通常 1 年期
6. **月度考核**：清洁质量 + 投诉率

## 关键考核指标

| 指标 | 标准 |
|------|------|
| **清洁质量合格率** | ≥ 98% |
| **客诉率** | < 0.5% |
| **员工到岗率** | ≥ 95% |
| **工时达标率** | 标准间 ≤ 30 分钟 |
| **绿色清洁认证** | 必选 |

## 注意事项

- 客用品（沐浴露/洗发水/拖鞋等）**不含**在清洁费内
- 客房布草洗涤**单独立项**（LDY）
- 客房小修小补（换灯泡/维修水龙头）由**工程部**负责
