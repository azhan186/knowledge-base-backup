# 医保个人账户跨省共济政策

## 核心结论

国家医保局、财政部正式印发《职工基本医疗保险个人账户跨省共济经办规程（试行）》，通过全国统一医保信息平台实现职工医保个人账户跨省支付近亲属医疗费用，**一人可帮全家**，但**额度有限、不得二次转赠**。

## 适用对象

**共济人：** 职工基本医保参保人

**被共济人：** 近亲属，包括——配偶、父母、子女、兄弟姐妹、祖父母、外祖父母、孙子女、外孙子女（须为基本医保参保对象）

> 一个共济人可与多名近亲属建立关系；一个被共济人也可接受多人共济。

## 规则要点

1. **额度管理**：共济人设定共济额度后，该额度由被共济人使用，共济人本人无法再使用该额度
2. **不得二次转赠**：被共济人获得的额度不能再共济给第三人
3. **解除自动返还**：解除共济关系时，未使用额度自动恢复至共济人账户

## 可共济范围

| 场景 | 可否使用 |
|------|---------|
| 定点医疗机构就医（个人负担费用） | ✅ |
| 定点零售药店购买药品/器械/耗材（个人负担） | ✅ |
| 缴纳城乡居民医保个人缴费部分 | ✅ |
| 缴纳长期护理保险个人缴费部分 | ✅ |

## 开通渠道

国家医保服务平台App / 省级医保服务平台 / 参保地医保经办机构窗口
