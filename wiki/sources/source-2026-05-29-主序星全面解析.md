# 主序星全面解析

> 视角：恒星一生中最稳定、最漫长的"正常态"——我们夜空 90% 的恒星处于此阶段。

## 核心定义

**主序星** = 恒星一生中最稳定、最漫长的阶段，其核心稳定地将氢聚变为氦，处于引力与辐射压力的完美平衡。

## 关键特征

| 特征 | 说明 |
|------|------|
| **核心氢燃烧** | 氢原子核聚变成氦原子核，恒星一生最核心的主营业务 |
| **流体静力平衡** | 向内的引力与向外的核聚变辐射压形成精准的动态平衡 |
| **质量决定一切** | 主序星阶段能待多久、表面温度、颜色、最终怎么死，几乎全部由初始质量决定 |

## 太阳的状态

- 已稳定燃烧约 **46 亿年**
- 还将持续约 **50 亿年**
- 此刻的阳光 = 太阳核心氢聚变的直接产物

## 质量决定演化路径

| 初始质量 | 主序星阶段时长 | 最终结局 |
|---------|--------------|---------|
| 0.08-0.5 太阳质量 | 数千亿年 | **氦白矮星** |
| 0.5-8 太阳质量 | 数十亿-百亿年 | **碳氧白矮星** |
| 8-25 太阳质量 | 数百万-数千万年 | **中子星（脉冲星）** |
| >25 太阳质量 | 数百万年 | **黑洞** |

## 类比：炉子最稳定燃烧

点燃→稳定燃烧（主序星）→煤烧尽（红巨星）→熄灭（白矮星/中子星/黑洞）
