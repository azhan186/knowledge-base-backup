# 四柱基础理论

> 视角：八字命理的**四柱**——年月日时，**每柱一天干一地支**。

## 四柱 = 8 字

**四柱** = 年柱 + 月柱 + 日柱 + 时柱 = **4 × 2 = 8 字**

## 年柱判定

- **分界**：以**立春**交节时刻划分，**不是农历正月初一**
- 例：公历 1998 年正月初三生，但若当年立春是正月初八 8:53，则年柱仍为上一年丁丑

## 月柱

- **分界**：以**节**为分界（不是中气）
- 12 个月对应 12 节：
  - 立春（寅月）→ 惊蛰（卯月）→ 清明（辰月）...
- 月柱天干由"**五虎遁**"推出

## 五虎遁（元）

**年上起月**：从年干推月干

| 年干 | 正月起 |
|------|--------|
| 甲己 | 丙寅 |
| 乙庚 | 戊寅 |
| 丙辛 | 庚寅 |
| 丁壬 | 壬寅 |
| 戊癸 | 甲寅 |

> **口诀**：甲己之年丙作首；乙庚之岁戊为头；丙辛之岁寻庚上；丁壬壬寅顺水流；若问戊癸何处起，甲寅之上好追求。

## 五鼠遁

**日上起时**：从日干推时干

| 日干 | 子时起 |
|------|--------|
| 甲己 | 甲子 |
| 乙庚 | 丙子 |
| 丙辛 | 戊子 |
| 丁壬 | 庚子 |
| 戊癸 | 壬子 |

## 核心口诀

> **年柱**定祖宗 → **月柱**定父母 → **日柱**定自己 + 配偶 → **时柱**定子女

**日柱最重要**——日干为"我"。
