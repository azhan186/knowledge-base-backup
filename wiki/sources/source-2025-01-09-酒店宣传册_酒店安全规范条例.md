# 万豪酒店宣传册：酒店安全规范条例（2025-01）

> 视角：万豪**安全规范条例**——客人行为规范 + 应急指南。

## 基础信息

- **创建时间**：2025-01-09
- **版本**：**Final**（最终版）
- **类型**：安全规范 + 应急指南
- **使用方**：万豪全球品牌

## 客需遵守

| 规范 | 说明 |
|------|------|
| **禁烟** | 客房**全面禁烟**（罚款 500 元起）|
| **禁火** | 严禁蜡烛/明火/电热器 |
| **禁宠** | 部分酒店**禁宠物**（导盲犬除外）|
| **禁噪** | 22:00 后**降低音量** |
| **禁摄** | 涉及其他客人**禁止拍摄** |

## 应急指南

### 火灾

1. **保持冷静**
2. **弯腰**（避开烟雾）
3. **湿毛巾捂口鼻**
4. **走楼梯**（**绝不**乘电梯）
5. **到集合点**（大堂外广场）

### 地震

1. **躲到桌下/墙角**
2. **远离窗户**
3. **不要乘电梯**
4. 震动停止后**有序撤离**

### 医疗

- **客房电话**红色紧急按钮
- **30 分钟**内有医生
- **最近医院**：瑞金医院
- **120** 直拨

### 安保

- **可疑人员/物品** → 立即报告
- **遗失物品** → 礼宾部
- **贵重物品** → 保险箱
- **人身安全** → 安保部 24h

## 关键安全设施

| 设施 | 位置 |
|------|------|
| **灭火器** | 每层楼道 |
| **烟雾报警器** | 每客房 |
| **喷淋系统** | 全楼 |
| **应急照明** | 走廊 + 楼梯 |
| **AED** | 大堂 + 健身房 |
| **急救箱** | 前台 + 健身房 |
