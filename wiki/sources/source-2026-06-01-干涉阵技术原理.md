# 干涉阵技术原理

> 视角：多个小型天线联手——用"分散小尺寸"换"等效大口径"。

## 核心定义

**干涉阵** = 让多个小型天线"联手"，通过"**干涉**"物理现象，模拟出一个口径极大的虚拟望远镜。

核心原理：
- 多个分散天线同时接收同一天体电磁波
- 精确比较每对天线信号**时间差（相位差）**
- 通过**傅里叶变换**把时间差还原成清晰图像

> 关键：干涉阵用"分散的小尺寸"换"等效的大口径"，获得极高分辨率。

## 生活类比

### 🎵 类比一：音乐厅手机录音阵列

20 位朋友分散坐在音乐厅不同位置，每人用手机同时录音。通过比较不同座位收到声音的**时间差**，可以精确算出每个乐器声源的位置。

**基线（天线间距）越大，分辨力越高**。

## 三大要素

| 要素 | 说明 |
|------|------|
| **天线分布** | 多个天线分布在不同地理位置 |
| **时间同步** | 原子钟精确对齐信号时间戳 |
| **数据合成** | 傅里叶变换把时间差→空间图像 |

## 关键参数

| 参数 | 说明 |
|------|------|
| **基线长度（B）** | 两天线间最大距离 |
| **观测波长（λ）** | 决定单天线分辨率 |
| **角分辨率** | θ ≈ λ/B（基线越长、波长越短，分辨率越高）|

## 应用与分类

| 分类 | 说明 |
|------|------|
| **连接式干涉阵** | 天线间用光纤/电缆连接（如 VLA、ALMA）|
| **VLBI** | 望远镜独立工作，靠原子钟 + 后期相关处理 |
| **e-VLBI** | 高速光纤实时传输，处理延迟低 |

## 关键应用

- **VLA（美国）**：27 面 25 米天线，Y 形排列
- **ALMA（智利）**：66 面射电望远镜，最高海拔 5000 米
- **EHT**：8 个望远镜组成的全球 VLBI 网
- **CVN（中国 VLBI 网）**：上海 + 乌鲁木齐 + 昆明等
