# 脉冲星构建宇宙GPS详解

> 视角：未来星际航行的灯塔坐标系——两步走实现"宇宙 GPS"。

## 原理：两步走

### 第一步：天眼在地面绘制"宇宙灯塔数据库"

**关键步骤**：
1. **找到最准的脉冲星**——毫秒脉冲星（误差几百万年 < 1 秒）
2. **精确测定每颗脉冲星**——三维位置、自行（横向移动）、距离
3. **建立"计时模型"**——精确预测每颗脉冲星每个脉冲到达太阳系中心的时刻
4. **输出产品**：脉冲星星表 + 计时模型数据库

### 第二步：航天器在太空自主定位

**工作流程**：
1. 飞船携带小型 X 射线望远镜，锁定多颗已知脉冲星
2. 记录脉冲到达飞船的精确时刻，与星表预测的同一脉冲到达太阳系中心时刻对比
3. 时间差 × 光速 = 飞船到太阳系中心在该脉冲星方向上的投影距离
4. **三点定位**（二维平面）→ **四点定位**（三维空间）

## 关键挑战：飞船上的钟

飞船时钟不可能像地面那样准，**必须同时观测多颗脉冲星进行差分计算**，在定位的同时消除自身时钟误差。

## 天眼在其中的核心角色

天眼本身不会发出导航信号，它**是纯粹的星图绘制者**——在贵州深山里年复一年精确测量脉冲星，建立顶级宇宙导航星图。

**类比**：天眼类似大航海时代的天文台，通过观测天象出版《天文年历》，让航海家能六分仪定位。天眼 = 在深山天文台里为未来星际航海家编纂最精确的《脉冲星导航星历》。
