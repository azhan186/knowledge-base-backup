# 宇宙星体种类总览

> 视角：从行星到超星系团的完整宇宙层级——人类对宇宙结构的全景认知。

## 宇宙层级（从小到大）

| 层级 | 典型尺度 | 说明 |
|------|---------|------|
| **基本粒子** | < 1 飞米 | 夸克、轻子、玻色子 |
| **原子** | ~1 埃 | 化学元素的基础 |
| **分子** | ~1 纳米 | 化学反应的基础 |
| **尘埃/星云** | 光年级 | 恒星诞生地 |
| **恒星/行星系** | ~光日-光年 | 太阳系尺度 |
| **星团** | ~10-100 光年 | 数百-数十万颗恒星 |
| **星系** | ~10 万光年 | 千亿颗恒星 + 星气 + 暗物质 |
| **星系群/团** | ~数百万光年 | 数十-数千个星系 |
| **超星系团** | ~数亿光年 | 数万个星系 |
| **可观测宇宙** | 465 亿光年半径 | 138 亿年历史 |

## 星体分类

| 类别 | 说明 |
|------|------|
| **恒星** | 自己发光（如太阳）|
| **行星** | 围绕恒星运行（如地球）|
| **卫星** | 围绕行星运行（如月球）|
| **矮行星** | 满足行星条件但未"清空轨道"（如冥王星、谷神星）|
| **小行星** | 太阳系内岩石小天体 |
| **彗星** | 冰+尘埃组成的"脏雪球" |
| **星云** | 星际气体+尘埃云 |

## 特殊天体

| 类型 | 说明 |
|------|------|
| **白矮星** | 中小质量恒星死亡遗骸 |
| **中子星/脉冲星** | 大质量恒星死亡遗骸（极致致密）|
| **黑洞** | 引力强到光逃不出 |
| **超新星** | 大质量恒星爆炸瞬间 |
| **类星体** | 极遥远、极亮的活动星系核 |
