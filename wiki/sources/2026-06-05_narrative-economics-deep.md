# 叙事经济学深入篇

## 叙事劫持决策的四步闭环

1. **情感触发**：恐惧/贪婪/归属叙事 → 绕开理性系统
2. **因果赋义**：识别反派 + 救赎之路 + 道德许可 = 赌博变成"正义之战"
3. **社会加压**：信息瀑布 + "这么多人不可能都错了" = 不参与成本巨大
4. **自我锁死**：认知失调 → 只接受证实自己观点的信息，成为最狂热传播者

## vs主流经济学三大冲突

| 冲突 | 主流经济学 | 叙事经济学 |
|------|-----------|-----------|
| 世界观 | 模型世界，故事=噪声 | 真实世界，叙事驱动经济 |
| 决策 | 孤立原子独立决策 | 社会网络节点，互相传染 |
| 宏微观 | 微观基础→宏观现象 | 宏观叙事本身就是微观基础 |

## AI时代新形态

- **算法传人**：推荐系统放大叙事，R₀指数级提升
- **合成现实**：深度伪造视频可在辟谣前造成市场闪崩
- **微观靶向**：同一产品对不同群体用不同叙事版本，群体隔离在信息茧房

## 叙事免疫系统

1. 溯源练习：问"这个想法哪来的，讲述者利益立场是什么"
2. 找出叙事脚本：反派+行动+灾难的结构
3. 价值与叙事剥离：看水分有多少
4. 主动关注反叙事：刻意寻找唱反调的信息源
