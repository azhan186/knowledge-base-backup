# 电子布年内5轮提价涨幅100%（2026-06-08）

**来源**：央视财经

##行业概况

**电子布**（电子级玻璃纤维布）是电子工业关键材料：
-覆铜板（CCL）
-印制电路板（PCB）
- 下游应用：**数据中心服务器、智能手机**等电子产品

##价格上涨

-截至6 月初：常用规格电子布完成**年内5轮提价**
- **均价7.4 元/米**
- 与去年三季度低点相比**涨幅达100%** ⚠️

##供需现状

###重庆某电子玻纤布企业
-订单饱和，产线满负荷运行
- **电子布产能利用率超过100%**
-排产周期超过 **2 个月**
- 个别极薄布及超性能高阶布排产周期超过 **4 个月**

### 浙江杭州某覆铜板企业原料仓库
-库存从1-1.5 个月 → **7 天左右** ⚠️
- 新到的原料刚入库就被投入生产

##产业链

```
叶蜡石、石英砂等矿物原料
 ↓熔化拉丝
 → 电子纱
 ↓织造
 → 电子布
 ↓ 和树脂、铜箔复合
 →覆铜板
 ↓蚀刻
 →印制电路板
 ↓支撑/连通
 →各类电子设备
```

##产能扩张

###扩产中的企业

**AI 服务器配套用的高端电子布产线**：
-河南光远
-泰山玻纤
- 广东建滔

**传统消费电子市场的电子布产线**：
- 中国巨石
- 广东建滔

###设备国产化
- 中国玻璃纤维工业协会秘书长**刘长雷**介绍
- **第一批国产化玻璃纤维电子布喷气织机设备已开始批量上市**

###关键瓶颈
- 电子布扩产相对快，但 **电子纱扩产时间更长**
- 一条新电子纱生产线从建设到投产需 **约1 年**
- →客观制约产能扩展节奏

##展望
-供需紧张情况可能延续
- 市场何时恢复供需平衡**有待观察**
