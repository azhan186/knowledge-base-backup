# AI幻觉专家篇

## 度量框架

- **原子事实分解**：文本T → 原子事实集合F，每个附二值标签v(f)∈{0,1}
- **幻觉率(HR)**：被证伪原子事实/总原子事实
- **加权幻觉率**：引入权重向量，关键事实（药物剂量）权重极高
- **忠实度分数**：NLI模型判断每个子句是否被源材料蕴含

## 类型学

| 维度 | 类型 | 说明 |
|------|------|------|
| 来源 | 外在幻觉 | 编造源材料没有的信息（RAG可对抗） |
|      | 内在幻觉 | 文本自身逻辑矛盾（更难） |
| 域 | 封闭域 | 可与源材料比对检测 |
|      | 开放域 | 锚是全球知识库，难度指数级 |
| 本质 | 事实性 | 违背客观事实 |
|      | 忠实度 | 不忠于用户指令（意图性幻觉） |

## Agentic RAG工程实践

- **自知之窗**：生成流+反思流并行，后者无法挑问题时答案才放行
- **程序化验证层**：数值/逻辑任务强制转代码执行，沙箱运行
- **优雅降级**：知识稀疏区不直接回答，用引语拼凑+明确告知无高置信度

## 终极前沿

- **世界模型**（海马体）+ **语言模型**（皮层）分离 = 时效性幻觉根本解
- **对抗性共生演化**：数据毒化-模型崩溃负反馈循环，需可信度评分机制
- **任务级认知对齐**：智能体长程任务不发生幻觉，行动空间需形式化约束验证
