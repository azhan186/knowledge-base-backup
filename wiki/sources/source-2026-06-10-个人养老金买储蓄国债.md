# 个人养老金能买储蓄国债了！6月10日起

> 时间：2026-06-10  
> 来源：央视财经

## 核心政策

**6 月起，电子式储蓄国债正式纳入个人养老金产品范围**——加入了个人养老金制度的市民，可以在账户里直接购买电子式储蓄国债。

**6 月最新一期储蓄国债**今天（6-10）正式发售。

## 银行渠道

| 银行 | 渠道 |
|------|------|
| **工商银行** | 手机 App 个人养老金专区 |
| **民生银行** | 手机 App 个人养老金专区 |
| **兴业银行** | 手机 App 个人养老金专区 |
| **所有银行** | 银行卡 + 身份证件 前往柜面办理 |

## 个人养老金产品池变化

```
此前四大类：存款 / 理财 / 基金 / 保险
  ↓
新增：储蓄国债（电子式）
  ↓
现五大类：存款 / 理财 / 基金 / 保险 / 储蓄国债
```

## 储蓄国债"入池"的优势（王向楠 解读）

> 中国社科院保险与发展研究中心副主任 王向楠

- **由财政部发行**
- **以国家信用为基础**
- **风险水平低**
- **起购金额低**
- **流动性较好**

## 差异化配置策略

### 长期投资者（养老目标）
- 部分资金配置**储蓄国债**（稳定利息）
- 同时配置**养老基金**、**养老理财**（平衡风险与收益）

### 临近/已退休投资者
- **适当提高**储蓄国债、养老存款、养老保险配置比例
- 目标：本金安全 + 稳定现金流

## 战略意义

### 1. 个人养老金制度扩容
- 从 4 类产品 → 5 类产品
- **制度建设的重要一步**
- 为投资者提供**更低风险、更稳健**的资产配置选择

### 2. 储蓄国债的角色转变
- 从"老年人专属" → 进入主流养老配置
- **国家信用 + 长期锁定**的双重优势
- 与现有产品形成**风险梯度互补**

### 3. 银行端的新业务
- 个人养老金 App 专区**新增国债购买入口**
- 多家银行同步上线
- 渠道铺设速度反映监管推动力
