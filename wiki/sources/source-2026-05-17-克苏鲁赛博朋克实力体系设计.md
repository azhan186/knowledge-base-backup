# 克苏鲁赛博朋克实力体系设计

> 视角：克苏鲁 + 赛博朋克世界里，"实力"是**危险、不稳定、需付出代价的"状态"**。

## 核心哲学

> 在克苏鲁+赛博朋克的世界里，"实力"不是一种可以量化的资产，而是一种**危险、不稳定、且需要付出代价的"状态"**。力量越强，离人性越远，离疯狂或彻底非人化就越近。

## 四把尺子替代"等级"

### 1. 认知深度：你窥见了多少真实？

**实力的核心**，也是**最危险的部分**。知道得越多，能力越诡谲，也越容易引来注视。

**表现**：
- 看见**常人看不见**的"信息流"
- 听懂克苏鲁的低语
- 触碰"**不该碰的知识**"

### 2. 异化程度：你还是"你"吗？

| 等级 | 名称 | 特征 | 威胁度 |
|------|------|------|--------|
| 0 | 普通人 | 未知领域 | 蓝 |
| 1 | 觉醒者 | 偶然接触，能力不稳 | 蓝 |
| 2 | 适应者 | 主动接触，能力成体系 | 黄 |
| 3 | 融合者 | 部分克苏鲁化，意识清醒 | 橙 |
| 4 | 异化者 | 完全克苏鲁化，意识**部分丧失** | 红 |
| 5 | 旧日 | **已不是人**，只是形态模仿 | 黑 |

### 3. 工具手段：你的"武器"是什么？

| 手段 | 代表 |
|------|------|
| **赛博义体** | 机械臂、脑机接口 |
| **魔法系统** | 异界能量、符文 |
| **AI 协作** | 与 AI 共享意识 |
| **维度入侵** | 借用其他维度的力量 |

### 4. 代价支付：你用什么换？

- **SAN 值**（理智/人性）
- **情感**（丧失共情）
- **记忆**（遗忘过去）
- **自由**（被旧日支配）
- **他人**（献祭、置换）

## 与传统"等级系统"的区别

| 传统等级 | 本体系 |
|----------|--------|
| **数字堆叠** | 状态**多维** |
| 升级 = 变强 | 升级 = 离人**更远** |
| 装备 = 数值 | 装备 = **代价标签** |
| 经验 = 升级 | 经验 = **异化** |

## 创作原则

1. **永远不要让主角"无代价变强"**——每次强大都要付出
2. **力量不是炫耀，是恐惧**——用力量时主角自己也怕
3. **Bashful（害羞）+ 危险**——强大但脆弱
4. **没有"满级"**——最强的人最接近**毁灭**

## 在《维度黎明》中的应用

- 林夜（融合者）：认知深度 + 异化程度 双线发展
- 零一（Boss）：AI 协作 + 维度入侵
- 亚特兰蒂斯（第五国）：融合者聚居地
