# 彗星全面解析

> 视角：太阳系外围的"脏雪球"——进入内太阳系时形成壮观的彗尾。

## 核心定义

**彗星** = 一颗由**冰、尘埃和岩石颗粒**组成的"脏雪球"，来自太阳系外围的柯伊伯带或奥尔特云，当进入内太阳系时被太阳加热释放出气体和尘埃形成彗尾。

## 结构

| 部分 | 说明 |
|------|------|
| **彗核** | 固态核心，直径通常 1-50 km |
| **彗发** | 彗核周围被升华的气体尘埃云，直径可达数十万 km |
| **尘埃尾** | 太阳光压推动尘埃形成，弯曲，黄色 |
| **离子尾** | 太阳风推动带电粒子形成，直，蓝色 |

## 起源

| 来源 | 说明 |
|------|------|
| **柯伊伯带** | 海王星轨道外（30-50 AU）的冰质天体聚集区，短周期彗星来源 |
| **奥尔特云** | 太阳系最外层（2000-200000 AU）假想球状云团，长周期彗星来源 |

## 著名彗星

- **哈雷彗星**：周期 76 年，下次回归 2061
- **海尔-波普彗星**：1997 年大彗星，肉眼可见 18 个月
- **67P/Churyumov-Gerasimenko**：罗塞塔任务着陆目标
- **C/2022 E3（ZTF）**：2023 年裸眼可见绿色彗星

## 任务里程碑

- **Giotto（1986）**：首次近距离飞掠哈雷彗星
- **Deep Impact（2005）**：撞击坦普尔 1 彗星
- **罗塞塔/菲莱（2014）**：人类首次在彗星表面着陆
