# DNA全面解析

## 核心概念

**DNA = 脱氧核糖核酸**，储存构建和维持生物体的全部指令

- 4个化学字母：A/T/C/G，配对规则A-T、C-G
- 双螺旋结构：像螺旋状梯子，横杆是配对的字母
- 每个细胞都带着同一套DNA副本，不同细胞查阅不同部分

## 历史演变

| 时间 | 里程碑 |
|------|--------|
| 1869 | 米歇尔发现"核素" |
| 1944 | 艾弗里证明DNA是遗传物质 |
| 1953 | 沃森/克里克提出双螺旋结构 |
| 1990-2003 | 人类基因组计划完成 |
| 2012- | CRISPR-Cas9基因编辑，2020诺贝尔化学奖 |

## 四大应用领域

1. **医学**：PCR检测、靶向治疗、基因疗法
2. **法医学**：DNA个人身份识别
3. **农业**：转基因作物
4. **消费级检测**：祖源分析、遗传病风险

## 四大争议

1. 编辑人类胚胎/"设计婴儿"
2. 隐私与基因歧视
3. 基因专利与生物剽窃
4. 合成生命与极端长寿
