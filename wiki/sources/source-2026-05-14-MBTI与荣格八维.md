# MBTI 与荣格八维

> 视角：MBTI 十六型人格 + 荣格八维认知功能——两个体系的关联。

## MBTI 十六型人格

| 类型 | 名称 | 特质 |
|------|------|------|
| **INTJ** | 建筑师 | 逻辑强、独立、战略思维 |
| **INTP** | 逻辑学家 | 理性、好奇、抽象思维 |
| **ENTJ** | 指挥官 | 果断、领导力、执行力强 |
| **ENTP** | 辩论家 | 机智、创意、善于发现问题 |
| **INFJ** | 倡导者 | 理想主义、有洞察力、忠诚 |
| **INFP** | 调停者 | 善良、创造性强、价值观驱动 |
| **ENFJ** | 主人公 | 有魅力、领导力、理想主义 |
| **ENFP** | 激励者 | 热情、创意、好奇心强 |
| **ISTJ** | 检查员 | 负责、传统、注重细节 |
| **ISFJ** | 守护者 | 忠诚、细致、乐于助人 |
| **ESTJ** | 总经理 | 务实、果断、组织能力强 |
| **ESFJ** | 供给者 | 热情、社交、乐于助人 |
| **ISTP** | 鉴赏家 | 动手能力强、冷静、理性 |
| **ISFP** | 探险家 | 艺术感、自由灵活、好奇心 |
| **ESTP** | 企业家 | 灵活、务实、善于危机处理 |
| **ESFP** | 表演者 | 热情、外向、活在当下 |

## 4 个二元维度

| 维度 | 两极 | 含义 |
|------|------|------|
| **E/I** | 外向 / 内向 | 能量来源 |
| **S/N** | 感觉 / 直觉 | 信息获取 |
| **T/F** | 思维 / 情感 | 决策方式 |
| **J/P** | 判断 / 感知 | 生活方式 |

## 荣格八维认知功能

**理论来源**：卡尔·荣格《心理类型》(1921)

8 个认知功能 = 4 个**外倾**（e）+ 4 个**内倾**（i）：

| 功能 | 简称 | 角色 |
|------|------|------|
| **Te** | 外倾思维 | 逻辑组织、效率 |
| **Ti** | 内倾思维 | 内在逻辑、原理 |
| **Fe** | 外倾情感 | 社交和谐、关系 |
| **Fi** | 内倾情感 | 价值观、真实 |
| **Se** | 外倾感觉 | 当下体验、感官 |
| **Si** | 内倾感觉 | 记忆、传统 |
| **Ne** | 外倾直觉 | 创意、可能性 |
| **Ni** | 内倾直觉 | 预见、深层洞察 |

## MBTI 与八维的对应

每个 MBTI 类型有**主导功能 + 辅助功能**：

| MBTI | 主导 | 辅助 | 第三 | 劣势 |
|------|------|------|------|------|
| **INTJ** | Ni | Te | Fi | Se |
| **INTP** | Ti | Ne | Si | Fe |
| **ENTJ** | Te | Ni | Se | Fi |
| **ENTP** | Ne | Ti | Fe | Si |
| **INFJ** | Ni | Fe | Ti | Se |
| **INFP** | Fi | Ne | Si | Te |
| **ENFJ** | Fe | Ni | Se | Ti |
| **ENFP** | Ne | Fi | Te | Si |
| **ISTJ** | Si | Te | Fi | Ne |
| **ISFJ** | Si | Fe | Ti | Ne |
| **ESTJ** | Te | Si | Ne | Fi |
| **ESFJ** | Fe | Si | Ne | Ti |
| **ISTP** | Ti | Se | Ni | Fe |
| **ISFP** | Fi | Se | Ni | Te |
| **ESTP** | Se | Ti | Fe | Ni |
| **ESFP** | Se | Fi | Te | Ni |

## 两个体系的关系

- **MBTI**：易上手、标签化、商业化（企业团建常用）
- **荣格八维**：更学术、更精准、解释力更强
- **结合使用**：MBTI 定大类，八维看深度
