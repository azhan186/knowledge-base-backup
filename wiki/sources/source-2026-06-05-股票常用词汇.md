# 股票常用词汇解释

## 股票本身

| 术语 | 含义 |
|------|------|
| 总股本 | 公司总股数 |
| 流通股本 | 可自由买卖的股数（大股东锁定部分除外） |
| 总市值 | 股价 × 总股本 |
| 蓝筹股 | 大盘稳健老牌公司 |
| 成长股 | 高增速但当前盈利少 |
| 周期股 | 随经济周期剧烈波动 |

## 价格与估值

| 术语 | 公式 | 要点 |
|------|------|------|
| 市盈率PE | 股价/每股盈利 | 高PE=高增长预期；低PE可能是一次性盈利 |
| 市净率PB | 股价/每股净资产 | PB<1=破净，比清算价值还便宜 |
| 股息率 | 每股年度分红/股价 | 股价跌时股息率反而变高，可能是陷阱 |
| 除权除息 | 分红后股价强制下调 | 总财富不变，1元现金替代1元市值 |

## 交易现场

| 术语 | 要点 |
|------|------|
| 涨停/跌停 | A股±10%限制，极端日放大情绪 |
| 换手率 | 3%活跃，>10%极高，分歧巨大 |
| 一手 | A股=100股，港股因公司而异 |

## 市场整体

| 术语 | 含义 |
|------|------|
| 牛市 | 长期普涨 |
| 熊市 | 长期普跌 |
| 猴市 | 反复震荡方向不明 |
| 指数 | 选一批代表性股票算出的数字，反映整体温度 |
