# 霍尔木兹海峡：160 艘油轮滞留 100 天 + IMO 警告

> 视角：海峡关闭的**前传背景**——**100 天前** 160+ 艘油轮已滞留，**IMO 警告海员安全**。

## 关键数据

| 指标 | 数据 |
|------|------|
| **滞留油轮数** | **160+ 艘**（主流）|
| **滞留时间** | **100+ 天** |
| **滞留地点** | 波斯湾水域 |
| **时间起点** | **2026-02 月底**地区冲突爆发 |
| **报告时间** | **2026-06-09** |

## 时间线（关键节点）

| 时间 | 事件 |
|------|------|
| **2026-02-28** | **美以**对伊朗发起**大规模军事行动** |
| **2026-02 月底** | 地区冲突爆发 → **油轮滞留开始** |
| **战事初期** | 波斯湾约 **1/4 油轮** 陆续驶离 |
| **近期** | **驶离进度明显放缓** |
| **2026-06-09** | 劳埃德船舶日报报道 + **IMO 警告** |
| **2026-06-11 凌晨** | 伊朗**正式宣布关闭海峡** |

## 劳埃德船舶日报

> "尽管战事初期位于波斯湾的约**四分之一油轮已陆续驶离**，但**近期驶离进度明显放缓**。霍尔木兹海峡附近区域**安全风险仍居高不下**，多数船东继续保持**高度谨慎态度**，**不愿贸然安排船舶通过相关海域**。"

## IMO（国际海事组织）警告

### 秘书长：多明格斯（Arsenio Dominguez）

> "**不能为了商业利益让海员冒险通过霍尔木兹海峡**。"

### 关切

- 部分船舶在**缺乏可信安全保障**的情况下
- 仍试图通过霍尔木兹海峡
- **当前地区局势依然高度动荡**
- **不具备可靠的安全保障条件**
- **安全航行根本无法得到保障**

## 海峡局势演进

### 2026-02-28：起点

- **美以**对伊朗发起**大规模军事行动**
- 海峡**未关闭**但**通行管制升级**
- 美国开始**海上封锁**伊朗

### 2026-04-13：美方正式封锁

- 美方对**所有进出伊朗港口船只**实施**封锁**
- （详见 [[伊朗公布最新损失]]）

### 2026-05-29：伊朗称"非法"

- 伊朗外交部称"**海上封锁违反停火**"

### 2026-06-09：IMO 警告（本日新闻）

- 劳埃德报道 **160 艘** 滞留
- IMO 秘书长**公开警告**

### 2026-06-10：伊朗损失

- 伊朗公布 **360 艘**船舶波及，**253 艘**沉没（**70%**）
- （详见 [[伊朗公布最新损失]]）

### 2026-06-11：伊朗反击

- 伊朗宣布**关闭霍尔木兹海峡**（**对所有船只**）
- （详见 [[伊朗关闭霍尔木兹海峡]]）

## 海员安全

### IMO 立场

- 联合国**专门机构**——**1948 年**成立
- 总部：**伦敦**
- 职责：**海上安全 + 海洋环境保护 + 海事法律**
- 海员安全是**核心使命**

### "不让海员冒险"

> "**不能为了商业利益让海员冒险**"——这是 IMO 的**基本立场**。

### 实践影响

- 保险公司**可能拒保**霍尔木兹航行
- 海员**拒绝**危险航行的权利
- 船东**道义责任**

## 关键观察

### "100 天"信号

- **100 天** 油轮滞留 = 实际上海峡**早已不开放**
- 但**官方**没宣布**关闭**（直到 6-11）
- 6-11 伊朗**官方宣布** = 法律意义上**正式关闭**

### 法律 vs 现实

| 维度 | 100 天前 | 6-11 之后 |
|------|---------|----------|
| **法律** | 通畅（理论上）| 关闭（官方）|
| **现实** | 滞留 160 艘 | 滞留**更多** |
| **保险** | 战争险 | 几乎不保 |
| **通行意愿** | 低 | 极低 |

## 全球航运影响

### 油轮市场

- **运费**：飙升
- **保险费**：**战争险费率** ↑ 10-100 倍
- **替代路径**：
  - **沙特东线管道**（可绕霍尔木兹）
  - **阿联酋 Habshan-Fujairah 管道**
  - **总替代运力**：约 **60% 海峡运量**

### 能源价格

- **短期**：WTI $90 → 可能 $130+
- **长期**：取决于**冲突持续时间**
- **替代效应**：美国/加拿大/巴西原油**涨价**

### 海员

- **拒绝**危险航行的权利
- **工资**可能**溢价**（危险补贴）
- **保险**可能拒赔
