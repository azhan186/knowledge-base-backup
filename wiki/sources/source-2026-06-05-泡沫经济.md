# 泡沫经济

## 核心定义

**资产价格远远脱离内在价值，被极度高估，膨胀到顶点后雪崩式破裂。**

本质：估值因子被叙事和情绪无限放大，脱离盈利因子约束的极端状态。

## 空瓶子类比

- 内在价值 = 几分钱
- 贴标签+限量观看+炒作 → 价格到10000元
- "里面不就是个空瓶子？" → 共识瓦解 → 跌回几分钱

三阶段：膨胀期→狂欢期→崩溃期

## 生成机制三角

| 要素 | 作用 |
|------|------|
| 叙事 | 简单强大的时代故事，半真但被无限放大 |
| 资金 | 低利率+新兴工具+杠杆，提供物理燃料 |
| 反身性 | 价格涨→改善基本面→验证高价→更多资金→再涨（最致命） |

## 历史标本

| 案例 | 核心特征 |
|------|---------|
| 郁金香狂热1637 | 价格与生长周期完全脱钩 |
| 南海泡沫1720 | 垃圾公司也能募资，泡沫顶部标志 |
| 1929大崩溃 | 保证金交易+螺旋式崩盘 |
| 日本泡沫1989 | 央行加息刺破，失去的三十年 |
| 互联网泡沫2000 | 市梦率取代市盈率，纳斯达克跌80% |

## 泡沫信号

1. CAPE远超长期均值
2. 新股民/散户开户爆发式增长
3. "这次不一样"成共识
4. 垃圾IPO盛行
5. 融资余额创历史新高
