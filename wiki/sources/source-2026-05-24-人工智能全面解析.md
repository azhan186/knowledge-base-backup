# 人工智能全面解析

> 分类：AI基础知识 | 标签：人工智能、机器学习、深度学习、AI伦理

---

## 核心定义

AI（人工智能）= 让计算机模拟人类某些智能行为（学习、理解语言、识别图像、做出决策、解决问题）

**关键区别**
- 传统程序：写死规则，"如果怎样，就做什么"
- AI：从大量例子中自己"找规律、学经验"，处理没见过的新情况

## 教孩子认猫 vs 教计算机认猫

| 方式 | 说明 |
|------|------|
| 传统编程 | 像写说明书一样定死规则——"猫有尖耳朵、胡须、会喵喵叫"，规则覆盖不到就认不出 |
| 机器学习 | 给成千上万张照片，告诉"这是猫/不是猫"，AI从例子中自己悟出猫的样子 |

## 历史演变

| 时代 | 里程碑 |
|------|--------|
| 1950s-1970s | AI诞生，早期乐观（证明定理、下棋），但逻辑推理无法解决理解问题，陷入低谷 |
| 1970s-1980s | 知识工程时代，"专家系统"（名医经验写成规则），死板、维护成本高，再次退潮 |
| 1990s-2010s | 机器学习崛起，1997年深蓝战胜国际象棋冠军，互联网+算力爆发 |
| 2012-至今 | 深度学习爆发，图像识别超越人类，AlphaGo、ChatGPT等生成式AI走进大众 |

## 现实应用

- **日常消费**：语音唤醒Siri/小爱、网购推荐、短视频推送、手机拍照虚化
- **行业赋能**：CT片辅助诊断、信用卡欺诈检测、工厂产品瑕疵检测
- **内容创作**：写邮件/代码/PPT大纲，生成图片/视频

## 争议与挑战

| 争议 | 说明 |
|------|------|
| 公平与偏见 | 训练数据含偏见→AI歧视特定人群（如简历筛选AI对女性打分更低）|
| 幻觉 | 大语言模型自信编造完全不存在的答案和引文 |
| 就业冲击 | 初级翻译、客服、数据录入等重复性工作被自动化替代 |
| 隐私与滥用 | AI换脸、深度伪造用于诈骗或虚假新闻 |
| 可解释性与对齐 | AI是"黑箱"；根本挑战：确保AI目标与人类价值观完全一致 |
