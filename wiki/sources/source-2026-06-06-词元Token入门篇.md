# 词元Token入门篇

> 视角：LLM 处理文本的最小单位

## 核心定义

**Token** = 大语言模型处理文本的**最小单位**。可以是：
- 整词（"苹果"）
- 子词（"苹"+"果"）
- 单字符（"a"）
- 标点、空格

## 关键比例（中英文）

| 语言 | 大致比例 |
|------|----------|
| 1 个汉字 | 约 1-2 个 token |
| 1 个英文单词 | 约 1.3 个 token |
| 1 个 token | 约 4 个英文字符 |
