# CT计算机断层成像原理

## 核心公式

**朗伯-比尔定律（X射线衰减）：**
```
I = I₀ × e^(-μd)
```

**CT值（亨氏单位）：**
```
CT值(HU) = 1000 × (μ_组织 - μ_水) / μ_水
```

---

## CT vs MRI

| 特性 | MRI | CT |
|------|-----|-----|
| 辐射 | 无 | 有（X射线） |
| 核心原理 | 氢质子共振/弛豫 | X射线衰减系数μ |
| 最佳观察 | 软组织、神经、关节 | 骨骼、钙化、肺、急性出血 |
| 扫描速度 | 长（数十分钟） | 极快（秒级） |

---

## 重建方法

- **滤波反投影**：黄金标准，先高通滤波再反投影
- **迭代重建**：主流，可降低辐射剂量并抑制金属伪影

---

## 关键组织CT值

水=0 HU，空气=-1000 HU，脂肪≈-100～-50 HU，脑白质≈25～35 HU，急性出血≈60～90 HU
