# 光伏发电全面解析

## 核心原理

**光生伏特效应**：光子照射硅等材料 → 电子被"撞"出 → 形成电流

- 太阳能电池（光伏组件/太阳能板）= 核心转换设备

## 发展历程

1839年发现效应 → 1954年贝尔实验室实用电池（6%效率）→ 1958年太空应用 → 2000年后全球规模化 → 中国将成本降低90%

## 四大应用

1. **大型地面电站**：荒漠戈壁，并入电网
2. **屋顶分布式**：自发自用，余电卖电网
3. **建筑一体化**：光伏玻璃/瓦
4. **移动与消费**：太阳能背包/充电宝/汽车

## 四大争议

| 争议 | 要点 |
|------|------|
| 环保账 | 1-2年发的清洁电即可抵消生产能耗 |
| 退役回收 | 25-30年寿命，铅等重金属处理 |
| 土地利用 | 农光互补/渔光互补解决方案 |
| 鸭子曲线 | 需配合储能解决日内供需错配 |

## 前沿挑战

- **效率天花板**：29.4%单结极限 → 钙钛矿/硅叠层可超40%
- **储能心脏**：稳定主力电源的必备条件
- **智能柔性**：可调度+薄膜电池
