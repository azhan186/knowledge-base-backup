# 流星全面解析

> 视角：来自太空的微小颗粒高速闯入大气层——夜空中划过的"瞬间星光"。

## 核心定义

**流星** = 来自太空的微小颗粒（**流星体**）高速（11-72 km/s）闯入地球大气层，与大气摩擦产生高温发光的现象。

## 关键名词区分

| 名词 | 说明 |
|------|------|
| **流星体** | 太空中未进入大气的颗粒 |
| **流星** | 进入大气后发光的瞬间现象 |
| **陨石** | 落到地面的残余物 |

## 三大类

| 类 | 说明 |
|---|------|
| **偶发流星** | 随机出现，每晚约 2-7 颗/小时 |
| **流星雨** | 地球穿越彗星遗留碎片流时的高强度爆发 |
| **火流星** | 亮度超过 -4 等（约金星最亮）的亮流星，可伴随声爆 |
