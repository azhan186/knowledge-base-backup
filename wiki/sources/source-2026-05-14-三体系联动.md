# 三体系联动实战

> 视角：三套体系不是孤立的——合起来是完整的**吉凶推导逻辑**。

## 核心认知

| 体系 | 核心 |
|------|------|
| **大运排法** | "**动静**" + "**应期**" |
| **神煞体系** | "**组合象**" + "**套餐**" |
| **格局论法** | "**顺用**" + "**逆用**" |

## 一、大运排法：核心组合

### 1. 干支一柱（盖头截脚）

| 类型 | 特征 | 效果 |
|------|------|------|
| **盖头** | 天干克地支（如戊子）| 上面盖住，下面动不了 |
| **截脚** | 地支克天干（如甲申）| 下面挖空，上面无根 |

### 2. 动静

- **大运流年引发原局动** → 事情发生
- **大运流年与原局同频** → 吉凶加深

### 3. 应期

事情发生的**时间点**：
- 大运引发 + 流年引发 = 应期
- 多重引发 = 应期提前

## 二、神煞体系：组合象

### 核心思想

**单个神煞无意义**，要**多个神煞 + 大运**配合。

### 常见"组合套餐"

| 套餐 | 含义 |
|------|------|
| **桃花 + 红鸾** | 婚期 |
| **驿马 + 桃花** | 远行遇感情 |
| **天乙 + 太极** | 贵人 + 学玄学 |
| **华盖 + 太极** | 艺术或宗教 |
| **七杀 + 羊刃** | 武职、危险 |

## 三、格局论法：顺用与逆用

### 顺用

- 身强 → 用食伤泄秀（适合表达、创作）
- 身弱 → 用印生扶（适合学习、依附）

### 逆用

- 身强用财（适合做生意）
- 身弱用比劫（合众、合伙）

## 四、完整推导流程

```
1. 看月令 → 定格局
2. 算日主 → 定身强/身弱
3. 取用神 → 找喜忌
4. 看大运 → 顺势/逆势
5. 查神煞 → 标"象"事件
6. 流年引发 → 应期
```

## 五、实战案例（通用推算）

**假设原局**：
- 身强丙火生申月
- 用神：水（官杀）
- 喜神：金（财生官）

**大运走水运** → 顺势 → 吉

**大运走火运** → 逆势 → 凶

**流年遇水** + **大运走水** → 应期在流年
