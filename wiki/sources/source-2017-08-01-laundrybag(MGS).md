# 万豪客用品设计稿：洗衣袋（2017）

> 视角：客房洗衣服务（LDY）专用**洗衣袋**——MGS 集团标准版。

## 基础信息

- **创建时间**：2017-08-01
- **类型**：客房**洗衣袋**设计稿
- **使用方**：万豪集团 MGS（Marriott Global Source）通用
- **使用人**：客人（客房内）+ 楼层洗衣房

## 洗衣袋功能

| 用途 | 说明 |
|------|------|
| **客房内** | 客人放置**待洗衣物** |
| **外送** | 楼层服务员**收集**到洗衣房 |
| **分类** | 不同颜色/标签区分水洗/干洗/熨烫 |
| **保护** | 防潮、防污染 |

## 规格（标准版）

- **材质**：高密度尼龙 + 内衬 PE 防水层
- **尺寸**：约 60×80 cm
- **颜色**：白色/灰色为主（万豪标准）
- **承重**：建议 ≤ 5 kg
- **印刷**：万豪 logo + 客房使用说明

## 客房内放置位置

- 衣柜内（**主用**）
- 床尾（客人显眼位置）
- 浴室（贴身衣物专用）

## 使用流程

1. **客人**将待洗衣物放入洗衣袋
2. 填写**洗衣单**（颜色、件数、要求）
3. 挂在**门外把手**或**放入洗衣篮**
4. 楼层服务员**早上收取**
5. 送**楼层洗衣房**或**外包干洗店**
6. **24-48 小时**内送回客房

## 万豪标准

- **洗涤质量**：5 级（不褪色、不变形、不缩水）
- **熨烫标准**：3 级（衬衫、裤子、裙装）
- **按时送回**：80% 以上当晚送回
- **损坏赔偿**：万豪标准**责任界定**
