# 中国天眼 FAST：500米口径球面射电望远镜

> 视角：聆听宇宙的"超级大锅"——地球上最大、最灵敏的单口径射电望远镜。

## 核心定义

**FAST** = 500米口径球面射电望远镜。不是用来看星星的"光学望远镜"，而是接收宇宙无线电波的巨型天线。直径 500 米，接收面积相当于 30 个足球场。

**vs 光学望远镜**：FAST 工作不是"看"，而是"听"——捕捉跨越千万年到达地球的极微弱宇宙无线电信号，像给宇宙装上听诊器。

## 三大历史阶段

| 时期 | 事件 |
|------|------|
| 1990s初 | 南仁东等人提出要在中国建世界级大射电望远镜 |
| 1994-2016 | 22 年磨一剑。南仁东用 12 年走遍贵州找天然喀斯特洼地，最终找到平塘大窝凼 |
| 2020 | 通过国家验收正式开放。阿雷西博（美）同年坍塌，FAST 成为全球唯一"巨眼" |

**遗憾**：南仁东在 FAST 落成一年后因病去世，被称为"天眼之父"。

## 四大科学任务

1. **寻找脉冲星**：核心工作。FAST 已发现 >900 颗新脉冲星，是同期全球其他望远镜的 3 倍以上
2. **绘制宇宙"气体地图"**：捕捉氢元素信号，研究星系演化
3. **破译快速射电暴（FRB）**：千分之一秒释放太阳一天能量的神秘现象
4. **搜寻地外文明**：捕捉异常无线电信号

## 主要挑战

| 挑战 | 本质 |
|------|------|
| **星链卫星干扰** | 马斯克数万颗低轨卫星群干扰观测 |
| **数据汪洋** | 每天产生的数据量巨大，筛选困难 |
| **维护压力** | 数千块面板暴露自然环境，需频繁高空"体检" |
