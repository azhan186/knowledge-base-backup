# 迷你吧表格说明

> 视角：主人**自制**的迷你吧管理表格——用于客房迷你吧的库存与销售管理。

## 核心信息

- **制作人**：张浩岚（楼层主管）
- **用途**：客房**迷你吧**（Mini Bar）的**库存 + 销售 + 补充**管理
- **使用范围**：上海万豪侯爵酒店客房部

## 表格功能

| 功能 | 说明 |
|------|------|
| **库存登记** | 每间客房迷你吧的当前商品数 |
| **消费记录** | 客人消费哪些商品 |
| **补充计划** | 每日补货数量 |
| **过期预警** | 临近保质期商品 |
| **营收统计** | 日/周/月营收报表 |
| **盘点核对** | 月度大盘点 |

## 核心指标

| 指标 | 说明 |
|------|------|
| **客单消费** | 平均每位客人迷你吧消费 |
| **补货频率** | 每日补货 vs 周补 |
| **过期率** | 月度过期商品占比（应 < 1%）|
| **营收目标** | 月度迷你吧营收（部门 KPI 之一）|

## 商品分类（典型万豪酒店迷你吧）

| 类别 | 例子 |
|------|------|
| **碳酸饮料** | 可口可乐、雪碧、芬达 |
| **果汁** | 鲜榨橙汁、苹果汁 |
| **矿泉水** | 巴黎水、依云 |
| **啤酒** | 青岛、科罗娜、嘉士伯 |
| **烈酒** | 芝华士、杰克丹尼（小瓶装）|
| **红酒** | 智利/澳洲餐酒 |
| **零食** | 薯片、坚果、巧克力 |
| **小吃** | 杯面（需开水冲泡）|
| **特色** | 上海特色点心 |

## 使用流程

1. **每日补货**：根据前日消费清单补足
2. **录入系统**：迷你吧 POS 系统
3. **巡检**：楼层主管每周抽查
4. **月度盘点**：财务 + 楼层联合
5. **过期处理**：登记 + 报废
