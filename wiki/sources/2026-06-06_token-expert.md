# 词元Token专家篇

> 视角：Token 的哲学与极限

## 关键议题

### 1. 字节级 vs 字符级
- **ByteLatent Transformer (BLT)**：字节级处理
- 摆脱固定词表的限制
- 罕见字符/代码优势

### 2. 嵌入空间的几何
- Token → **高维向量**
- 语义相似的 token 在向量空间**接近**
- "king - man + woman ≈ queen" 类比

### 3. 上下文窗口的真正极限
- **有效注意力**远小于标称窗口
- 128K 中 实际可用 ~32K-64K
- 注意力稀释 + 锚定效应

### 4. 多模态 Token
- 文本 + 图像 + 音频 → **统一 token**
- GPT-4V / Gemini 1.5 Pro
- **未来方向**：单模型处理多模态
