# 价格与价值深入篇

> 视角：价格如何偏离价值

## 价格形成机制

| 因素 | 影响 |
|------|------|
| **供需** | 直接决定短期价格 |
| **预期** | 自我实现的预言 |
| **情绪** | 恐惧/贪婪放大波动 |
| **信息** | 不对称造成套利 |
| **流动性** | 决定瞬时价格 |

## 价值评估方法

### 1. 绝对估值
- **DCF**：未来现金流折现
- **股息贴现模型（DDM）**
- **剩余收益模型（RIM）**

### 2. 相对估值
- **PE / PB / PS** 与同行比
- **EV/EBITDA** 跨资本结构比
- **PEG**（含增长）

### 3. 资产估值
- **重置成本**
- **清算价值**
- **期权定价**（Black-Scholes）
