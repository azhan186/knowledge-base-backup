# 餐厅清洁（PA_014）

> **LSOP No.**: RM-HSKP-PA_014 | **类别**: 公共区域（Public Area） | **生效日期**: 2019-08-08
> 编制：服务总监 | 批准：总经理

## 摘要

Each outlet should be clean and tidy at all times（公共区域清洁与服务标准）

## 关键要点

- 餐厅/宴会厅清洁
- 吸尘器使用规范

## 目的

Each outlet should be clean and tidy at all times
确保餐厅每天保持高标准的清洁状态

## 政策规定

Let employee know the job announcements, and require the employee to operate strictly according to the requirements
让员工了解工作注意事项，要求员工严格按照要求操作

## 程序

Vacuum carpet according to each outlet operating time.  （ensure this is done before the restaurant opens）. Spot clean if necessary.
每天根据各餐厅运作时间吸尘。（通常选定餐厅开餐之前）
Spot clean if necessary.
局部去渍
Clean wallpaper, wall cloth and mural painting with cleaning cloth.
用抹布清洁墙纸、墙布和壁画。
Check plants and flowers, change them if necessary.
检查植物和鲜花状态，及时更换
Clean window and glass.
窗和玻璃清洁
Clean wall skirt and floor wooden skirt.
墙裙和地角线清洁
Keep rest room cleanliness all the time.
保持卫生间清洁。
Deep clean carpet regularly, as per established schedule.
定期清洁地毯。
Clean vent and light fixtures.
高空清洁。
Report maintenance issues to PA Supervisor
及时向PA主管报告维修问题
