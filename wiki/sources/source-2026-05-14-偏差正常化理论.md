# 偏差正常化理论：Normalization of Deviance

> 视角：将原本不可接受的低标准/危险行为渐进认定为可接受——灾难的隐性温床。

## 理论定义

**偏差正常化**（Normalization of Deviance）：个人、群体或组织在长期工作中，将原本不可接受的低标准/危险行为认定为可以接受的普遍现象的**渐进过程**。

这个过程缓慢侵蚀安全边际，直至最初的异常行为在集体无意识中变成日常惯例，最终埋下灾难性后果的隐患。

## 理论起源：黛安·沃恩

美国社会学家**黛安·沃恩**（Diane Vaughan）提出，源于对 1986 年"挑战者号"航天飞机灾难的深刻剖析。

### 案例 1：挑战者号（1986）

- **事故肇因**：NASA 和承包商明知 O 形密封圈存在低温失效风险
- **关键决策**：此前未造成事故，便将风险标签为"可接受"
- **结果**：1 月 28 日航天飞机爆炸，**7 名宇航员遇难**

### 案例 2：哥伦比亚号（2003）

- 一块隔热泡沫脱落被视作"正常"而非致命威胁
- **结果**：返航解体，**7 名宇航员遇难**
- **悲剧重演**：类似逻辑导致灾难再次发生

## 运作机制："五步曲"

```
1. 出现异常行为（首次）
   ↓
2. 未造成伤害 → 暂时接受
   ↓
3. 反复出现 → 形成"这是正常的"共识
   ↓
4. 偏差被规范化 → 写入非正式规则
   ↓
5. 灾难性后果（累积到临界点）
```

## 跨领域应用

| 领域 | 表现 |
|------|------|
| **医疗** | 漏诊率渐进上升、洗手频率下降 |
| **航空** | 安全检查跳过 |
| **金融** | 风险模型放宽、加杠杆成为"新常态" |
| **软件工程** | 测试覆盖率下降、技术债累积 |
| **AI 系统** | 幻觉率上升、跳过验证步骤 |

## 防范措施

1. **保持"异物感"**：对异常保持敏感
2. **定期归零**：每季度重新校准"什么是可接受"
3. **外部视角**：邀请新人/外部专家审查
4. **历史回溯**：定期重温过去灾难的征兆
5. **红线文化**：明确"某些事永远不可接受"
