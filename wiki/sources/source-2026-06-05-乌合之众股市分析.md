# 乌合之众股市分析

## 核心命题

个体融入群体 → 智力下降、理性消退、情绪取代思考 → 冲动/轻信/易被煽动的"野蛮人"

## 五幕戏

| 幕 | 群体特征 | 股市对应 |
|----|---------|---------|
| 个体湮灭 | 独立意识消失，群体情绪淹没判断 | 羊群效应，加仓而非止损 |
| 简单叙事 | 只接受简单/绝对/形象的断言，靠传染扩散 | 概念炒作，口号投资 |
| 形象幻觉 | 被幻觉和形象打动，不被事实打动 | 市梦率，PPT造梦 |
| 情绪极端 | 不是狂喜就是恐慌，不会停在平衡点 | 暴涨暴跌，反身性燃料 |
| 领袖崇拜 | 群体是羊，需要领袖，断言不需要逻辑 | 跟单喊单，不跟逻辑 |

## 知识对应

| 已学概念 | 乌合之众镜下 |
|---------|------------|
| 反身性 | 群体偏见通过极端情绪提供行动力 |
| 泡沫经济 | 简单叙事+幻觉+情绪传染 = 全民狂欢 |
| 明斯基时刻 | 情绪从狂喜瞬切为恐慌，踩踏发生 |
| 逆向投资 | 逆向 = 与群体部落本能反向 |

## 核心启示

> 读懂乌合之众，不是嘲讽市场，而是在群体癫狂时警惕自己内心的冲动，站在稍远的地方冷静观察并利用人性规律。
