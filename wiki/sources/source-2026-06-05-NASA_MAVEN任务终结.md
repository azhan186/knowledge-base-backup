# NASA MAVEN任务终结

## 核心结论

运行超过11年的NASA火星大气探测器MAVEN正式退役。2025年12月失联后，调查认定无法恢复，其成果为载人火星任务辐射防护提供重要依据。

## 任务时间线

| 时间 | 事件 |
|------|------|
| 2013年11月 | 发射升空 |
| 2014年9月 | 进入火星轨道（原计划任务1年） |
| 2025年12月6日 | 最后一次通信，随后失联 |
| 2026年2月 | NASA成立异常情况调查委员会 |
| 2026年6月3日 | 正式宣布任务终结 |

## 故障原因（初步分析）

- 脱离火星遮挡区时处于高速自旋状态
- 轨道姿态异常
- 机载电池电量耗尽
- 通信系统失去供电 → 失联
- 根本原因仍在调查中

## 科学贡献

**研究目标：** 火星上层大气、电离层及其与太阳活动的相互作用；探究火星大气向太空流失的过程

**实际价值：** 为未来载人火星任务提供辐射防护与安全设计依据
