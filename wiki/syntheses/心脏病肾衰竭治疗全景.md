## 一、核心概念速查

| 概念 | 一句话 |
|-----|-------|
| [[冠心病]] | 冠状动脉变窄/堵住→心肌缺血→胸痛或心梗 |
| [[心力衰竭]] | 心脏泵血无力→乏力、气喘、脚肿 |
| [[心律失常]] | 心脏跳得太快/太慢/乱跳→心慌、头晕、晕倒 |
| [[心脏瓣膜病]] | 心脏"门"关不严或打不开→效率下降 |
| [[慢性肾脏病]] | 肾功能持续下降，分1-5期，5期为尿毒症 |

**类比：心脏 = 水泵，血管 = 水管，冠状动脉 = 给水泵供电的专用电缆**

## 二、关键结论：降压药与肾衰竭的关系

**绝大多数情况：降压药不是肾衰竭的元凶，而是保护肾脏的功臣。**

真正伤肾的是：
1. 长期未被控制的高血压
2. 全身动脉粥样硬化（堵塞肾脏血管）

### 出现"吃药导致肾衰竭"的常见原因
1. 肾脏本身已有严重问题（长期高血压/动脉硬化导致肾功能本就在下降）
2. 普利/沙坦类药物的"功能性"血肌酐升高（用药初期升高<30%是良性表现，代表肾脏开始"低压力工作"）
3. 药物引发的急性肾间质损伤（停药后可能恢复）
4. 联合用药不当的"三重打击"（沙坦/普利 + 利尿剂 + 非甾体止痛药）

---

## 三、治疗框架（按阶段）

### 阶段一：调整降压方案（保肾降压，非停药）

| 药物类型 | 处理方式 |
|---------|---------|
| 普利类/沙坦类 | 肌酐升高<30%可继续用，是护肾关键药物 |
| SGLT2抑制剂（列净类） | 近年突破，一线护肾药 |
| 利尿剂 | 若导致血容量不足则减量/停用 |
| β受体阻滞剂 | 与心内科协商换用对肾影响更小的类型 |

**血压目标**：无蛋白尿<140/90mmHg；有蛋白尿<130/80mmHg

### 阶段二：肾脏保护治疗（非药物核心）

| 措施 | 具体做法 |
|-----|---------|
| 限盐 | <5克/天 |
| 优质低蛋白饮食 | 0.6-0.8g/kg/天，以蛋、奶、肉、鱼为主 |
| 控制钾、磷 | 限制高钾食物；减少高磷食物 |
| 严格避免肾毒性 | 禁用非甾体抗炎药；造影前水化；禁用含马兜铃酸中药 |
| 控制其他因素 | 控糖、降蛋白尿、纠正贫血、纠正酸中毒 |

### 阶段三：终末期（尿毒症期）
血液透析、腹膜透析或肾移植

## 四、中药的定位与安全边界

**定位：辅助治疗，绝非替代。西药是底线，中药是辅助。**

### 有用的方面
- 改善症状：乏力、食欲不振、皮肤瘙痒、失眠、夜尿多
- 协同控制蛋白尿：黄芪、大黄、金水宝、百令胶囊等
- 调理整体稳定血压

### 绝对禁区
| 类别 | 举例 |
|-----|------|
| 绝对禁用（肾毒性） | 关木通、广防己、青木香、雷公藤、斑蝥、朱砂、雄黄 |
| 需极度谨慎 | 含钾量高的中药材（可致高钾血症） |

### 安全流程
1. **绝不自行停西药**——底线
2. 找三甲医院**中西医结合肾病科**（不是普通中医）
3. 带齐所有资料：诊断、化验单、用药清单
4. 从**"加药"**开始而非"换药"，观察1-2周
5. **严密监测**：频繁复查血压、肾功能、血钾

## 五、老年患者心理支持

**核心目标：帮老人找回对生活的掌控感**

### 沟通话术
| ❌ 错误 | ✅ 正确 |
|--------|--------|
| "您肌酐又高了" | "今天多走了50步，非常棒" |
| "您必须低盐饮食" | "这个蒸鱼您都吃完了，食谱成功了" |
| "再不吃药就等着尿毒症" | "我们的目标是和病和平共处，让您照样能去公园、能看孙子" |

### 具体策略
- 让患者**参与用药决策**
- 建立**每日具体小任务**（如"9点半客厅抬腿10次"）
- 让患者**记录血压体重**，做成图表
- 家人帮忙**过滤数据信息**，只告知"这周很稳定"
- 用**身体感受**校准而非数据："今天胸闷吗？"
- **同伴力量**：病情相似但状态良好的病友现身说法
- **允许坏情绪**："你想发火就发火，我就在这儿陪你"

---

## ⚠️ 绝对禁止

1. ❌ 自行停降压药（特别是洛尔类如美托洛尔）——突然停药可引发心梗反跳
2. ❌ 自行调整蛋白质摄入量——必须由营养师计算
3. ❌ 相信"逆转肾衰竭"的偏方保健品
4. ❌ 服用任何含马兜铃酸的中药
