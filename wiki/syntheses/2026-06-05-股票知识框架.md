# 第一系列：股票投资十二篇

## 一、核心公式体系（由浅入深）

```
第1层  股价 = EPS（盈利）× PE（市盈率）
       ↓ 加深
第2层  超额收益 = 你的认知 - 市场的共识
       ↓ 再加深
第3层  股票 = 时间预期报酬的权利
              ↕
        市场 = 反身性 + 非遍历性 + 复杂适应系统
```

**三个层次**：
- **入门层**：股票是所有权证书，买的是公司一小部分
- **资深层**：股价由EPS和PE共同决定，赚的是认知与共识的差
- **专家层**：股票是时间的权利，市场是自反性的，永不被迫离场是核心

---

## 二、市场博弈结构

| 对手 | 优势 | 散户应对 |
|------|------|---------|
| 机构投资者 | 研究团队+算法+速度 | 难以短线竞争 |
| 大股东/高管 | 内部信息 | 大规模增减持是强信号 |
| 散户 | 无 | 逆向投资：人恐我贪，人贪我恐 |

**护城河**：用时间换空间的耐心 + 情绪掌控力

---

## 三、定价机制

- **EPS（每股盈利）**：慢变量，公司经营成果
- **PE（市盈率）**：快变量，市场情绪/预期/偏见

股价上涨的两条路：
1. 盈利驱动（EPS↑）→ 真实价值增长
2. 估值驱动（PE↑）→ 市场情绪/预期提升

---

## 四、风险体系

### 黑天鹅（防不住，只能靠系统扛）
- 特征：事前不可预知、冲击巨大、事后可解释
- 应对：杠铃策略 + 冗余现金 + 识别脆弱性
- 案例：2008金融危机、2020新冠

### 灰犀牛（防得住，但人性让你站在原地）
- 特征：高概率、有明显预警、被集体忽视
- 应对：警惕"这次不一样" + 信号清单 + 提前行动
- 案例：2015年A股杠杆牛市崩塌

### 脆弱性 vs 反脆弱
- 脆弱 = 高杠杆 + 现金流紧绷
- 反脆弱 = 手握净现金 + 能反向扩张

## 五、泡沫→崩溃的完整剧本

```
半真叙事（时代故事）
    +
廉价资金（低利率/杠杆/新投资工具涌入）
    +
反身性正反馈（价格涨→基本面改善→验证高价→更多资金）
    ↓
繁荣-崩解六阶段
    ①启动 ②加速 ③自我验证 ④黄昏时刻 ⑤崩溃点 ⑥反向反身性
    ↓
明斯基时刻（庞氏融资断裂的瞬间）
    ↓
流动性海市蜃楼：所有人同时站卖方，对面无人出价
```

**三阶段演化**：对冲融资 → 投机融资 → 庞氏融资（崩溃前夜）

## 六、反身性核心

**双向反馈环**：
- 认识函数：你对市场的认知
- 参与函数：你的行为改变市场
- 两者互相喂食，直到现实无法承载

**索罗斯三问**：
1. 当前价格中内含了什么主流偏见？
2. 这个偏见通过什么路径改变基本面？
3. 离现实约束的边界还有多远？

---

## 七、群体心理（乌合之众五幕戏）

1. **个体湮灭**：羊群效应，加仓而非止损
2. **简单叙事**：口号投资，概念炒作
3. **形象幻觉**：市梦率，PPT造梦
4. **情绪极端化**：暴涨暴跌，反身性燃料
5. **领袖崇拜**：跟单喊单，不跟逻辑

---

## 八、关键法则

| 法则 | 说明 |
|------|------|
| 复利的敌人是"打断" | 一次清零，之前积累全部归零 |
| 决策的终极是放弃 | 99个机会不在认知圈→不动；第100个非对称机会→雷霆行动 |
| 逆向 = 与部落本能反向 | 人恐我贪，人贪我恐 |
| 活下去是最高法则 | 不被市场消灭，才有机会享受后续 |

---

## 十、核心指标速查

| 指标 | 含义 | 要点 |
|------|------|------|
| PE（市盈率）| 多少年回本 | 高PE=高增长预期；低PE可能是一次性盈利 |
| PB（市净率）| 股价vs每股净资产 | PB<1=破净，比清算价值还便宜 |
| 股息率 | 每年现金分红/股价 | 股价跌时股息率反而高，可能是陷阱 |
| 换手率 | 成交量/流通股本 | 3%活跃，>10%极高，分歧巨大 |
| CAPE | 经周期调整的市盈率 | 识别整体估值极端区间 |

# 第二系列：叙事经济学四篇

## 核心公式

**R₀ = c × p × d**（基本再生数）

- c = 每人每天分享次数
- p = 每次分享被接受并再传播的概率
- d = 叙事保持传染力的天数

R₀ >> 1 → 叙事野火蔓延，成为经济"流行病"

---

## 四步劫持闭环

1. **情感触发**：恐惧/贪婪/归属叙事 → 绕开理性系统
2. **因果赋义**：识别反派 + 救赎之路 + 道德许可
3. **社会加压**：信息瀑布 + "这么多人不可能都错了"
4. **自我锁死**：认知失调 → 成为最狂热的叙事传播者

---

## 三大永恒剧本（集体无意识结构）

| 剧本 | 内核 | 历代化身 |
|------|------|---------|
| 炼金术 | 低价值→高价值的神奇转化 | 南海泡沫→互联网→加密货币 |
| 启示录 | 末日将至，末日方舟得救 | 黄金→比特币→火星船票 |
| 巨人杀手 | 小人物团结战胜大坏蛋 | 散户vs华尔街→民粹主义 |

**关键**：新叙事成功不是因为有多新，而是成功嫁接到古老的永恒剧本上。

---

## AI时代新形态

- **算法传人**：推荐系统放大叙事，R₀指数级提升
- **合成现实**：深度伪造视频可在辟谣前造成市场闪崩
- **微观靶向**：同一产品对不同群体用不同叙事版本，群体隔离在信息茧房

---

## 专家终极姿态：苏格拉底式对话者

1. **承认安住于故事之中**：知道自己必然活在某个经济故事里，无需羞愧
2. **保持提问而非回答**：
   - "在这个故事里，谁的痛苦被静静地省略了？"
   - "如果这个故事的结局二十年后才来，中途的人都怎么活？"
3. **成为场域**：让极端对立的叙事同时被看见，不被任何单一剧本完全劫持

> 身在故事中，心在故事外。**不是答案的拥有者，而是问题的持守者。**
