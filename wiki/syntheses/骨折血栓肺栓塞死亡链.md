# 骨折-血栓-肺栓塞：死亡链完整解析

> 三大概念联动：[[骨折机制]] + [[血栓机制]] + [[肺血栓栓塞症]]

## 核心命题

**腿骨骨折是肺栓塞的最高危场景之一。**

这是魏尔肖"血栓形成三要素"在骨折病人身上同时、急剧、高强度爆发的结果。

---

## 死亡链全景

```
腿骨骨折（股骨/胫骨）
         │
         ├── 骨茬刺伤静脉血管内壁      → 血管壁损伤
         ├── 剧痛+固定，整条腿不能动    → 血流瘀滞
         └── 全身应激反应，肝脏大量生成凝血因子 → 血液高凝
                   │
                   ▼
         下肢深静脉血栓（DVT）形成
                   │
         患者开始活动（甚至只是翻身）
                   │
                   ▼
              血栓脱落
                   │
         顺血流：下腔静脉 → 右心房 → 右心室 → 肺动脉
                   │
                   ▼
         肺栓塞（"淤泥块"卡死"净水过滤器"）
                   │
         呼吸（氧合）+ 循环（泵血）两大系统被同时破坏
                   │
                   ▼
            急性右心衰竭 → 猝死
```

---

## 三要素激活对比

| 三要素 | 正常血栓形成 | 骨折场景 |
|--------|-------------|---------|
| **血管壁损伤** | 动脉粥样硬化斑块破裂 | 骨折骨茬直接刺伤、撕裂静脉内壁 |
| **血流瘀滞** | 长期卧床、久坐 | 剧痛+石膏/牵引固定，肌肉泵作用消失 |
| **血液高凝** | 遗传疾病、肿瘤、脱水 | 骨折应激反应，肝脏大量生成纤维蛋白原 |

> 骨折场景的特殊性：**三要素同时激增，强度远高于普通情境**

## 临床干预节点

### 预防（打断死亡链）

| 措施 | 内容 |
|------|------|
| **危险分层** | Caprini评分表，下肢大手术直接极高危 |
| **基础预防** | 早期踝泵运动、弹力袜、间歇充气加压 |
| **药物预防** | 低分子肝素/磺达肝癸钠（"肚皮针"），术前术后规律注射 |
| **早期筛查** | 定期监测D-二聚体；疑似时安排血管彩超或CTPA |

### 高龄患者特殊考量

- **症状更隐匿**：很少典型胸痛，需警惕"不典型信号"（呼吸急促、神志改变、血压下降）
- **策略**：物理方法用尽，药物极度审慎；骨科-老年科共管模式
