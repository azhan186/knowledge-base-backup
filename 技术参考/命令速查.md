# 命令速查

## NAS 系统
```bash
uname -a           # 系统信息
uptime             # 运行时间
free -h            # 内存状态
df -h /            # 磁盘状态
hostname -I        # IP 地址
```

## OpenClaw
```bash
openclaw status              # 状态
openclaw gateway status      # Gateway 状态
openclaw devices list        # 设备列表
openclaw pairing approve     # 审批配对
openclaw logs --follow       # 查看日志
openclaw channels add        # 添加频道
```

## 文件操作
```bash
ls -la /path/to/dir          # 列出目录
mkdir -p /path/to/dir        # 创建目录
mv old new                    # 移动/重命名
rm -rf /path                 # 删除（谨慎）
```
