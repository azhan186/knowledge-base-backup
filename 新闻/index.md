# 新闻库

独立新闻三层架构：Raw Sources（原始素材）→ syntheses（合成）→ concepts（概念沉淀）

## 目录结构

```
新闻/
├── Raw Sources/2026/   ← 原始新闻（按主题分类）
│   ├── 国际/
│   ├── 时政/
│   ├── 法律/
│   ├── 科技/
│   └── 财经/
├── syntheses/          ← 合成笔记（按主题索引）
└── concepts/            ← 概念/实体沉淀
```

## 主题分类

- [[国际]]：俄乌冲突、美伊互袭等
- [[时政]]：国内政策、社会治理
- [[法律]]：典型司法案例
- [[科技]]：AI、芯片、量子等产业科技
- [[财经]]：金融市场、宏观经济、产业动态
