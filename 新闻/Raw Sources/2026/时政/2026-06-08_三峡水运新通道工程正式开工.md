# 三峡水运新通道工程正式开工，"十五五"首个国家重大标志性工程（2026-06-08）

## 来源
- **时间**：2026-06-08 开工
- **来源**：央视/新华社（"十五五"国家重大工程通稿）
- **地点**：湖北省宜昌市

## 核心事件

**三峡水运新通道工程**在湖北宜昌正式开工建设——这是我国"**十五五**"期间开工建设的**首个国家重大标志性工程**。

**战略意义**：
- 推动**长江经济带**发展战略实施
- 构建以**国内大循环为主体、国内国际双循环**相互促进的新发展格局

## 投资与工期

| 部分 | 工期 | 投资 |
|------|------|------|
| 三峡枢纽新通道 | **112 个月** | 总投资约 **772 亿元** |
| 葛洲坝航运扩能 | **95 个月** | （含在 772 亿内） |

## 工程组成

### 1. 三峡枢纽新通道
- 三峡枢纽将形成 **四线船闸** 格局（+ 既有升船机）
- **总通过能力**：3.36 亿吨/年

### 2. 葛洲坝枢纽航运扩能
- 主要建设：拆除三号船闸 + 左侧新建两线单级船闸 + 上下游引航道扩挖
- 葛洲坝枢纽将形成 **四线船闸** 格局
- **总通过能力**：3.6 亿吨/年

## 新通道技术参数

| 项目 | 参数 |
|------|------|
| **通道全长** | 6680 米 |
| **结构** | 三部分：上游引航道 / 船闸主体段（五级"阶梯"）/ 下游引航道 |
| **每个闸室** | 长 280 米 × 宽 40 米 × 水深 8 米（**比现有三峡船闸深 3 米**） |
| **每扇闸门高度** | 相当于 **13 层居民楼** |
| **可通行船型** | **万吨级** 大型船舶（现主流为 5000 吨级） |
| **通航方式** | 两侧航道实现**双向通航** |

## 关键性能提升

| 指标 | 现状 | 新通道 |
|------|------|--------|
| 主流通航船型 | 5000 吨级 | **10000 吨级（万吨级）** |
| 年通过能力 | — | **3.36 亿吨** |
| 万吨级船舶货运成本 | — | **下降约 30%** |

## 经济效益

- **年节约成本估算**：船舶待闸成本 + 替代陆路运输费用 合计可达 **180 亿元/年**
- **服务对象**：成渝等**西南地区**的大宗商品（更低成本 + 更准时的通江达海）
- **战略效应**：
  - 提升长江黄金水道运输效率
  - 优化供应链体系
  - 增强西南地区产业市场竞争力

## 船闸原理

```
每扇闸门挡水 / 蓄水 / 排水 / 控水
    ↓
分隔上下游水位
    ↓
帮助船只"阶梯式"升降过坝
    ↓
五级船闸主体段 = 整个工程的"心脏"
```
