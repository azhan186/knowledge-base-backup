# 国防部：2026 年全军院校计划招收约 1.75 万名普通高中毕业生（2026-06-09）

## 来源
- **时间**：2026-06-09 下午
- **来源**：国防部新闻发布会
- **发言人**：张晓刚大校（国防部新闻发言人）
- **背景**：2026 年全国高考进行中

## 核心数据

- **全军 22 所院校**
- **2026 年计划招收约 1.75 万名普通高中毕业生**

## 招生三大特点

### 一、生长军官贯通培养
- 本科教育阶段**打牢**学员的：
  - 科学文化
  - 军政素质
  - 专业基础
- **注重指挥素养培育**，夯实长远发展潜力
- 为部队岗前培训奠定扎实基础

### 二、加强政治军官针对性培养
- 依托**国防科技大学**开设 2 个专业招收青年学生：
  - **马克思主义理论**
  - **思想政治教育**
- **从源头上巩固建强我军政治军官队伍**

### 三、临床医学"5+3"一体化培养
- **2026 年起**在 3 所军医大学建立：
  - **陆军军医大学**
  - **海军军医大学**
  - **空军军医大学**
- **培养体系**：
  - 5 年医学本科教育
  - + 3 年硕士专业学位研究生教育
  - + 住院医师规范化培训
- 目标：**培育高素质专业化的卫勤医疗队伍**

## 第四方向

- 依托院校**特色优势学科**，继续开展**新兴技术领域**人才培养

## 关键术语

| 术语 | 含义 |
|------|------|
| 生长军官 | 从高中直接招录、本科培养的军官 |
| 政治军官 | 部队中负责政治工作的军官 |
| 政治整训 | 深化军队政治建设的整训行动 |
| 卫勤 | 部队卫生勤务（医疗保障） |
| 5+3 一体化 | 5 年本科 + 3 年规培硕士衔接培养 |

## 战略意义

```
深化政治整训
    ↓
精准对接部队战建备治人才需求
    ↓
依据新军队院校教育培训任务规划
    ↓
三大方向：
├─ 生长军官贯通（指挥素养 + 长远发展）
├─ 政治军官培养（国防科大 2 个专业）
└─ 卫勤医疗（5+3 一体化，三大军医大学）
    +
特色新兴技术学科
    ↓
加快建成世界一流军队
```
