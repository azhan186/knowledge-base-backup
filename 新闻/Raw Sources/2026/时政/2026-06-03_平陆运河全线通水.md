# 世纪工程！全线通水——西部陆海新通道骨干工程平陆运河

## 要点

西部陆海新通道骨干工程平陆运河建设迎来重大节点：三大航运枢纽中的马道枢纽和企石枢纽开始充水，运河正式开启全线通水，进入有水调试阶段。预计今年9月全线通航。

## 核心数据

| 指标 | 数据 |
|------|------|
| 全长 | 134.2公里 |
| 起点 | 南宁横州市平塘江口 |
| 终点 | 北部湾 |
| 水位落差 | 65米 |
| 总投资 | 727.2亿元 |
| 土石方工程量 | 约3.15亿立方米（三峡工程3倍） |
| 通航等级 | 5000吨级（国内最高） |
| 建成后货运缩短航程 | 560公里以上 |

## 世界之最

- **马道枢纽**：水位落差最大的省水船闸（29.6米，约10层楼高）+ 规模最大的内河省水船闸（闸室长300米、宽34米，相当于1.5个标准足球场）
- **阀门启闭速度**：开门1分钟、关门30秒，世界最快
- **三级省水池叠合布置**：世界首创，马道枢纽用1302根、企石枢纽用1152根钢筋混凝土柱子支撑

## 运河经济带

- 南宁港为江侧起点"第一承运港"，产品可直接从北部湾出海，物流成本直线下降
- 南宁港六景作业区铁路专用线工程抓紧建设中
- 新能源船舶智造基地预计6月底开工，达产后年产能约70艘新能源船舶
- 现代绿色化工、先进钢铁材料、有色金属、人工智能等临海沿江大工业已形成

## 重大工程联动

黄桶至百色铁路、钦州港大榄坪自动化集装箱泊位等加快建设。
