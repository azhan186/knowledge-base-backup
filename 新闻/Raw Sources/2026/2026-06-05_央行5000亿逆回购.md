# 央行宣布5000亿元买断式逆回购操作

**日期：** 2026年6月5日（操作日）
**分类：** 财经 / 货币政策
**来源：** 中国人民银行

---

## 核心事件

央行宣布2026年6月5日以固定数量、利率招标、多重价位中标方式开展**5000亿元买断式逆回购操作**，期限3个月（92天），到期日2026年9月5日。目的为保持银行体系流动性充裕。

## 操作详情

| 项目 | 内容 |
|------|------|
| 操作量 | 5000亿元 |
| 方式 | 买断式逆回购，固定数量、利率招标、多重价位中标 |
| 期限 | 3个月（92天） |
| 操作日期 | 2026年6月5日 |
| 到期日 | 2026年9月5日（遇节假日顺延） |
| 目的 | 保持银行体系流动性充裕 |

## 相关概念

- 买断式逆回购：央行向市场投放流动性的工具，一级交易商以债券为质押向央行融入资金
- 银行体系流动性：金融机构在央行的存款准备金水平，影响信贷投放能力
