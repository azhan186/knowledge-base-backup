# 美法官裁定：特朗普政府 H-1B 签证政策违法（2026-06-08）

## 来源
- **时间**：2026-06-08（美东时间）
- **来源**：央视/新华社转发，路透/美联社

## 一、核心裁决：H-1B 签证 10 万美元费用违法

### 裁决方
- **马萨诸塞州联邦地区法院** 法官（姓名未披露）

### 裁决对象
- **特朗普政府** 2025 年 9 月签署的 H-1B 签证政策
- 被告机构：**美国国务院** + **美国公民及移民服务局（USCIS）**

### 裁决要点

| 项目 | 内容 |
|------|------|
| 政策内容 | H-1B 签证申请费从数千美元 → **10 万美元** |
| 裁决认定 | 此款项"无论叫什么名字，都是税款的一种"，具有**税收性质** |
| 法律依据 | 联邦移民法赋予总统限制外国人入境的权力，但**未授权总统征税** |
| 处置方式 | 国务院 + USCIS **不得执行**该政策 |

### 案件背景

- **原告**：20 名**民主党籍**州检察长
- **诉求**：H-1B 政策超越行政权力，影响各州雇用高技能外国专业人才
- **涉及签证**：H-1B 是美国临时工作签证（非移民签证），允许美国公司雇用外籍专业技术人员，主要被**美国科技企业**用于吸引高学历外籍人才

### 消息来源

### 政策内容

| 项目 | 内容 |
|------|------|

| 适用 | 申请美国**商务或旅游签证**（B-1/B-2）的外国公民 |
| 启动时间 | **2026-07-01** |
| 结束时间 | **2026-12-31** |
| 形式 | 加快签证**面试预约**流程 |

## 四、影响分析

### H-1B 政策被否的影响
- 美国科技企业（FAANG 等）外籍员工招聘**预期重置**
- 印度外包公司（Infosys、Tata、Wipro）业务模式承压
- 特朗普政府可能**上诉**到联邦巡回上诉法院

### 加急服务的背景
- 美国非移民签证预约**长期积压**（数月等待）
- 750 美元定价反映"**付费优先**"思路
- 与 H-1B 政策被否形成对比：一边限制，一边便利化
